Types of Formulas -- 1-2-3 lets you enter four types of formulas:

Numeric    Performs calculations with numbers and uses one or more of the
           arithmetic operators. For example, +B5*5 multiplies the value
           in cell B5 by 5.
String     Performs calculations with strings (text enclosed in quotation
           marks or labels in a worksheet). For example, the formula
           +"Mr. "&B2 joins Mr. (and a space) and the label in cell B2.
Logical    Performs true/false tests on numeric or string values and
           returns 1 if true and 0 if false. Logical formulas use logical
           operators. For example, +A1>500 returns 1 if the value in
           cell A1 is greater than 500 and 0 if the value in A1 is less
           than or equal to 500.
@Function  Performs specific database, date-and-time, financial, logical,
           mathematical, statistical, scientific, or string calculations.
           For example, the formula @SUM(B10..F10) calculates the sum
           of the values entered in the range B10..F10.
