Cannot use ranges that are not in memory

The following conditions could have caused this error:

* You specified a range in a file on disk or the range name of an
   external table. For this command, the range you specify must be in
   an active file. Specify a range in an active file.

* You specified an undefined range name. Use /Range Name Create to create
   the range name you used in the command.
