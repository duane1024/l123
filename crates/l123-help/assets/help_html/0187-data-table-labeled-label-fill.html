/Data Table Labeled Label-Fill

/Data Table Labeled Label-Fill -- Specifies a label-fill character for the
formula label range and/or column variable range. A label-fill character
is a formatting character that lets you center a label in a formula label
range. The - (hyphen) is the default label-fill character.

Generally, you use label-fill characters when your formula range contains
one formula and your formula label range spans two or more adjacent
columns. You can also use label-fill characters in a column variable
range if the range contains labels, as it might in a database table.

1. Select /Data Table Labeled Label-Fill if you need to specify a label-
    fill character other than a - (hyphen) for the formula label range
    or the column variable range.
2. Specify the character you want to use.
