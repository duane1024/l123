when none was loaded. Select Load from the ALT-F10 (ADDIN) menu to read an
add-in file into memory.

No add-ins in memory --  You attempted to remove an add-in from memory when
no add-ins were loaded.

Settings sheet is empty -- You selected Settings System Cancel or Settings
File Cancel when there were no add-in files listed on the settings sheet.

Settings sheet is full -- You attempted to add the name of an add-in file to
a full settings sheet, one that lists the names of 15 add-in files. Use
Settings File or Settings System from the ALT-F10 (ADDIN) menu to delete
files from the settings sheet you want to modify.
