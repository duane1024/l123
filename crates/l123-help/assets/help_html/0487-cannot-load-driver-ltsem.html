Cannot load driver "LTSEM"

Cannot load driver "LTSEM" -- Not enough memory was available to read LTSEM
into memory, or LTSEM was damaged or deleted from your hard disk, or you are
already using the maximum number of files your operating system allows.

Cannot load driver "LTSLEAF" -- Not enough memory was available to read
LTSLEAF into memory, or LTSLEAF was damaged or not found on your hard disk,
or you are already using the maximum number of files your operating system
allows.

Cannot load "file_name" -- Not enough memory was available to read into
memory the file named in the error message, or the file named in the error
message was damaged or deleted from your hard disk, or you are already using
the maximum number of files your operating system allows.

Cannot read "file_name" --  The file may have been damaged, or a hardware
error occurred, for example, the door of your diskette drive was open
while you attempted to read a file from the diskette.
