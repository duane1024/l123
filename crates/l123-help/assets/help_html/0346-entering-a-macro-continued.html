Entering a Macro (continued) -- How-To Tips

    Enter the macro instructions as one or more labels in a column. When
   a macro label begins with a / (slash), \ (backslash), : (colon), number, or
   any character that puts 1-2-3 in VALUE mode (for example, +  -  @  . or  #),
   type a label prefix ( '  ^  or  " ) first.

    Except for very short macros, divide the macro instructions among
   several cells in a column to make the macro easier to read and debug.

    For macro instructions enclosed in { } (braces), enter the entire
   instruction in one cell.

    To end a macro, use {QUIT} or follow the last cell of the macro
   instructions with an empty cell.
