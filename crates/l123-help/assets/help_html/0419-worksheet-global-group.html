/Worksheet Global Group

/Worksheet Global Group -- Turns GROUP mode on or off for the current file.
Use GROUP mode when you want all the worksheets in a file to look the same.
When you turn on GROUP mode, l123 assigns the current worksheet's cell
formats and settings to all worksheets in the current file. Until you
turn off GROUP mode, changes you make to one worksheet with any of the
following commands affect every worksheet in the file:
    /Worksheet [Column, Page, Titles]
    /Worksheet Delete Column or Row and /Worksheet Insert Column or Row
    /Worksheet Global [Col-Width, Format, Label, Prot, Zero]
    /Range [Format, Label, Prot, Unprot]

CAUTION  Turning on GROUP mode makes permanent changes to the settings for all
worksheets in the current file. Turning off GROUP mode does not reverse those
changes.

1. Select /Worksheet Global Group and select Enable or Disable.
    Enable turns on GROUP mode with the results described above.
    Disable (default) turns off GROUP mode for the current file.
