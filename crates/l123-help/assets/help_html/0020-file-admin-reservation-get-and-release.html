/File Admin Reservation Get and Release

/File Admin Reservation Get and Release -- Control the current file's
reservation.

1. Select /File Admin Reservation.
2. Select Get or Release.
    Get      Gets the reservation for the current file if it is available
             and no one has saved the file since you read it into memory.
             When you get the reservation, you are the only person who
             can save changes to the file.
    Release  Gives up the reservation for the current file so someone else
             can get it. Before you release the reservation, be sure you
             use /File Save if you want to save changes you made to the file.
