Documenting a Macro

Documenting a Macro -- Document the macro's range name by entering the name
to the left of the macro's first cell. If the range name starts with a
backslash (such as \N), remember to use a label prefix ( '  "  or ^ )
when you enter it. Document the macro instructions by entering comments
about them in a column to the right of the instructions.

In the following example, the macro named NEW_SHEET is in column B, the
macro's name is documented in A1, and the macro instructions are
documented in column C.

       A             B              C

1 NEW_SHEET      /wisa~       Insert new worksheet after current one
2                {RIGHT}      Move to column B in new worksheet
3                /wcs30~      Widen column B to 30 characters
4
