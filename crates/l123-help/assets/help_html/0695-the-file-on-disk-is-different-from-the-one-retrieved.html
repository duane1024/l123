The file on disk is different from the one retrieved

You used /File Admin Reservation Get to try to get a reservation for the
current file, but the version of the file on disk has changed since
you retrieved it. l123 will not give you the reservation if the file
on disk is different from the file you are using.

Use /File Retrieve to read the most recent version of the file into
memory. If the file is read-only, use /File Admin Reservation Get
to try to get the reservation.
