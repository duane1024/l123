/Data External Other Refresh

/Data External Other Refresh -- Re-executes Data Query and Data Table
commands and updates database @functions at an interval you specify.
This command is useful if you are working with external databases that
other users may be changing frequently and you need to ensure that the
information you are working with is current.

Automatic   Updates database @functions, data queries, and what-if tables
             at a specified time interval.
Manual      Updates database @functions, data queries, and what-if tables
             only on command.
Interval    Specifies the time interval, measured in seconds (0 through
             3600), at which l123 automatically updates database
             @functions, data queries, and what-if tables.

NOTE  After you select /Data External Other Refresh Automatic, you cannot
use the Data Query or Data Table commands to create or update queries or
what-if tables. To restore the use of the Data Query and Data Table
commands, select /Data External Other Refresh Manual.
