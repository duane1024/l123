/Data Table Labeled Formulas

/Data Table Labeled Formulas -- Specifies the formula range and the formula
label range. The formula range is the range that contains the formula(s)
you want to use to calculate the table and the labels that describe
the formulas.

The formula label range contains a copy of the labels in the formula
range and can also contain values, blank cells, and labels that do not
appear in the formula range. When you calculate a labeled table,
1-2-3 uses the formula label range in two ways: to determine which
formulas to use with the input values in the corresponding row, column,
or worksheet of the table, and to determine the placement of the
results of the calculations.

1. Select /Data Table Labeled Formulas.
2. Specify the formula range.
3. Specify the formula label range.
