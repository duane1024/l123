/Range Trans (continued)

5. If you specified a three-dimensional FROM or TO range, select one of
    the following:

    Rows/Columns         Transposes the data in the FROM range from a
                         horizontal arrangement to a vertical arrangement,
                         or vice versa.
    Columns/Worksheets   Copies the first column in each worksheet of the
                         FROM range to the first worksheet in the TO
                         range, the second column in each worksheet of the
                         FROM range to the second worksheet in the TO range,
                         and so on.
    Worksheets/Rows      Copies the first row in each worksheet of the FROM
                         range to the first worksheet in the TO range, the
                         second row in each worksheet of the FROM range to
                         the second worksheet in the TO range, and so on.
