/Graph Options Advanced Text [First, Second, Third] Font

/Graph Options Advanced Text [First, Second, Third] Font -- Sets fonts for
the graph-text groups in printed graphs. In displayed graphs, l123
uses one font for all graph text, so this command does not affect graph
display unless you use an external utility program to add other display
fonts.

1. Select /Graph Options Advanced Text.
2. Select First, Second, or Third to select the graph-text group.
3. Select Font and select one of the following:
    1 - 8     To set the corresponding font for the selected graph-text group.
    Default   To use the default font setting for the selected
              graph-text group. The defaults are 3 for the first graph-text
              group and 1 for the second and third graph-text groups.

NOTE  l123 provides eight font settings for graph text. The variety of
fonts available to you, however, depends on your printer.
