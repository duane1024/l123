Macro Commands (continued)

    {WINDOWSOFF}
        Freezes the worksheet area of the screen during a macro until
        1-2-3 reaches a {WINDOWSON} command or the macro ends.

    {WINDOWSON}
        Unfreezes the worksheet area of the screen after {WINDOWSOFF}.

    {WRITE string}
        Copies string to the open text file, starting at the current
        byte-pointer position in the open text file.

    {WRITELN string}
        Same as {WRITE} except that it adds a carriage-return and
        line-feed sequence to the string it writes to the open file.
