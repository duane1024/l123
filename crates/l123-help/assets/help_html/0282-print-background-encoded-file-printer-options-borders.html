/Print [Background, Encoded, File, Printer] Options Borders

/Print [Background, Encoded, File, Printer] Options Borders -- Sets borders
(specified columns and/or rows) to print on each page and lets you print the
worksheet frame.

1. Select /Print [B,E,F,P] Options Borders.
2. Select one of the following:
    Columns    Specifies columns you want to print at the left margin of
               each page and every print range. Use Columns to create a
               vertical border on a page.
    Rows       Specifies rows you want to print above your print ranges.
               Use Rows to create a horizontal border on a page.
    Frame      Prints the frame on each page for every print range.
    No-Frame   Cancels printing of the frame.
3. If you selected Columns or Rows, specify a range that includes the
    columns or rows you want to use as a border. Do not include in your
    print range the columns or rows you used as borders because l123
    will print those columns and rows twice.
