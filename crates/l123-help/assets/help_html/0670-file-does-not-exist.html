File does not exist

The following conditions could have caused this error:

* You specified a file on disk that did not exist in the specified directory.
   Select /File List to find out what files are in the current directory
   or what files are active.

* You specified an active file that is not stored in the current
   directory. When l123 prompts you for the file name, enter the path
   and the file name, or press F3 (NAME) and select the file name from
   the displayed list.
