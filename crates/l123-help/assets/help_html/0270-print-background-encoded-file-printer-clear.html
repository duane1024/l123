/Print [Background, Encoded, File, Printer] Clear

/Print [Background, Encoded, File, Printer] Clear -- Resets some or all current
print settings.

1. Select /Print [B,E,F,P] Clear and select one of the following:
    All        Clears the current print range and graph name; clears all
               borders, headers, and footers; resets all formats and
               options to their default settings.
    Range      Clears the current print range.
    Borders    Clears all borders, including the worksheet frame.
    Format     Resets margins, page length, layout, fonts, colors, setup
               strings, and graph settings to their default settings.
    Image      Clears the name of the graph you selected to print.
    Device     Resets the printer name, interface, and baud-rate settings
               to their default settings.
2. Select new settings from the /Print menu, or select Quit to return 1-2-3
    to READY mode.
