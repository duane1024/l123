Text File Definitions

Text File
    A file on disk in ASCII format. A line from a text file created from
    data in a 1-2-3 worksheet might look like this:

    Sheryl  Peterson   1400     300     80

Delimited Text File
    A file on disk in ASCII format that contains characters (delimiters)
    that separate data. The delimiters can be commas, spaces, colons, or
    semicolons. All labels must be enclosed in quotation marks. To create
    a delimited text file, you can use a text editor to enter the
    delimiters in a text file. Some programs have commands or translate
    utilities that create delimited text files. A line from a delimited
    text file might look like this:

    "Sheryl","Peterson",1400,300,80
