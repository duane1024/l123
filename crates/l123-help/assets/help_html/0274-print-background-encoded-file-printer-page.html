/Print [Background, Encoded, File, Printer] Page

/Print [Background, Encoded, File, Printer] Page -- /Print Printer Page and
/Print Background Page advance the paper in the printer to the top of the
next page. /Print File Page and /Print Encoded Page insert as many lines
in a file as l123 needs to advance a printed page to the next page and
include the footer in the file.

1. Select /Print [B,E,F,P] Page.

If you specified a footer, l123 prints the footer at the bottom of the
current page before advancing to the next page. If the paper was at the
top of a page when you selected /Print [B,E,F,P] Page, l123 prints the
header and footer on the page.
