/Range Name Create

/Range Name Create -- Creates a defined range name by assigning the name
to a range address. Use /Range Name Create to create range names for use
in formulas and commands, and to assign names to macros.

1. Select /Range Name Create.
2. Type a name up to 15 characters long and press ENTER.

    1-2-3 does not distinguish between uppercase and lowercase letters
    in range names. Do not start a range name with the character !, and
    do not include spaces, commas, semicolons, periods, or any of the
    following characters in a range name:

            +    *    -    /    &    >    <    @    #  {

    Do not create names that look like cell addresses, such as P12 or EX100,
    or names that begin with numbers, such as 20DEC. Do not use @function
    names, key names, or macro command keywords as range names.
