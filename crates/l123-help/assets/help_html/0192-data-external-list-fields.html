/Data External List Fields

/Data External List Fields -- Lists the field names in an external table to
which you are currently connected and lists the data type, width, column
label (if any), and description (if any) of each field in the columns to the
right of the field names. In addition, l123 creates a sixth column in which
you can enter field creation strings, if you want to use the results of this
command for a table definition. An NA in a column to the right of a field
name indicates that information of a particular type -- for example, a field
description -- is not available for that particular field.

1. Select /Data External List Fields.
2. Specify the range name of the table whose fields you want to list.
3. Specify the first cell of the range or the name of the range in which
    you want the field information to appear.

    CAUTION  l123 writes over existing data in the location you specify for
    the list of field names. Make sure you specify a location that does not
    contain data you need.
