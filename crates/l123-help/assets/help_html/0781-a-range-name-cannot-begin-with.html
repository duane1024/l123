A range name cannot begin with << >>

The following conditions could have caused this error:

* For /Range Name Create, you specified a range name that started with
   << >> (double angle brackets).

* For /Range Name Labels, the labels range you specified contained a label
   that started with << >>.

* For /Data External Create or Use, you specified an external table range
   name that started with << >>.

Range names cannot begin with double angle brackets because l123 uses
those characters to denote file references.

To name a range in another active file, move the cell pointer to that
file before selecting /Range Name Create.
