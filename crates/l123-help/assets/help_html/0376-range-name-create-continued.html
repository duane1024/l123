3. Specify the range you want to name.

    If you specified an existing range name, l123 highlights the range
    to which the name is currently assigned. To assign the name to a
    different range, press BACKSPACE to remove the highlight, and specify
    a new range.

When you save a file or extract data to a new file, l123 saves the range
names with the file.
