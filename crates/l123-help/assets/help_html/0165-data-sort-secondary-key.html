/Data Sort Secondary-Key

/Data Sort Secondary-Key -- Specifies the column (field) the contents of which
determine the order for records with identical primary sort-key entries.

1. Select /Data Sort Secondary-Key.
2. Specify the address of any cell in the field you want to use to sort
    records that have identical primary sort-key entries.
3. Select A (ascending) or D (descending).
    See /Data Sort Primary-Key for information on this choice.

In the following example, NAME is the primary-key field, SALES is the
secondary-key field, and the sort order for both keys is ascending.

     Before /Data Sort Go                After /Data Sort Go

     NAME       SALES                    NAME       SALES
     Baker      40,000                   Baker      30,000
     Cataldo    35,000                   Baker      40,000
     Baker      30,000                   Cataldo    35,000
