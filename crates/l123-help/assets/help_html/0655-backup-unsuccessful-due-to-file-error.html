Backup unsuccessful due to file error

You used /File Save Backup to save a .BAK file that was marked read-only
on disk. You cannot delete or save data in a read-only file.

If you need to save the file, use /System to get to the DOS prompt and
use the DOS ATTRIB command to clear the read-only attribute from the file.
Then type exit to return to 1-2-3, and use /File Save Backup to save the file.
