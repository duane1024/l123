Specifying Ranges

Specifying Ranges -- Many l123 commands and formulas require you to
indicate, or specify, the range of data you want to use in the command
or formula. You can specify a range in the following ways:

Highlighting a range means moving the cell pointer to the first cell
of the range and expanding the highlight to cover the whole range.

Typing a range address means typing the cell addresses of any two
diagonally opposite corners of a range and separating the cell addresses
by one or two periods, for example, A:A1..A:C5. If the range is in another
file, precede the range address with the file reference, for example,
<<SALES93.WK3>>A:A1..A:C5.

Using a range name means entering the name you assigned to a range using
/Range Name Create or Labels. If the range is in another file, precede
the range name with the file reference, for example, <<SALES93.WK3>>MARCH.
To list all range names in the current file, press F3 (NAME). You can specify
a range name from this list by highlighting the name and pressing ENTER.
