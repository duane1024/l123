/Data External Delete

/Data External Delete -- Permanently removes an external table from an
external database. You cannot delete an external table if

    The administrators of the external database from which you want to
   delete a table have restricted access to the database.
    The DataLens driver does not allow you to delete external tables.

1. Select /Data External Delete.
2. Specify the name of the DataLens driver for the external database
    containing the table you want to delete.
3. Specify the name of the external database containing the table you
    want to delete.
4. Specify the name of the external table you want to delete.
5. Select No to return to the /Data External menu without deleting the
    table or Yes to delete the table.
