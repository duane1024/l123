@Function Introduction

@Function Introduction -- The 1-2-3 @functions are built-in formulas that
perform specific database, date-and-time, financial, logical, mathematical,
statistical, scientific, or string calculations. You can use @functions
alone or as part of other formulas.

The @function syntax is @FUNCTION(argument1,argument2,...,argumentn).

@FUNCTION, the @function name, tells 1-2-3 which calculation to perform.
Arguments are the data 1-2-3 uses to perform the calculation. The
arguments can be any length, providing the total number of characters
in the cell that contains the @function does not exceed 512.

To enter an @function, type the @function name, enclose all arguments
in parentheses, and press ENTER. If an @function does not require an
argument, just enter the @function name without parentheses in a cell.

NOTE  In Help, optional arguments are enclosed in [ ] (brackets). All other
arguments are required.
