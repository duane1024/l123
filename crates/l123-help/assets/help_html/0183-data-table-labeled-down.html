/Data Table Labeled Down

/Data Table Labeled Down -- Specifies the row variable range and input
cells. The row variable range is a range that contains rows of input
values, organized by columns. A row variable range can be one or more
columns, as long as each column contains a separate set of input values.

NOTE  No part of a row variable range can be in a row or column that also
contains part of another variable range or formula label range.

1. Select /Data Table Labeled Down if you have a row variable range.
2. Specify the row variable range.
    l123 displays the address of the leftmost column of input values in
    the row variable range. Press ENTER to identify this set of input
    values as the set to which you will next assign an input cell.
3. Specify the input cell for that column of input values. If you entered
    a row variable range that contains more than one column, l123 repeats
    the prompts for assigning input cells for each additional column of
    input values, moving from left to right.
