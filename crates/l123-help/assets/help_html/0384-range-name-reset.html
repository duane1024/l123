/Range Name Reset

/Range Name Reset -- Deletes all defined range names in the current file
(and their notes, if they have notes) but leaves the data in the named
ranges unchanged. /Range Name Reset does not delete range names assigned
to external tables with /Data External Use.

1. Make sure the cell pointer is in the file whose range names you want
    to delete.
2. Select /Range Name Reset.

In formulas that use any of the deleted range names, l123 replaces
the range names with the corresponding range addresses. For example,
if SALES defined the range B14..H14 and you delete SALES, l123 changes
the formula @SUM(SALES) to @SUM(B14..H14).

Use /Range Name Delete to delete one range name in the current file.
