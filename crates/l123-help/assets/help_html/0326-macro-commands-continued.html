Macro Commands (continued)

    {FORMBREAK}
        Ends a {FORM} command. Macro execution continues at the macro
        instruction that immediately follows the {FORM} command.

        CAUTION  Use {FORMBREAK} only within a call-table subroutine.
        Using it anywhere else causes the macro to terminate with an error.

    {FRAMEOFF}
        Suppresses display of the worksheet frame (worksheet letter, column
        letters, and row numbers).

    {FRAMEON}
        Redisplays the worksheet frame hidden by {FRAMEOFF}.
