/Data Sort

/Data Sort -- Arranges the data in a range in the order you specify. For
example, use /Data Sort to alphabetize the records in a database table.

Data-Range     Selects the range you want to sort.
Primary-Key    Determines the primary field for ordering the rows in the
               range or the records in the database table. (This field is
               called the primary sort key.)
Secondary-Key  Selects the field that determines the order for records or
               rows with identical primary sort-key entries. For example, if
               the NAME field is the primary sort key and the SALES field is
               the secondary sort key, 1-2-3 sorts any records with the same
               NAME by SALES entries.
Extra-Key      Defines all sort keys after the primary and secondary keys.
Reset          Clears all /Data Sort settings.
Go             Sorts the range according to the current selections and returns
               1-2-3 to READY mode.
Quit           Returns 1-2-3 to READY mode without sorting rows or records.
