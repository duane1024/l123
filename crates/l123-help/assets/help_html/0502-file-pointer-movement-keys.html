File Pointer-Movement Keys

File Pointer-Movement Keys -- Move the cell pointer between active files.

CTRL-END CTRL-PG DN               Moves to the cell you last highlighted
                                   in the previous active file.

CTRL-END CTRL-PG UP               Moves to the cell you last highlighted
                                   in the next active file.

CTRL-END END                      Moves to the cell you last highlighted
                                   in the last active file.

CTRL-END HOME                     Moves to the cell you last highlighted
                                   in the first active file.
