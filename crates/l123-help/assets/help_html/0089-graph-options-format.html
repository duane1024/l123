/Graph Options Format

/Graph Options Format -- Sets the way 1-2-3 displays lines in line, mixed,
HLCO, and XY graphs.

1. Select /Graph Options Format.
2. Select Graph to format all data ranges at once or A, B, C, D, E, or F
    to format one data range.
3. Select one of the following:
    Lines       Displays connecting lines between the points.
    Symbols     Displays a symbol at each point along the line.
    Both        (Default) Displays both connecting lines and symbols.
    Neither     Displays neither connecting lines nor symbols, hiding the
                line in the graph unless you created data labels for it
                with /Graph Options Data-Labels.
    Area        Fills the space between the line and the line below it
                (or if there is no line below it, between the line and
                the x-axis or zero line) with colors or hatch patterns.
4. Select Quit to return to the /Graph Options menu.
