/Move

/Move -- Transfers a range of data, including cell formats and protection
status, to another range in the same file. You can use /Move to reorganize
data within a file, but you cannot use /Move to transfer data between files.

1. (Optional) If you want to highlight the range from which you want to
    move data, move the cell pointer to the first cell in the range.
2. Select /Move.
3. Specify the range you want to move FROM.
4. Specify the range you want to move TO. You need to specify only
    the first cell in the TO range.

    CAUTION  Make sure the TO range does not contain data you need, because
    l123 replaces the existing data with the data you move. Moving data into
    or out of the first or last cell of a range that is named or referred
    to in a formula changes the range definition.
