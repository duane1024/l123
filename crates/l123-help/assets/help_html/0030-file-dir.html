/File Dir

/File Dir -- Changes the directory that 1-2-3 uses when you save, read, or
list files. /File Dir overrides the /Worksheet Global Default Dir setting
for the current session. For information on creating directories,
see your DOS documentation.

1. Select /File Dir.
2. Press ENTER to accept the current directory, or type a different
    directory and press ENTER.

For example, to change the directory from C:\123 to C:\EXPENSES, type
c:\expenses and press ENTER.

Use /Worksheet Global Default Dir to change the default directory for
future sessions.
