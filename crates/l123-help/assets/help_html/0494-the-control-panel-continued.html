The Control Panel (continued)

The second line of the control panel displays

       The characters you are typing or editing if you are entering or
      editing data.
       The l123 menu if l123 is in READY mode and you press / (slash).
       The prompt if you select a command in which l123 prompts you for
      information.

The third line of the control displays

       A submenu or a description of the highlighted command when l123
      displays a menu in the second line.
       A list of names if you select a command that prompts you for the
      name of a file, range, graph, or settings sheet.
