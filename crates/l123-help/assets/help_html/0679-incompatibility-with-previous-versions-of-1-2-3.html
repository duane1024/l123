Incompatibility with previous versions of l123

The following conditions could have caused this error:

* You could not save a l123 Release 3 file as a .WK1 file because the
   file contained more than one worksheet. Use the Translate utility
   to convert the file to .WK1 format. For information on Translate,
   see Appendix A of Reference.

* You could not save a l123 Release 3 file as a .WK1 file because the
   file was sealed. Use /File Admin Seal Disable to unseal the file
   before you try to save it. To unseal the file, you must know the
   password that was used to seal the file.
