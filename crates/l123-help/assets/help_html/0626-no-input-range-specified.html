No input range specified

The following conditions could have caused this error:

* You tried to perform a data query operation without specifying an input
   range. Use /Data Query Input to specify an input range, and repeat the
   data query operation by pressing F7 (QUERY) or selecting a Data Query
   command.

* You tried to parse a column of long labels without specifying
   an input column. Select /Data Parse Input-Column and specify the column of
   labels you want to parse, and then select Go.
