5. Press ENTER to accept the default range name (the table name), or type a
    new range name for the table and press ENTER.

    NOTE  If the table name starts with $ or !, contains a . (period), or
    could be interpreted as a cell address, 1-2-3 does not display a default
    range name for the table. You must type a range name and press ENTER.

6. Press ENTER, or, if necessary, specify a table creation string.

    NOTE  A table creation string is a piece of information an
    external database uses to create a new external table. Not all external
    databases require you to enter a table creation string. If you
    do not know whether you need to enter a table creation string, open
    DATALENS.WK3 and see the worksheet containing information about the
    DataLens driver you will use.
