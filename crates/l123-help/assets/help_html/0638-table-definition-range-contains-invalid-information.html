Table definition range contains invalid information

You tried to use /Data External Create Go to create an external table,
and the table definition contains a data type, field width, column
label, field description, and/or field creation string that the
DataLens driver does not recognize.

Check the spelling and/or format of the entries in the table definition,
make the necessary changes, and then select /Data External Create Name and
Go to create the table.

For information on the data types, field widths, column labels, field
descriptions, and field creation strings you can use in a table
definition, open the DATALENS.WK3 online document.
