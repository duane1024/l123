Invalid string argument

You were running a macro and one of the following problems occurred:

* For an {OPEN} command, the access-type argument was an empty string ("").
   You must use the letter r, w, m, or a (or a reference to a cell that
   contains one of those letters) as the access-type argument.

* For a macro command that requires a string argument, the
   argument was a number, numeric formula, or reference to a cell that
   contains a number or numeric formula.

* For a {DISPATCH} command, the range name used as the location argument
   was undefined or did not exist, or the specified location was blank or
   contained a numeric value.
