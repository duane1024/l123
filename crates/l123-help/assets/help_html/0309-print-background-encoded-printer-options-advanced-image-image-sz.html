/Print [Background, Encoded, Printer] Options Advanced Image Image-Sz

/Print [Background, Encoded, Printer] Options Advanced Image Image-Sz -- Sets
the size and shape of a printed graph. The default graph shape is a rectangle
with a width-to-length ratio of four to three -- for every four units in
width, there are three units in length.

1. Select /Print [B,E,P] Options Advanced Image Image-Sz.
2. Select Margin-Fill (default), Length-Fill, or Reshape.
    Margin-Fill  Without changing the default graph shape, sizes a graph to
                 fill the page between the left and right margins.
    Length-Fill  Without changing the default graph shape, creates the
                 largest graph possible using the length you enter.
    Reshape      Sizes a graph according to the width and length you enter,
                 overriding the default graph shape.
3. If you selected Length-Fill, enter the graph length (the number of
    standard lines long).
    If you selected Reshape, enter the graph width (the number of standard-
    size characters wide) and length (the number of standard lines long).
