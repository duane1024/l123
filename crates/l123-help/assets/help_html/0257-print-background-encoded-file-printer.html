/Print [Background, Encoded, File, Printer]

/Print [Background, Encoded, File, Printer] -- Selects the destination for
printing data and graphs and lets you use the following print options:

Range        Specifies the range you want to print.
Line         Produces a blank line on the printer or in a file.
Page         Advances the paper in the printer to the top of the next page or
             inserts blank lines in a file.
Options      Selects and names the format settings for the print job.
Clear        Resets some or all current print settings.
Align        Resets the page number to 1.
Go           Prints the specified range or graph.
Image        Selects the graphs you want to print.
Hold         Returns l123 to READY mode without closing the print job.
Quit         Returns l123 to READY mode, closing the print job.
