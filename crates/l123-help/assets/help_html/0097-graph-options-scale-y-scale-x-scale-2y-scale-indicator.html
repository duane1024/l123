/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Indicator

/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Indicator -- Sets whether
1-2-3 displays a scale indicator for the selected scale and, if so, the
contents of the indicator. 1-2-3 displays the scale indicator parallel to the
corresponding axis. You can use it either to clarify the scale's order
of magnitude or to display other text.

1. Select /Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Indicator.
2. Select Yes, None, or Manual.
    Yes      (Default) 1-2-3 uses the scale indicator to report the
             scale's order of magnitude. For example, when the scale's
             order of magnitude is 10E+3 (10 to the third power), 1-2-3
             displays Thousands in the scale indicator.
    None     1-2-3 suppresses display of the scale indicator.
    Manual   1-2-3 displays the text you specify for the scale indicator.
3. If you selected Manual, specify the indicator text either by typing
    it and pressing ENTER or by pressing \ (backslash) and specifying a
    cell that contains the text. 1-2-3 truncates text that does not fit in
    the allotted area.
