Numeric Formats for /Range Format, /Worksheet Global Format, and /Graph
Options Scale [Y, X, 2Y] Format -- If you selected a numeric format
other than General, specify the number of decimal places (0 through 15)
you want 1-2-3 to display.

Fixed            Displays numbers with up to 15 decimal places, a minus sign
                  for negatives, and a leading zero for decimal values.
Sci              Displays numbers in scientific (exponential) notation.
Currency         Displays numbers with a currency symbol and thousands
                  separators.
, (Comma)        Displays numbers with thousands separators.
General          Displays numbers without thousands separators (default).
Percent          Displays numbers as percentages (that is, multiplied by
                  100) with a percent sign.

NOTE  Use /Worksheet Global Default Other International if you want to
change the settings for the currency symbol and its position, the thousands
separator, and the decimal point symbol.
