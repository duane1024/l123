/Worksheet Global Default Ext

/Worksheet Global Default Ext -- Sets the extension l123 automatically uses
when you select a File command that lists or saves worksheet files.

1. Select /Worksheet Global Default Ext.
2. Select List or Save.

    List  Sets the extension l123 uses when you select /File Combine,
          Erase, List, Open, or Retrieve. The initial List setting is WK*,
          which means l123 will list all files whose extensions begin
          with WK, such as .WK3, .WK1, and .WKS.
    Save  Sets the extension l123 uses when you select /File New, Save,
          or Xtract. The initial Save setting is WK3.

3. Type an extension and press ENTER.
4. To update the l123 configuration file (123R34.CNF) so l123 uses the
    new Ext settings in future sessions, select /Worksheet Global
    Default Update.
