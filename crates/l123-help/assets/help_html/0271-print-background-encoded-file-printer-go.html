/Print [Background, Encoded, File, Printer] Go

/Print [Background, Encoded, File, Printer] Go -- Sends your data to a printer
or file.

1. Specify a print range (/Print [B,E,F,P] Range) or a graph
    (/Print [B,E,P] Image).
2. If you are printing on a printer, make sure the printer is online and
    the paper is at the top of a page.
3. If you are beginning a new print job, select /Print [B,E,F,P] Align.
4. Select Go.
5. (Optional) Print additional ranges or graphs with different options
    as part of the same print job by changing your options and selecting
    the new ranges or graphs. Then select Go again.
6. Select Quit to close the print job and return 1-2-3 to READY mode.
