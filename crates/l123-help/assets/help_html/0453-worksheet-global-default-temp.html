/Worksheet Global Default Temp

/Worksheet Global Default Temp -- Sets the directory in which 1-2-3
saves temporary files when it saves them on disk.

Temporary files, which have the extension .TMP, are files 1-2-3 creates during
operations such as printing. 1-2-3 stores a temporary file on disk when
the file is too large to store in available memory. 1-2-3 deletes all
temporary files when you select /Quit to end the 1-2-3 session.

1. Select /Worksheet Global Default Temp.
2. Specify the new temporary directory.

    NOTE  Make sure the directory you specify is not a read-only directory;
    otherwise, 1-2-3 will display an error message when it attempts to create
    temporary files.

3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the new
    Temp setting in future sessions, select /Worksheet Global Default Update.
