/Worksheet Global Default Temp

/Worksheet Global Default Temp -- Sets the directory in which l123
saves temporary files when it saves them on disk.

Temporary files, which have the extension .TMP, are files l123 creates during
operations such as printing. l123 stores a temporary file on disk when
the file is too large to store in available memory. l123 deletes all
temporary files when you select /Quit to end the l123 session.

1. Select /Worksheet Global Default Temp.
2. Specify the new temporary directory.

    NOTE  Make sure the directory you specify is not a read-only directory;
    otherwise, l123 will display an error message when it attempts to create
    temporary files.

3. To update the l123 configuration file (123R34.CNF) so l123 uses the new
    Temp setting in future sessions, select /Worksheet Global Default Update.
