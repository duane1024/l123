/Worksheet Global Default Other Undo

/Worksheet Global Default Other Undo -- Turns the undo feature on or off.
Turning undo off increases available memory and the speed at which l123
recalculates formulas and executes commands.

NOTE  For information about the undo feature, see "Undoing an Action"
in Chapter 1 of Reference.

1. Select /Worksheet Global Default Other Undo.
2. Select Enable to turn undo on or Disable (default) to turn
    undo off.
3. To update the l123 configuration file (123R34.CNF) so l123 uses the
    new undo setting in future sessions, select /Worksheet Global Default
    Update.
