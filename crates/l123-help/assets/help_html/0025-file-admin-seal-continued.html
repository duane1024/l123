/File Admin Seal (continued)

1. Select /File Admin Seal and select one of the following:
    File                 Seals the current file and the reservation setting.
    Reservation-Setting  Seals only the reservation setting for the current
                         file.
    Disable              Unseals the current file and/or the reservation
                         setting.
2. Type a password and press ENTER. A password can include up to 15
    characters. 1-2-3 is case-sensitive for passwords.
3. If you selected File or Reservation-Setting, type the same password
    again and press ENTER.

    CAUTION  Remember your password. You must use the same password to
    unseal the file or its reservation setting.
