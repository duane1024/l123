Key column outside Sort range

You used /Data Sort Primary-Key, Secondary-Key, and/or Extra-Key to
specify one or more sort-key fields that were not in the data range you
were sorting.

Select /Data Sort Data-Range and specify a data range that
includes the fields you want to sort by, or select /Data Sort Primary-Key,
Secondary-Key, and/or Extra-Key and specify sort-key fields that are in
the data range you are sorting.
