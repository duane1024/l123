/Print [Background, Encoded, Printer] Options Setup

/Print [Background, Encoded, Printer] Options Setup -- Lets you use a setup
string to specify printer attributes not available on the l123 menus, such as
underlining and double-strike mode, if the attributes are available on
your printer.

A setup string is a series of characters preceded by a \ (backslash)
that gives instructions to a printer. For example, the setup string
\027G instructs an Epson printer to print data in double-strike mode.
Check your printer manual for a list of printer control codes, and then
see Appendix C in Reference to learn how to convert printer control
codes into l123 setup strings.

NOTE  Do not use setup strings and print commands together for printer
attributes. Use either all menu commands or all setup strings.
1. Select /Print [B,E,P] Options Setup.
2. Type \ (backslash) and a setup string (up to 512 characters) and press
    ENTER. To change a displayed setup string, press ESC and enter a new
    string.
