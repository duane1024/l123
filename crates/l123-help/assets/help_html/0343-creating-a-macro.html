Creating a Macro

Creating a Macro -- A macro is a series of 1-2-3 commands and keystrokes
that defines a 1-2-3 task. Creating a macro involves the following steps:

Locating      Choose a location for the macro.
Entering      Enter the macro by typing the instructions directly in
               the worksheet or by using the 1-2-3 record feature.
Naming        Assign a range name to the macro.
Documenting   Enter the macro range name next to the first cell of the macro
               and add comments about what each line of the macro does.
Running       Use the macro to see if it works as intended.
Debugging     If the macro does not produce the results you want it to,
               find and edit the macro instructions. If necessary, use
               STEP mode to help you debug the macro.
Saving        Save the macro in a file for future use.
