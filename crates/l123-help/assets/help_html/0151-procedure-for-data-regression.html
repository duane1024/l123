Procedure for /Data Regression

1. Select /Data Regression.
2. Select X-Range and specify the independent variables.
    The values for each variable must be in a column, and the columns
    must be adjacent to one another.
3. Select Y-Range and specify the dependent variable.
    The Y range must be a single column and must have the same number of
    rows as the X range.
4. (Optional) Select Intercept and select either Compute (to calculate
    the y-axis intercept) or Zero (to use zero as the y-axis intercept).
    Do not select Zero unless your data is such that, if all the
    independent variables equaled zero, the dependent variable would
    also equal zero.
5. Select Output-Range and specify the output range.
6. Select Go to calculate the regression.
7. Select Quit to return 1-2-3 to the /Data menu.
