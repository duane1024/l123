/Worksheet Global Default Printer Interface

/Worksheet Global Default Printer Interface -- Identifies the printer port
through which your printer and computer are connected.

1. Select /Worksheet Global Default Printer Interface.
2. Select one of the following:
    1 (Parallel 1), 2 (Serial 1), 3 (Parallel 2), 4 (Serial 2),
    5 (Output Device LPT1), 6 (Output Device LPT2), 7 (Output Device LPT3),
    8 (Output Device COM1), 9 (Output Device COM2). If you selected 2 or 4
    (Serial), select a baud rate from the menu, and set the printer to accept
    8 data bits, 1 stop bit (or 2 stop bits at 110 baud), and no parity.
    See your printer manual for instructions on making these settings.
    Select 5, 6, 7, 8, or 9 if your printer is on a local area network.
3. To update the l123 configuration file (123R34.CNF) so l123 uses the new
    Interface setting in future sessions, select /Worksheet Global
    Default Update.

The corresponding /Print command is /Print [B,E,F,P] Options Advanced Device
Interface.
