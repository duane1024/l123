Saving Files in l123 Release 2 (.WK1) Format

Saving Files in l123 Release 2 (.WK1) Format -- To save a file in Release 2
(.WK1) format, specify .WK1 as the extension when you save the file. You
cannot save a .WK3 file as a .WK1 file under the following conditions:

    If the file contains more than one worksheet, use the Translate utility.
   For information on Translate, see Appendix A of Reference.
    If the file is sealed, unseal it with /File Admin Seal Disable before
   you try to save it.

If the file contains other .WK3 features, l123 saves the file and
displays an error message stating that some information was lost. If you
do not want to lose the information permanently, save the file as a
.WK3 file.

CAUTION  Symphony cannot properly calculate Release 3 @functions and
@functions that have new arguments. If you want to use a .WK3 file
with Symphony, use the Translate utility.
