Creating an Automatic Graph (continued)

1. Make sure the data you want to graph is organized as described on the
    previous screen.
2. If necessary, change the current /Worksheet Global Default Graph setting
    to match the orientation of the graph data (by columns or by rows).
3. If necessary, select /Graph Reset Ranges to clear any existing /Graph X
    or A - F settings.
4. Use /Graph Type to select the type of graph you want to create.
    The default graph type is Line.
5. (Optional) Use /Graph Options and /Graph Type Features to add titles,
    legends, and other enhancements to the automatic graph.
6. Place the cell pointer anywhere within the range of data you want
    to graph.
7. Select /Graph View or press F10 (GRAPH) to display the automatic graph.
8. Press any key to redisplay the worksheet.
