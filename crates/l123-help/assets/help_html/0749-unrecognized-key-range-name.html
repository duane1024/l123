Unrecognized key/range name

You were running a macro and one of the following problems occurred:

* l123 did not recognize the first word after an open brace as a macro
   key name, macro command keyword, or subroutine range name.

   Check for these problems: An incorrectly spelled key name, keyword,
   or subroutine range name; a missing space between a macro
   command keyword and the command's first argument, or between a
   key name and the repetition number; or an undefined or nonexistent
   subroutine range name.

* For a {FORM} command, the include list, the exclude list, or the
   first column of the call table included something that l123 did not
   recognize as a keyname (for example, a macro command).
