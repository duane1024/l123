Too many open files

You had too many files in memory with reservations when you used a File,
Graph, Print, or Data command that creates or opens files.

Save your work, and either use /Worksheet Delete File to delete some
active files or use /File Admin Reservation Release to release the
reservation for one or more active files.

NOTE  The number of open files (active files with reservations) you
may have at one time depends on DOS. If you are using 1-2-3 with a
version of DOS that is earlier than DOS 3.3, make sure the FILES= command
in your CONFIG.SYS file is FILES=20. This will let 1-2-3 open a maximum
of 13 active files with reservations.
