Invalid field name

The following conditions could have caused this error:

* You tried to perform a data query operation when the first row of the
   output range was blank. Copy some or all of the field names in the input
   range to the first row of the output range.

* You tried to perform a data query operation when the first row of the
   input range was blank. Enter field names in the first row of the range.

* You tried to perform a data query operation when none of the field names
   in the first row of the output range matched a field name in the input
   range. Copy some or all of the field names in the input range to the
   first row of the output range.
