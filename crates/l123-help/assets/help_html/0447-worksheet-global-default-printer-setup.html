/Worksheet Global Default Printer Setup

/Worksheet Global Default Printer Setup -- Lets you enter a setup string,
which is a series of characters that controls printing. For information
about setup strings, see Appendix C of Reference.

CAUTION  To avoid complications when printing, do not use setup
strings to control printer settings that you can control by using
Print [B,E,F,P] Options and Worksheet Global Default Printer commands.

1. Select /Worksheet Global Default Printer Setup.
2. Enter a setup string of up to 512 characters in the form \nnn\nnn
    (a backslash followed by a three-digit decimal number).
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses
    the new Setup setting in future sessions, select /Worksheet Global
    Default Update.

The corresponding /Print command is /Print [B,E,F,P] Options Setup.
