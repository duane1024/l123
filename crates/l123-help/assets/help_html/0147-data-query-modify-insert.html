/Data Query Modify Insert

/Data Query Modify Insert -- Adds the records from the output range to the
bottom of the input range.

1. Before you use /Data Query Modify Insert, you must specify the input,
    criteria, and output ranges. The first row of the output range must
    contain field names, and the remaining rows must contain data.
    Use /Data Query Output to make sure that the output range includes
    all the records you want to insert in the input range.

    CAUTION  Do not enter any data below the input range. When 1-2-3
    expands the input range to accommodate the new records, it could
    write over data entered just below the input range.

2. Select /Data Query Modify Insert.
