/Print [Background, Encoded, File, Printer] Hold

/Print [Background, Encoded, File, Printer] Hold -- Lets you leave a print job
open and returns l123 to READY mode so you can make changes to the worksheet
and then return to the /Print [B,E,F,P] menu to continue the same print job,
or open a new worksheet file and then continue the same print job.
/Print [Background, Encoded, File, Printer] Hold does not stop your printer if
it is already printing.

1. Select /Print [B,E,F,P] Hold.

    l123 leaves a print job open until you do one of the following:

       Return to the /Print [B,E,F,P] menu to complete the print job.
       Cancel a background print job (/Print Cancel).
       Select a different printer or port (/Print [B,E,P] Options Advanced
      Device Name or Interface).
       Select a different print destination (Background, Encoded, File, or
      Printer).
       Press CTRL-BREAK.
