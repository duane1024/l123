File contention prevented accessing the file

The following conditions could have caused this error:

* You tried to read or save a file at the same time that someone else was
   using the file. This error can occur during /File Combine, /File Open,
   /File Retrieve, /File Save, or /File Xtract operations. Wait a few
   minutes and then try to complete the operation.

* You used a file name that is reserved by DOS when you
   were saving or extracting a file. For a list of reserved file names,
   see your DOS documentation. Save the file with a different name.
