The /X Commands (continued)

    /XNprompt~[location]~
        Displays a prompt (which can be as long as the screen width) in
        the control panel and pauses for your response. When you type a
        numeric response and press ENTER, 1-2-3 enters your response as a
        number in location. If you do not specify a location (/XNprompt~~),
        1-2-3 enters your response in the current cell. /XN produces the same
        result as {GETNUMBER}.

    /XQ
        Ends the macro and returns 1-2-3 to READY mode. /XQ produces the
        same result as {QUIT}.

    /XR
        Ends a subroutine called by a {subroutine}, /XC, or {MENUCALL}
        command. Starts the next repetition of a subroutine called by a
        {FOR} command. /XR produces the same result as {RETURN}.
