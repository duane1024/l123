/File Import

/File Import -- Reads data from a text file on disk into the current
worksheet beginning at the current cell.

CAUTION  As a precaution, save the current file before using /File Import.

1. Move the cell pointer to an area of the worksheet large enough for
    all of the imported data.
2. Select /File Import and select one of the following:
    Text      Imports each line of data as a long label and enters it in
              one cell in the worksheet. 1-2-3 enters each successive line
              of data in the same column below the cell pointer. If a line
              of text is longer than 511 characters, 1-2-3 truncates the
              line. Select Text to import both text and numbers from a
              text file that is not in delimited format.
    Numbers   Imports labels and numbers from a delimited text file and
              enters them in separate cells in the worksheet. When a text
              file is not delimited, Numbers imports only numbers.
