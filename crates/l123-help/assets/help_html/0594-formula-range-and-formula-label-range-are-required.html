Formula range and formula label range are required

The following conditions could have caused this error:

* You tried to create a labeled table without specifying a formula
   range and formula label range. Use /Data Table Labeled Formulas to
   specify the formula and formula label range, and then select /Data Table
   Labeled Go to create the table.

* You tried to create a labeled table without specifying any of
   the necessary ranges. Use /Data Table Labeled Formulas and /Data Table
   Labeled Down, Across, and/or Sheets to specify the ranges for the
   table, and then select Go to create the table.
