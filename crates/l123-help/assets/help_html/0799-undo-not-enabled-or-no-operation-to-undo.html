Undo not enabled, or no operation to undo

One of the following conditions could have caused this error:

* You invoked the undo feature, but undo is currently turned off.
   To turn undo on so you can use the feature for the rest of the 1-2-3
   session, select /Worksheet Global Default Other Undo Enable. To have
   1-2-3 start with the undo feature automatically on in future sessions,
   select Update from the /Worksheet Global Default menu after selecting
   /Worksheet Global Default Other Undo Enable.

* You invoked the undo feature after clearing a 'Memory full' error message.
   When the memory full error occurred, 1-2-3 erased the undo history,
   so you cannot undo operations you performed prior to that error.

* You invoked the undo feature without having performed an undoable
   operation either since the last time you invoked the undo feature or
   since 1-2-3 last erased the undo history.
