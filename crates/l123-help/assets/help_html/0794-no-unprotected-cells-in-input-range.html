No unprotected cells in Input range

The following conditions could have caused this error:

* The input range you specified for /Range Input contained no
   unprotected cells. Use /Range Unprot to unprotect some of the cells
   in the input range before using /Range Input.

* The range you used as the {FORM} input-location argument in a macro
   contained no unprotected cells. Use /Range Unprot to unprotect
   some of the cells in the range before running the macro.
