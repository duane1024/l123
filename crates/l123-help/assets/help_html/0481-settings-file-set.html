Settings File Set

Settings File Set  -- Specifies an add-in file to read into memory
automatically each time 1-2-3 reads the current worksheet file into memory.
If the add-in file contains an application, Set also lets you tell 1-2-3 to
run the application automatically when 1-2-3 reads the worksheet file into
memory.

1. Select Settings File Set and specify an add-in file to read into memory
    automatically with the current worksheet file. 1-2-3 adds the name of the
    add-in file to the settings sheet.

2. If the add-in file contains an application, 1-2-3 prompts you to select
    Yes to start this application each time 1-2-3 reads the worksheet file
    into memory, or No if you do not want 1-2-3 to start the application
    automatically.

    NOTE  You can invoke only one application automatically with each
    worksheet file, although you can automatically load up to 15 add-in files.
