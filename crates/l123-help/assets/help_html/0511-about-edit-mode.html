About EDIT Mode

EDIT mode lets you change the data in a cell. You can switch 1-2-3 to EDIT
mode by pressing F2 (EDIT) when you are typing an entry or after you
complete the entry.

    To switch 1-2-3 to EDIT mode, press F2 (EDIT) while you are typing
   a label, number, or formula.
    To edit a completed entry, move the cell pointer to the cell that
   contains the entry you want to edit and press F2 (EDIT).
    1-2-3 changes to EDIT mode when you make an entry that 1-2-3 cannot
   accept. It places the cursor at the point where it detects an error.
    1-2-3 changes to EDIT mode when you enter information at a prompt.

To change data when 1-2-3 is in EDIT mode, use the editing keys.
