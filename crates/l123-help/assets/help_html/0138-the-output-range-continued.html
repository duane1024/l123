The Output Range (continued)

       If the input range contains more than one named database table,
      and the tables have duplicate field names, you must enter the table
      name with the field name separated by a . (period). For example,
      SALES92.QTR1 refers to QTR1 (the field) in the table named SALES92, and
      SALES93.QTR1 refers to QTR1 (the field) in the table named SALES93.

       You can enter one or more formulas to create computed columns in
      the output range. A computed column is a column in which 1-2-3
      calculates values for a group of records. You can use any type of
      formula except @@, @CELL, @COLS, @HLOOKUP, @INDEX, @INFO, @IRR, @N,
      @NPV, @ROWS, @S, @SHEETS, @STD, @STDS, @VAR, @VARS, @VLOOKUP, and the
      database @functions to create computed columns. For example, if you
      enter the formula +SALES*0.07 in the first row of the output range,
      1-2-3 will multiply the values in SALES times .07 when it enters the
      records in the output range. For more information, see "The Output
      Range" in Chapter 3 of Reference.
