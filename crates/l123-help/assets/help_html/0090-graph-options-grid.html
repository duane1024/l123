/Graph Options Grid

/Graph Options Grid -- Adds or removes grid lines in a graph. All graph types
except pie charts can include grid lines.

1. Select /Graph Options Grid.
2. Select one of the following:

    Horizontal     Draws grid lines from the first and/or second y-axis,
                   depending on the current /Graph Options Grid Y-Axis
                   setting.
    Vertical       Draws grid lines from the x-axis.
    Both           Draws both horizontal and vertical grid lines.
    Clear          Clears all grid lines.
    Y-Axis         Determines whether horizontal grid lines originate from
                   the y-axis (default), the second y-axis, or both.

3. If you selected Y-Axis, select Y, 2Y, or Both, depending on whether
    you want horizontal grid lines to originate from the y-axis, the second
    y-axis, or both y-axes, respectively.
