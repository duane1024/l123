Table

Table -- Creates a one-column table in a worksheet that lists add-in
applications, @functions, or macro commands in memory.

1. Decide on a location for the table. The table can be in any active
    file. It will occupy one column and as many rows as there are add-in
    applications, @functions, or macro commands in memory, plus one blank row.
2. Select Table.
3. Select @Functions, Macros, or Applications.

    @Functions   Creates a table of all add-in @functions in memory.
    Macros       Creates a table of all add-in macro commands in memory.
    Applications Creates a table of all add-in applications in memory.

4. Specify the location you decided on in step 1 and press ENTER. You need
    to specify only the first cell of the table's location.

CAUTION  Make sure the table location does not contain data you need,
because l123 writes over existing data when it creates the table.
