Part of file is missing

The following conditions could have caused this error:

* The file you specified may have been truncated.
* The file you specified may be missing the end-of-file character.
* Your disk may be damaged.
* The disk-drive head may not be properly aligned.

Clear the error message by pressing ESC. Press ENTER. If possible,
1-2-3 will read some or all of the file into memory.
