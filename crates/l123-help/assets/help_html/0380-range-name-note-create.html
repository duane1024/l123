/Range Name Note Create

/Range Name Note Create -- Creates a note for a defined range name or lets
you edit an existing range name note.

1. Select /Range Name Note Create.
2. Specify the range name for which you want to create or edit a note.
3. Type or edit the range name note and press ENTER.

    The note can include up to 512 characters.

NOTE  To edit a note for an undefined range name, redefine the range name
with /Range Name Create or Labels, and then select /Range Name Note Create.
