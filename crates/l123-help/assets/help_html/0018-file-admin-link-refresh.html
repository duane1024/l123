/File Admin Link-Refresh

/File Admin Link-Refresh -- Recalculates formulas in active files that
refer to data in other active files or files on disk. Formulas linked
to other files do not automatically update when you read the files
that contain the formulas into memory. Use /File Admin Link-Refresh
to update linked formulas after you use /File Open or /File Retrieve.

1. Select /File Admin Link-Refresh.

NOTE  If /Worksheet Global Recalc is set to Manual, l123 does not
update links but displays the CALC indicator after you select /File
Admin Link-Refresh. Press F9 (CALC) to update the linked formulas in
the current file.

See Linking Files with Formulas for information on writing formulas
that refer to data in other files.
