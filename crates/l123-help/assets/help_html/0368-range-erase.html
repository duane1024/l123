/Range Erase

/Range Erase -- Erases the data in a range but leaves the cell format(s) for
the range intact.

1. Select /Range Erase.
2. Specify the range you want to erase.

l123 erases the data from the specified range, leaving the cells blank.
/Range Erase does not affect the cell format of the erased cells, nor
does it delete range names associated with the erased cells.

Note how /Range Erase differs from the following commands:

/Worksheet Delete Sheet erases selected worksheets within an active file.
/Worksheet Erase deletes all active worksheets and files from memory
and replaces them with one blank worksheet.
