LCF file access error (continued)

The following condition could have caused this error:

* The file LOTUS.LCF was damaged or deleted from your hard disk. You must
   recreate this file before you can print. To recreate this file, start the
   Install program from your hard disk, select First-Time Installation, and
   select your equipment. Then start l123 again and try to print. For
   information on starting Install, see "Starting Install After First-Time
   Installation" in Chapter 5 of Getting Started.
