/Print [Background, Encoded, File, Printer] Options Name

/Print [Background, Encoded, File, Printer] Options Name -- Creates, selects,
deletes, and lists print settings names. You assign a print settings name to
the current print settings so you can use the same settings as often as you
want. Print settings include the print range, a current or named graph,
and all settings you select with /Print [B,E,F,P] Options.

The Print [B,E,F,P] Options Name commands perform the following tasks:

Use          Makes the print settings assigned to the name you specify
             the current print settings.
Create       Assigns a name to the current print settings and modifies
             settings already named.
Delete       Deletes one print settings name.
Reset        Deletes all print settings names in the current file.
Table        Lists all print settings names in the current file.
