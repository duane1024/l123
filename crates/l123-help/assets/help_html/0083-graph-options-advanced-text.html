/Graph Options Advanced Text

/Graph Options Advanced Text -- Sets colors, fonts, and sizes for the text
in displayed and printed graphs.

1. Select /Graph Options Advanced Text.
2. Select one of the three graph-text groups:

    First        First line of graph title.
    Second       Second line of graph title, axes titles, and legends.
    Third        Graph notes, data labels, scale indicators, x-axis and
                 pie-slice labels, and scale numbers.

3. Then select one of the following:

    Color        Sets a color for the selected graph-text group.
    Font         Sets a font for the selected graph-text group.
    Size         Sets a size for the selected graph-text group.
    Quit         Returns you to the /Graph Options Advanced Text menu.
