Increasing Available Memory

Increasing Available Memory -- When you are using 1-2-3, there are several
factors that affect the amount of memory available for your files. These
factors include the size of your operating system and other programs in
memory, the size of the drivers you selected in the 1-2-3 Install program,
the type of data in your worksheets, and the structure of your worksheets.

To increase available memory, do one or more of the following:

    Save your active 1-2-3 file(s), use /Worksheet Erase, and then retrieve
   your file(s) again.
    If the undo feature is turned on, use /Worksheet Global Default Other
   Undo Disable to turn the undo feature off.
    Delete worksheets, files, and data that you no longer need in memory.
    If you have programs in memory in addition to 1-2-3, end one or
   more of those programs.

For more information on memory management, see Appendix D of Reference.
