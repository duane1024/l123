/Data Sort Procedure

1. Select /Data Sort.
2. Select Data-Range and specify the range you want to sort.
3. Select Primary-Key and specify the primary sort key and the sort order
    (ascending or descending). Ascending order is from A to Z if you are
    sorting labels or from smallest to greatest value if you are sorting
    values; descending order is from Z to A or from greatest to smallest
    value.
4. (Optional) Select Secondary-Key and specify the secondary sort key and
    the sort order.
5. (Optional) Select Extra-Key, assign a number to the extra sort key, and
    specify the sort order for the sort key.
6. (Optional) Repeat step 5 for each additional sort key you want to
    specify after the primary, secondary, and first extra sort keys.
7. Select Go to sort the range.
