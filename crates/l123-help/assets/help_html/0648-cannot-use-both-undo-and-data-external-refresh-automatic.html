Cannot use both undo and Data External Refresh Automatic

You tried to select /Data External Other Refresh Automatic when the
undo feature was on, or you tried to turn on the undo feature after you
selected /Data External Other Refresh Automatic. You cannot have
the undo feature turned on and /Data External Other Refresh set to
automatic at the same time.
