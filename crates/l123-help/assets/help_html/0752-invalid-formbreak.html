Invalid FORMBREAK

You were running a macro and l123 encountered a {FORMBREAK} command
somewhere other than within a {FORM} call-table subroutine.
{FORMBREAK} commands are allowed only within a {FORM} call-table
subroutine.

Remove the {FORMBREAK} command from the macro, or if it is within a
subroutine, either move the {FORMBREAK} command to the {FORM} call-table
subroutine or use the {BRANCH} or {DISPATCH} command to transfer control
from the {FORM} call-table subroutine to the subroutine location.
