/Range Value

/Range Value -- Copies a range of data, replacing any copied formulas
with their current values.

1. If the CALC indicator appears at the bottom of the screen, press
    F9 (CALC) to update formulas. If there are linked formulas in the
    range you want to copy, you also need to use /File Admin
    Link-Refresh to make sure the values are updated.
2. Select /Range Value.
3. Specify the FROM range (the range whose current values you want to
    copy).
4. Specify the TO range (the location for the copied values). You need to
    specify only the first cell of the TO range. If you are copying a
    three-dimensional range, be sure there are enough worksheets in the TO
    range to fit the copied data.

    CAUTION  Make sure you specify a TO range that does not contain data,
    because 1-2-3 writes over the existing data with data you copy. Formulas
    that acted on previous contents of the TO range now act on the new data.
