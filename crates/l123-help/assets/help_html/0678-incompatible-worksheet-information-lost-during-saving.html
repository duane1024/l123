Incompatible worksheet information lost during saving

1-2-3 saved a 1-2-3 Release 3 file in Release 2 (.WK1) format, but the
file contained Release 3 features that could not be saved in the file.

See Saving Files in 1-2-3 Release 2 (.WK1) Format for specific
information about what happens to Release 3 features when saved in a
.WK1 file.
