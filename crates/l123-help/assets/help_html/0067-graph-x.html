/Graph X

/Graph X -- Sets the X data range for a graph. The way l123 uses the X
data range depends on the graph type you select.

    In line, bar, stacked bar, mixed, and HLCO graphs, l123 uses the
   X data range as the x-axis labels.
    In pie charts, l123 uses the X data range as the pie-slice labels.
    In XY graphs, l123 uses the X data range as the set of values
   against which it plots the values in the A - F data ranges. The values in
   the X data range determine the x-axis scale.

1. Select /Graph X.
2. Specify the range you want to use as the X data range.
