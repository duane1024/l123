File already exists in memory

You used /File New, /File Open, /File Retrieve, /File Save, or /File Xtract
and specified a file that was already active. If you used /File Retrieve
or /File Save, the file was active but it was not the current file.

To replace an active file with the version of the file on disk, move
the cell pointer to the active file you want to replace and use /File
Retrieve. l123 deletes the active version of the file and reads the
file from disk into memory.
