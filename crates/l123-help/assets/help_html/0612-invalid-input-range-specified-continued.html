Invalid input range specified (continued)

* You tried to perform a data query operation when the input range was
   an external table, but the range name you specified with /Data Query
   Input was no longer in use. Use /Data External Use to reassign the range
   name to the external table you want to use as the input range, or use
   /Data Query Input to specify a range name that is currently in use.

* You specified a range in a file on disk, but the range was too large to
   fit in memory. Read the file containing the range into memory or specify
   a smaller range.
