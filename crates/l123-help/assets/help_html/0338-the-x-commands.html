The /X Commands

The /X Commands -- In addition to the macro commands, l123 includes
the /X ("slash X") commands. The /X commands were originally included
in l123 Release 1A and have been retained for compatibility. The table
below lists each /X command and the corresponding macro command.

    /X command                  Corresponding macro command

   /XClocation~                 {subroutine}
   /XGlocation~                 {BRANCH location}
   /XIcondition~                {IF condition}
   /XLprompt~[location]~        {GETLABEL prompt,location}
   /XM[location]~               {MENUBRANCH location}
   /XNprompt~[location]~        {GETNUMBER prompt,location}
   /XQ                          {QUIT}
   /XR                          {RETURN}
