/File Xtract

/File Xtract -- Extracts a range of data by copying the data from an active
file and saving it in a worksheet file on disk. /File Xtract saves all
settings associated with worksheets in the extracted file. Before you use
/File Xtract, note the following:

    You cannot create a Release 2 (.WK1) file by extracting data from a .WK3
   file if the file contains more than one worksheet. Before you extract data
   to a .WK1 file, see Saving Files in l123 Release 2 (.WK1) Format.
    You cannot extract data from a sealed file.
    If you are using a file and have the reservation, use /File Admin
   Reservation Release to release the reservation before you use /File Xtract.
    If you extract formulas, be sure to extract all the data that is referred
   to by the formulas. If you extract data in a named range, be sure to
   extract the entire named range.
    Extracting formulas with three-dimensional ranges to a file that contains
   fewer worksheets than are in the three-dimensional ranges may produce
   unexpected results.
