/Print [Background, Encoded, File, Printer] Line

/Print [Background, Encoded, File, Printer] Line -- Advances the paper in a
printer to the next line (/Print Printer Line or /Print Background Line)
or inserts one blank line in a file (/Print Encoded Line or /Print File Line).

Use /Print [B,E,F,P] Line to insert blank lines between print ranges or
graphs if you are printing several ranges or graphs in the same print job.

1. Select /Print [B,E,F,P] Line.
