/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Width

/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Width -- Sets the maximum
width of the numbers 1-2-3 displays along the selected scale.

1. Select /Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Width.
2. Select Automatic or Manual.
    Automatic    (Default) 1-2-3 chooses a maximum width for numbers along
                 the selected scale based on the format and exponent
                 settings for that scale.
    Manual       Lets you specify the maximum width for numbers along the
                 selected scale.
3. If you selected Manual, type a number from 1 through 50 and press ENTER.

If a number on the scale does not fit within the maximum width, 1-2-3
displays asterisks instead of the number. To display the number, change
the width and/or format settings for the scale.
