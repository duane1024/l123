/Print [Background, Encoded, File, Printer] Options Advanced Layout Pitch --
Determines how much space l123 allocates for each printed character, and
therefore, how many characters fit on a printed line.

l123 offers three pitch options: Standard, Compressed, and Expanded. The
actual character width for each of these options depends on your printer.
For example, on an Epson FX series printer, standard width is 10 characters
per inch (cpi), compressed width is 17 cpi, and expanded width is 5 cpi.

1. Select /Print [B,E,P] Options Advanced Layout Pitch.
2. Select Standard, Compressed, or Expanded.
    Standard    Returns printing to the default pitch for your printer.
    Compressed  Prints narrower characters and more characters per inch.
    Expanded    Prints wider characters and fewer characters per inch.
