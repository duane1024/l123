At least one variable range must be specified

You tried to create a labeled table without specifying a variable
range. You need at least one variable range to create a labeled table.

Use /Data Table Labeled Down, Across, and/or Sheets to specify a row,
column, and/or worksheet variable range, and the corresponding input cells.
Then select /Data Table Labeled Go to create the table.
