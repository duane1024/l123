/Graph Options Color and /Graph Options B&W

/Graph Options Color and /Graph Options B&W -- Sets graph display and
printing to color or black and white. Color is the default setting for
color monitors except CGA monitors. B&W is the default setting for
monochrome and CGA monitors.

1. Select /Graph Options Color or /Graph Options B&W.

When you select Color, l123 uses the default colors for data ranges and
graph text unless you set other colors with /Graph Options Advanced Color
and /Graph Options Advanced Text [F,S,T] Color.

When you select B&W, l123 uses default hatch patterns for data ranges
unless you set other hatch patterns with /Graph Options Advanced Hatches.
