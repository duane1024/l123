/Data Query Extract

/Data Query Extract -- Copies to the output range all records in the input
range or ranges that match the criteria you specified in the criteria range.

1. Before you can use /Data Query Extract, you must specify the input,
    criteria, and output ranges. The input range can be in any file,
    active or on disk.
2. Select /Data Query Extract.

    If the input and output ranges are in a 1-2-3 file, 1-2-3 formats the
   records in the output range as they are formatted in the input range.
   If a field in the input range contains formulas, 1-2-3 converts the
   formulas to values in the output range.
    If the input range is an external table and the output range is in a
   1-2-3 worksheet, 1-2-3 formats cells containing dates and times with
   Date and Time formats.
    If the output range is an external table, the DataLens driver
   formats the records using formats that the external table supports.
