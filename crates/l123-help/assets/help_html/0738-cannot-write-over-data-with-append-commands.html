Cannot write over data with APPEND commands

You were running a macro that uses an {APPENDBELOW} or {APPENDRIGHT}
command to copy data to the end of a range, and the cells to which the
data would be copied already contained data. l123 does not allow
{APPENDBELOW} and {APPENDRIGHT} commands to write over existing data.

Check the {APPENDBELOW} or {APPENDRIGHT} command in the macro to see
what range you specified as the target location. (The target-location
argument is the first argument in both of these commands.) Then either
erase that range or use a different target location for the command.
