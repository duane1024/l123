ALT-F2 (RECORD) Step

ALT-F2 (RECORD) Step -- Helps you debug a macro by executing the macro one
step at a time. To debug a macro in STEP mode, do the following:

1. With l123 in READY mode, press ALT-F2 (RECORD) and select Step to
    turn on STEP mode.
2. Start the macro.
3. Press a key (the space bar is recommended) to execute the first macro
    instruction.
4. Continue pressing a key to move step by step through the macro. When
    you find the problem, end the macro by pressing CTRL-BREAK. (If the
    macro produces an error message, press F1 (HELP) to see an explanation
    of the error and then press ESC twice to end the macro.)
5. Edit the macro instructions to correct the error.
6. If necessary, repeat the macro in STEP mode to locate other problems.
7. To turn off STEP mode so you can run the macro normally, press
    ALT-F2 (RECORD) in READY mode and select Step again.
