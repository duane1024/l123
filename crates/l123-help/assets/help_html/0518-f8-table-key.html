F8 (TABLE) Key

F8 (TABLE) Key -- Repeats the last Data Table command you selected.

/Data Table 1   Calculates the results of one or more formulas that use
                one variable.

/Data Table 2   Calculates the results of one formula that uses two
                variables.

/Data Table 3   Calculates the results of one formula that uses three
                variables.

/Data Table Labeled
                Calculates the results of one or more formulas that use one
                or more variables.
