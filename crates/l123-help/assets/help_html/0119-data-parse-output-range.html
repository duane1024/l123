/Data Parse Output-Range

/Data Parse Output-Range -- Specifies the range where you want l123 to enter
the parsed data.

1. Select /Data Parse Output-Range.
2. Specify the range where you want l123 to enter the parsed data.

    CAUTION  l123 writes over any existing data in the specified range,
    so make sure the output range does not contain any data you need.

l123 does not enter the parsed data in the output range until you select
/Data Parse Go.
