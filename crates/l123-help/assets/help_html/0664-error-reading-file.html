Error reading file

The following conditions could have caused this error:

* l123 cannot retrieve or open a blank file. l123 creates a blank
   file on disk when you use /File New, or when you use /File Retrieve or
   /File Open with a l123 Release 1A or Symphony file. The blank file
   acts as place holder on disk until you save the file. Use /File
   List to check the size of the files on disk to ensure that you are
   not trying to read a blank file into memory.

* l123 could not read the file from disk because the file may have been
   corrupted. Clear the error message by pressing ESC and then press ENTER
   to see if l123 has read any of the file.
