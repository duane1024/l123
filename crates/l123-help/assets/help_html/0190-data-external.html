/Data External

/Data External -- Lets you exchange data between l123 and external
tables. An external table is a table in a database maintained with a
database management program other than l123.

Use      Connects l123 to an external table.
List     Lists the names of tables in an external database or lists
         the names of fields in an external table.
Create   Sets up the structure for a new table in an external
         database.
Delete   Removes a table from an external database.
Other    Sends a command to a database management program, re-executes
         Data Query and Data Table commands and updates database @functions,
         and specifies a character set for the translation of data that l123
         copies to or extracts from an external table.
Reset    Disconnects an external table.
Quit     Returns l123 to READY mode.
