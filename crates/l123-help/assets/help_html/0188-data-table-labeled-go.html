/Data Table Labeled Go

/Data Table Labeled Go -- Calculates the results of the formula(s) and
enters them in the table.

Before using /Data Table Labeled, you must set up the ranges l123 uses
to create a labeled table. For detailed procedures and examples on
/Data Table Labeled, see /Data Table Labeled in Chapter 3 of Reference.

1. Select /Data Table Labeled Go.

l123 calculates the formulas, replacing each variable in each formula
with the input values in the corresponding variable ranges. l123 enters
the result of each calculation in the table, in the cell at the
intersection of the appropriate worksheet, row, and column.
