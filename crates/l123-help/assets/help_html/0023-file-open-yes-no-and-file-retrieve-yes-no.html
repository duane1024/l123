/File Open Yes/No and /File Retrieve Yes/No

When you use /File Open or /File Retrieve and l123 displays a Yes/No
menu, it means that someone else is using the file and has the file's
reservation. Select Yes to open or retrieve the file without the
reservation. Select No if you do not want to open or retrieve the file
without the reservation.

CAUTION  If you open or retrieve a file without the reservation, l123
displays the RO (read-only) indicator at the bottom of the screen. l123
will not let you save changes to the file using the same file name if
you do not have the reservation.

To make changes to the file, use /File Admin Reservation Get to try
to get the reservation, or try to read the file again later when the
reservation may be available. You cannot get the reservation until the
other person releases it or stops using the file.
