Graph error - not enough memory to draw graph

You tried to view a graph, but not enough memory was available for 1-2-3 to
display it. This is a background error message; to remove it, either
select /Graph Reset Graph (to reset all current graph settings) or view
a graph that 1-2-3 can display successfully. To do the latter, try any of
the following:

* Free some memory and then display the current graph. For information on
   freeing memory, see Increasing Available Memory.

* Set smaller A - F data ranges for the current graph, or use /Graph Reset
   A - F or Options to reset some of the graph's data ranges or its options
   settings, and then display the modified graph.

* Use /Graph Name Use to display another named graph that is less complex.
