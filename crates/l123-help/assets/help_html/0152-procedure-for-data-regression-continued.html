Procedure for /Data Regression (continued)

When you select Go, 1-2-3 enters the following in the output range:

Constant                  The y-axis intercept.
Std Err of Y Est          The standard error of the estimated y value.
R Squared                 The reliability of the regression.
No. of Observations       The number of rows of data in the X and Y ranges.
Degrees of Freedom        The number of observations minus the number of
                           independent variables minus 1 (calculated
                           intercept) or the number of observations minus the
                           number of independent variables (zero intercept).
X Coefficient(s)          The slope for each independent variable.
Std Err of Coef.          The standard error of each of the x coefficients.
