/Data Sort Data-Range

/Data Sort Data-Range -- Specifies the range or database table you want to
sort.

1. Select /Data Sort Data-Range.
2. Specify the range or database table that you want to sort.
    The data range contains all the rows or records you want to sort.

    If you are specifying a database table
       Do not include the field names (the first row of the database
      table) in the data range.
       Include all the fields (columns) of the database table in the data
      range. For example, if the database table is in A2..G15 and you want to
      sort only the last three records in the table, specify A13..G15 as the
      data range.
