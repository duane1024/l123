/Print [Background, Encoded, Printer] Options Advanced Layout Line-Spacing

/Print [Background, Encoded, Printer] Options Advanced Layout Line-Spacing --
Determines how much space 1-2-3 allocates for each printed line, and therefore,
how many lines fit on a printed page.

1-2-3 offers two options: Standard and Compressed. The actual spacing
depends on the printer you use. For example, on an Epson FX series
printer, standard line spacing is 6 lines per inch (66 lines per page) and
compressed spacing is 8 lines per inch (88 lines per page) on 11" paper.

1. Select /Print [B,E,P] Options Advanced Layout Line-Spacing.
2. Select Standard or Compressed.
    Standard returns to the default line spacing for your printer.
    Compressed prints more lines per page.
