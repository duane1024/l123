/Data Regression

/Data Regression -- Performs multiple linear regression analysis. Use this
command to find the straight-line relationship between one or more
independent variables and a dependent variable and predict a value for
a dependent variable based on the values for the independent variable(s).

X-Range        Specifies up to 75 independent variables.
Y-Range        Specifies the dependent variable.
Output-Range   Specifies the range in which 1-2-3 places the results
               of the regression analysis.
Intercept      Determines whether 1-2-3 calculates the y-axis intercept or
               uses zero as the intercept.
Reset          Clears the X range, Y range, and output range settings and
               resets the Intercept setting to Compute.
Go             Calculates a data regression for the X range and Y range and
               enters the results in the output range.
Quit           Returns 1-2-3 to READY mode.
