/Data External Use

/Data External Use -- Connects 1-2-3 to an external table so you
can refer to the table in data queries and database @functions.

1. Make sure the cell pointer is in the file in which you want to create
    a range name for an external table.
2. Select /Data External Use.
3. Specify the driver for the database containing the table you want to use.
4. Specify the database containing the table you want to use.
5. Specify the name of the table you want to use.
6. Press ENTER to accept the default range name for the table (the table
    name), or type a new range name of up to 15 characters and press ENTER.

    NOTE  If the table name starts with $ or !, contains a . (period), or
    could be interpreted as a cell address, 1-2-3 does not display a default
    range name for the table. You must type a range name and press ENTER.

To refer to the table in subsequent data queries, Data External commands,
and database @functions, use the range name you assigned the table in step 6.
