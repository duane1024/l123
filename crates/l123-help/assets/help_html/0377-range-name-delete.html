/Range Name Delete

/Range Name Delete -- Deletes a defined range name (and its note, if it has
a note) but leaves the data in the range unchanged.

1. Select /Range Name Delete.
2. Specify the range name you want to delete.

In formulas that use the deleted range name, 1-2-3 replaces the deleted
range name with its range address. For example, if SALES is defined as
B14..H14 and you delete SALES, 1-2-3 changes the formula @SUM(SALES)
to @SUM(B14..H14).

Use /Range Name Reset to delete all range names in the current file
at once.
