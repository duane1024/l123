/Data Parse Input-Column

/Data Parse Input-Column -- Specifies the single-column range that contains
the data you want to parse.

1. Select /Data Parse Input-Column.
2. Specify the single-column range of labels you want to parse.

       The first cell in the input column must contain a format line.
       Every other cell in the range should contain either a format
      line or a long label.
       1-2-3 ignores blank cells and cells containing values.
