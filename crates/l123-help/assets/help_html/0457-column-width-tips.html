Column-Width Tips

Column-Width Tips -- The column width can affect how 1-2-3 displays values
and labels in the worksheet.

Values    When a column is too narrow to display an entire value as
           it is formatted, 1-2-3 displays * (asterisks) instead of
           the value. To display the value, the column must be one
           character wider than the length of the value as formatted.
           To redisplay the value, use /Worksheet Column Set-Width to
           widen the column or use /Range Format to change the value's
           cell format to one requiring fewer characters.

Labels    If a label is longer than the cell it occupies and the cells
           to the right are blank, 1-2-3 displays the label across the
           blank cells. If the cells to the right contain data, 1-2-3
           displays only the part of the label that fits in the cell
           where it is entered. To display the entire label, use
           /Worksheet Column Set-Width to widen the column.
