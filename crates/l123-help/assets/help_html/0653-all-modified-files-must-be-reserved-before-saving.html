All modified files must be reserved before saving

You used /File Save and selected [ALL MODIFIED FILES], but you did not have
the reservation for one or more of the files you tried to save. 1-2-3 cannot
save a file with its existing name if you do not have the reservation.

Select /File List Active to check the reservation status of each active file.
When you identify the read-only (RO) file(s), move the cell pointer to
each file and use /File Admin Reservation Get to try to get the reservation.
If you get the reservation for a file, the RO indicator disappears from
the bottom of the screen.

If you get reservations for all files, select /File Save and specify
[ALL MODIFIED FILES]. If you do not get the reservation for every file,
you can save individual files for which you have the reservation.
