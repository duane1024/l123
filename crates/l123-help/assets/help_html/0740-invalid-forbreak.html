Invalid FORBREAK

You were running a macro and 1-2-3 encountered a {FORBREAK} command
somewhere other than within a subroutine called by a {FOR} command.
{FORBREAK} commands are allowed only within {FOR} subroutines.

Remove the {FORBREAK} command from the macro, or if it is within a
subroutine, use a {FOR} command instead of a {subroutine} or /XC command
to call that subroutine.
