Error replacing extracted records

The following conditions could have caused this error:

* You selected /Data Query Modify Extract, edited the contents of
   any row in the input or criteria range or the contents of the first row
   of the output range, and then selected /Data Query Modify Replace.
   Select /Data Query Modify Extract again, edit only those records in the
   output range that you want to change, and then select /Data Query
   Modify Replace.

* You selected /Data Query Modify Replace when the range address of the
   output range included more rows than there were records in the output
   range. Specify a one-row output range with /Data Query Output, select
   /Data Query Modify Extract, edit the records in the output range, and
   select /Data Query Modify Replace again.
