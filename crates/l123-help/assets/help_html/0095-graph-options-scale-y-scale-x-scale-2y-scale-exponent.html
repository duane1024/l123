/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Exponent

/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Exponent -- Sets the order
of magnitude for numbers along the selected scale. The order of magnitude
is the power of 10 by which the scale numbers must be multiplied to reflect
the values you are graphing. For example, if you are graphing values between
1,000 and 10,000 and the scale numbers range from 1 through 10, the order of
magnitude is 3 (10E+3, or 10 to the third power).

1. Select /Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Exponent.
2. Select Automatic or Manual.
    Automatic    (Default) l123 sets the order of magnitude for the selected
                 scale based on the values you are graphing. For example, if
                 you are graphing values between 0 and 5,000, l123 sets
                 the order of magnitude as 3 (10 to the third power).
    Manual       Lets you set the order of magnitude for the selected scale.
3. If you selected Manual, specify an integer from -95 to 95.

If the numbers along a scale represent an order of magnitude other than 0,
l123 uses the graph scale indicator to identify the order of magnitude.
