Cannot invoke operating system

You selected /System or were running a macro that included a /S instruction
or {SYSTEM} command, and one of the following problems occurred:

* 1-2-3 could not locate the COMMAND.COM file. If your COMMAND.COM file
   is on a removable disk, make sure the disk is in the drive and then
   reselect /System or run the macro again.

* 1-2-3 could not access the COMMAND.COM file because you were already
   using the maximum number of files your operating system allows. To
   solve this problem, use /Worksheet Delete File to delete one or more
   of your active files; use /File Admin Reservation Release to release
   the reservation of one or more active files; or end another application
   program, if one is running. Then reselect /System or run the macro again.

* You have insufficient memory to invoke the operating system. See
   Increasing Available Memory for more information.
