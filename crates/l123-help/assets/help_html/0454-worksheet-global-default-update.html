/Worksheet Global Default Update

/Worksheet Global Default Update -- Updates the l123 configuration file
(123R34.CNF) with the settings you specify with the other Worksheet
Global Default commands and with /Data External Other Refresh Interval.

1. Select /Worksheet Global Default Update.

l123 saves the current Worksheet Global Default settings and /Data External
Other Refresh Interval settings in the l123 configuration file (123R34.CNF)
stored in the directory you started l123 from. l123 uses these settings
each time you start l123 until you change the settings and reselect
/Worksheet Global Default Update.

Use /Print [B,E,F,P] Options to override the default printer settings.
Use /File Dir to override the default directory setting.
