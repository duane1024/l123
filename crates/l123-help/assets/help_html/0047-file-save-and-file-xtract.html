/File Save and /File Xtract

    If you specified a file name that already exists, select one:

    Cancel    Returns l123 to READY mode without saving the file.
    Replace   Writes over the file on disk with the active version of
              the file.
    Backup    Renames the file on disk with the extension .BAK and saves
              the active version of the file.

    CAUTION  When you select Replace, l123 erases the files that you are
    replacing on disk before it saves the data in the files. If l123
    displays an error message ('Disk full') after you select Replace, save
    the data on another disk.
