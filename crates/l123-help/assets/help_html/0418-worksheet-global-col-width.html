/Worksheet Global Col-Width

/Worksheet Global Col-Width -- Sets the width of all columns in the current
worksheet except those columns whose widths you set individually with
/Worksheet Column. In GROUP mode, this command affects all worksheets in the
current file.

1. Select /Worksheet Global Col-Width.
2. Specify a column width either by entering a number from 1 through 240 or
    by using    or    to adjust the width and pressing ENTER.
