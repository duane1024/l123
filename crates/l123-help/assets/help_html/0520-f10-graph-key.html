F10 (GRAPH) Key

F10 (GRAPH) Key -- Displays the current graph or creates an automatic graph
using the data surrounding the cell pointer.

See Creating an Automatic Graph for information on setting up an
automatic graph.

See Creating a Graph with the /Graph Menu for information on setting
up a graph using the menu.

See /Graph View for information on different ways to view a graph.
