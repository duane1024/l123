File name already exists on disk

You tried to read a l123 Release 1A or Symphony file into memory when
a l123 .WK3 file with the same name already existed on disk.

When l123 reads a Release 1A or Symphony file into memory, it converts
the file to Release 3 format by creating a new file with the same
file name and a .WK3 extension. You cannot have two files with the
same file name and extension in the same directory.

Press ESC twice, select /File Save, and specify a new name for the
Release 1A or Symphony file.
