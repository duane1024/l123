/Worksheet Global Zero

/Worksheet Global Zero -- Determines whether l123 displays a zero, a label,
or nothing in cells in the current worksheet that contain either the number
zero or a formula that evaluates to zero. If the current file is in
GROUP mode, this command affects all worksheets in the current file.

1. Select /Worksheet Global Zero and select one of the following:

    No     (Default) Does not suppress the display of zeros.
           Displays a zero in cells whose value is zero.
    Yes    Suppresses the display of zeros. Cells whose value is zero
           appear blank.
    Label  Displays a specified label in cells whose value is zero.

2. If you selected Label, enter a label for l123 to display.
