Invalid format line

You selected /Data Parse Go or /Data Parse Format-Line Edit and the input
column you specified contained an invalid format line. After the | (vertical
bar), format lines can contain only the following characters:

L  V  D  T  S  >  *

NOTE  l123 accepts uppercase and lowercase letters in format lines.

Use F2 (EDIT) and the editing keys to correct the invalid format line.
