/Print [Background, Encoded, Printer] Options Advanced Image Rotate

/Print [Background, Encoded, Printer] Options Advanced Image Rotate --
Determines whether l123 prints graphs upright or rotated (sideways) on the
page. The dimensions of a rotated graph depend on the setting you selected
with /Print [B,E,P] Options Advanced Image Image-Sz.

1. Select /Print [B,E,P] Options Advanced Image Rotate.
2. Select No (default) to print the graph upright on the page, or select
    Yes to rotate the graph 90 degrees counterclockwise.

Use /Print [B,E,P] Options Advanced Layout Orientation Landscape
to rotate ranges of data as well as graphs if your printer is capable of
landscape printing.
