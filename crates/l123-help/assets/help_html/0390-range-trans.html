/Range Trans

/Range Trans -- Copies data in a range, transposing the copied data
and replacing any copied formulas with their current values.

1. If the CALC indicator appears at the bottom of the screen, press
    F9 (CALC) to update formulas. If there are linked formulas in the
    range you want to transpose, you also need to use /File Admin
    Link-Refresh to make sure the values are updated.
2. Select /Range Trans.
3. Specify the FROM range (the range whose data you want to transpose).
4. Specify the TO range (the location for the transposed data). You need
    to specify only the first cell of the TO range. If you are transposing
    a three-dimensional range, be sure there are enough worksheets in the
    TO range to fit the transposed data.

    CAUTION  Make sure you specify a TO range that does not contain data
    you need, because l123 writes over existing data with the data you copy.
    Specifying overlapping FROM and TO ranges may result in data loss.
