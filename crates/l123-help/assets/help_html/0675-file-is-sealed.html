File is sealed

The following conditions could have caused this error:

* You tried to change a sealed file. When a file is sealed using /File
   Admin Seal, you cannot change some graph, print, range, worksheet, and
   reservation settings that have been sealed in the file. Use /File Admin
   Seal Disable to unseal the file. You must know the password that was
   used to seal the file in order to unseal it.

* You tried to extract data from a sealed file. Use /File Admin Seal
   Disable to unseal the file before using /File Xtract.
