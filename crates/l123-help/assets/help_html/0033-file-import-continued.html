4. Specify the name of the text file you want to import.

l123 imports data from a text file either as text, numbers, or both,
depending on the contents of the text file and the option (Text or
Numbers) you selected.

NOTE  Numbers in a text file can contain decimal points, but should not
contain commas because commas act as delimiters.

If data is organized in columns in the text file but is imported as
long labels, use /Data Parse to place data into separate cells.
