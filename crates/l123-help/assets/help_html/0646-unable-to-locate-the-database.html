Unable to locate the database

The following conditions could have caused this error:

* You selected a Data External command and specified an external database
   that did not exist. You may have misspelled the external database name.
   When you specify the external database again, make sure you use the
   correct spelling.

* You selected /Data External Other Command or Translation and specified
   the name of an external database that did not exist or was not in use.
   Specify the name of an existing external database or, if the database
   exists but is not in use, select /Data External Use to connect to an
   external table in that database and then select /Data External Other
   Command or Translation again.
