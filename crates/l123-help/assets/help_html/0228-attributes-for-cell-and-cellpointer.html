Attributes for @CELL and @CELLPOINTER

Attribute      Returns
---------      ------
address        The absolute cell address, abbreviated form (col and row)
col            The column letter, as a value from 1 through 256
color          The color setting for negative values
contents       The contents of the cell
coord          The absolute cell address, full form (worksheet, col, row)
filename       The name of the file that contains the cell
format         The cell format
parentheses    The parentheses setting for negative values
prefix         The label prefix
protect        The protection status
row            The row number, from 1 through 8192
sheet          The worksheet letter, as a value from 1 through 256
type           The type of data in the cell (blank, value, label)
width          The column width
