/Data Query Modify

/Data Query Modify -- Inserts records from the output range in the input
range or replaces records in the input range with records from the
output range. The input range must be in either an active file or an
external table.

Extract        Copies to the output range the records in the input range
                that match the criteria in the criteria range. Unlike /Data
                Query Extract, this command remembers the location in the
                input range for the extracted records, so you can modify
                the records and return them to their original location.
Replace        Replaces the original records in the input range with the
                corresponding records from the output range.
Insert         Adds the records from the output range to the bottom of the
                input range.
Cancel         Cancels /Data Query Modify without updating any
                records in the input range.
