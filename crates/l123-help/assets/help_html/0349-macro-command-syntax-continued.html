Macro Command Syntax (continued)

String        Use a literal string (a string enclosed in quotation
               marks), string formula, or reference to a cell that
               contains a label or string formula.

Location      Use a cell or range address, range name, or
               formula that evaluates to an address or range name.

Condition     Use a logical formula or reference to a cell that
               contains a logical formula.

Some macro commands use optional arguments. In Help for macro commands,
optional arguments appear in [ ] (brackets). For macro commands that use
two or more optional arguments, you can skip an optional argument and
include a subsequent one; to do so, include an argument separator for
the skipped argument.
