/Worksheet Erase

/Worksheet Erase -- Erases all active worksheets and files and replaces
them with one blank worksheet. This command also disconnects 1-2-3
from any external databases you are currently using.

CAUTION  Before using /Worksheet Erase, make sure you save any active
worksheets and files containing data that you need.
1. Select /Worksheet Erase.
2. Select No to return 1-2-3 to READY mode without erasing anything or
    Yes to erase all active worksheets and files. If you selected Yes
    and have worksheets you have changed but not saved, 1-2-3 displays
    another No/Yes menu.
       Select No to cancel /Worksheet Erase and return 1-2-3 to READY mode.
       Select Yes to erase all active worksheets and files.

Note the differences between /Worksheet Erase and the following commands:
/Worksheet Delete Sheet deletes one worksheet from memory.
/Worksheet Delete File deletes one active file from memory.
/File Erase deletes one file from disk.
