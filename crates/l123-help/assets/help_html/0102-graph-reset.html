/Graph Reset

/Graph Reset -- Returns some or all of the current graph settings to the
default settings.

1. Select /Graph Reset.
2. Select one of the following:
    Graph      Resets all current graph settings.
    X          Resets the current X data-range setting.
    A - F      Resets the current A, B, C, D, E, or F data-range setting
               and the corresponding data-label setting, if any.
    Ranges     Resets all current data-range settings (X and A - F) and
               data-label settings.
    Options    Resets all current /Graph Options settings.
3. Select Quit to return to the /Graph menu.
