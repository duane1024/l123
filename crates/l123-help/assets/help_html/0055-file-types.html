File Types

File Types

Worksheet   Files with extensions that begin with .WK or the extension
             specified with /Worksheet Global Default Ext [List, Save]
Print       Files with a .PRN extension, which store worksheet data
             in ASCII format.
Graph       Graph files with a .PIC or .CGM extension, depending on the
             current /Worksheet Global Default Graph setting.
Other       All files in the current or specified directory.
Active      All files in memory.
Linked      Files that are linked by formulas to the current file.
