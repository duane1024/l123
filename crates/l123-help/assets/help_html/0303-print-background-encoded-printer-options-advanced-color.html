/Print [Background, Encoded, Printer] Options Advanced Color

/Print [Background, Encoded, Printer] Options Advanced Color -- Selects a
color for printed text. This command does not affect colors in graphs. Use
Graph Options Advanced commands to set colors in graphs.

1. Select /Print [B,E,P] Options Advanced Color.
2. Select the number (1 - 8) that corresponds to the color you want
    for the specified print range.

You can select one color for each print range. If you want to print
several ranges in different colors, specify the print range, specify
the color, and print each range separately.
