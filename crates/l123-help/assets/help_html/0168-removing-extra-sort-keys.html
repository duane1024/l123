Removing Extra Sort Keys

1. Select /Data Sort Extra-Key.
2. Specify the number of the extra sort key you want to remove.
    For example, if you have six extra sort keys and you want to remove the
    sixth extra sort key, type 6 and press ENTER.
3. Specify a cell address in the field you specified as the preceding
    extra sort key. For example, if you are removing the sixth extra sort
    key, specify a cell in the field you defined as the fifth extra sort
    key (Extra-Key 5).
4. Specify the sort order you specified for the preceding extra sort key.
    For example, if you are removing the sixth extra sort key and the sort
    order for Extra-Key 5 is ascending, specify ascending sort order.
