/Graph Options

/Graph Options -- Lets you enhance a graph with the following commands:

Legend         Creates legends for the A - F data ranges.
Format         Sets the display of lines in line, mixed, XY, and HLCO
               graphs, and creates area graphs.
Titles         Adds graph titles, axis titles, and footnotes.
Grid           Adds or removes graph grid lines.
Scale          Determines axis scaling, controls the appearance of the
               numbers along an axis, and controls the number of x-axis labels
               l123 displays.
Color          Lets you display and print graphs in color, if you
               have color equipment.
B&W            Lets you display and print graphs in black and white.
Data-Labels    Labels individual points or bars in a graph.
Advanced       Sets colors, hatch patterns, fonts, and text sizes for
               a graph.
Quit           Returns you to the /Graph menu.
