    @INFO(attribute)
        Returns system information for the current session.
        Attribute can be one of the following 11 strings:

        directory     The current directory
        memavail      The current amount of available memory
        mode          The current mode
        numfile       The current number of active files
        origin        The cell in the top left corner of the screen at the
                      time of the most recent recalculation
        osreturncode  The value returned by the most recent /System
                      command or {System} macro command
        osversion     The current operating system version
        recalc        The current recalculation mode
        release       The release number of the 1-2-3 product being used
        system        The name of the operating system
        totmem        The total memory (amount available and being used)
