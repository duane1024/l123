Error closing printer driver

You selected /Print [B,E,P] Go and then tried to use /Print [B,E,P]
Options Advanced Device Name or Interface to change the printer name or
interface before closing the current print job with /Print [B,E,P] Quit.

Press ESC twice to clear this screen and the error message, select Quit
to close the current print job, and then use /Print [B,E,P] Options Advanced
Name or Interface to change your settings for the next print job.
