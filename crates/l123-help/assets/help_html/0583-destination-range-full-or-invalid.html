Destination range full or invalid

You tried to enter a table, table definition, or list of information in
one of the following types of locations:

* A range too close to the bottom or right edge of a worksheet to
   accommodate the entire table, table definition, or list. Reselect the
   command and specify a range beginning far enough away from the bottom or
   right edge of the worksheet to accommodate all of the information.

* A range in a file on disk. 1-2-3 cannot enter data in a file on disk.
   Reselect the command and specify a range in an active file as the range
   in which you want 1-2-3 to enter the table, table definition, or list.

* A protected range. 1-2-3 cannot enter data in a protected range.
   Reselect the command and specify an unprotected range as the range in
   which you want 1-2-3 to enter the table, table definition, or list.
