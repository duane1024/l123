/Print [Background, Encoded, Printer] Options Advanced Device

/Print [Background, Encoded, Printer] Options Advanced Device -- Selects a
a printer on which to print from a list of printers you selected in Install,
and selects the printer port and baud rate.

1. Select /Print [B,E,P] Options Advanced Device.
2. Select Name or Interface.
3. If you selected Name, l123 lists numbers (or numbers and letters
    if you installed more than nine printers) that correspond to the
    printers you selected in Install. Select the number or letter of
    the printer you want to use.

    If you selected Interface, specify the printer port and the correct
    baud rate (if you selected a serial port) for the printer.
