/Data Query Input

/Data Query Input -- Specifies one or more ranges that contain database
tables. Database tables contain the records you want to use with a Data
Query command. You must use this command before you can complete other
Data Query commands.

1. Select /Data Query Input.
2. Specify the input range. Be sure to include the row that contains the
    field names and all records in the range.
       To specify an external table, enter the range name you assigned it
      with /Data External Use.
       To specify more than one input range, enter an argument separator
      after typing or highlighting each input range. The argument separator
      can be a , (comma), . (period), or ; (semicolon). When you have
      specified all the input ranges you want to use, press ENTER.
