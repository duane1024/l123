/File Retrieve

/File Retrieve -- Reads a worksheet file from disk into memory, replacing
the file that was current when you last selected /File Retrieve. If you want
to read another file without replacing the current file, use /File Open.

1. If you want to save the current file before you replace it, use /File Save
    first.
2. Select /File Retrieve.
3. Specify the name of a file you want to retrieve.
4. If the file is password-protected, type the password and press ENTER.
5. If someone else is using the file and has the file reservation, l123
    displays a Yes/No menu. Select Yes to retrieve the file without the
    reservation, or select No if you do not want to retrieve the file
    without the reservation.

    CAUTION  If you do not have the reservation, l123 will not let you
    save changes to the file using the same file name.
