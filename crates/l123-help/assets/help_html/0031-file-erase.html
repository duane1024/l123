/File Erase

/File Erase -- Erases a file on disk, but does not remove the active version
of the file. If you want to erase the file you are using from disk and have
the reservation, you must use /File Admin Reservation Release to release
the reservation before you can erase the file on disk. If someone else is
using the file and has the reservation, you cannot erase the file on disk.

Note how /File Erase differs from the following commands:
/Worksheet Erase erases all active files from memory.
/Worksheet Delete File erases one active file from memory.

CAUTION  /File Erase permanently erases a file on disk, so make sure that
you no longer need the data in the file before you erase it.

1. Select /File Erase.
2. Select Worksheet, Print, Graph, or Other.
3. Specify the name of the file you want to erase.
4. Select No to cancel the command without erasing the file, or select
    Yes to erase the file on disk.
