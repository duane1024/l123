Error accessing temporary file

1-2-3 creates and uses temporary files to perform some types of data query
operations.

The following conditions could have caused this error:

* You tried to perform a data query operation, and the disk containing
   the directory you designated as your temporary file directory (using
   /Worksheet Global Default Temp) was full. Free disk space on that
   disk and repeat the data query operation.

* You tried to perform a data query operation, and 1-2-3 encountered a
   hard-disk failure when it tried to access the temporary file. See your
   technical support person for assistance.

* The door to the disk drive was open. Close the drive door and try the
   operation again.
