/Data External Create

/Data External Create -- Sets up the structure for a new table in an
external database.

Name        Connects to the external database in which you want to
            create the new table and assigns a name to the table.
Definition  Lets you create or use a table definition.
Go          Creates a new external table.
Quit        Returns 1-2-3 to READY mode.
