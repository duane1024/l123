Settings System Directory

Settings System Directory -- Lets you specify the default directory you want
l123 to search for add-in files.

1. Select Settings System Directory.
2. To select the default directory, press ENTER.
3. To change the directory, press ESC, type a drive letter, : (colon),
    \ (backslash), and the directory name. Press ENTER.
4. If you change the default directory, select Settings System Update to
    save the change in the add-in configuraton file.

    Unless you specify a directory with Settings System Directory, l123
    looks for add-in files in the default directory.
