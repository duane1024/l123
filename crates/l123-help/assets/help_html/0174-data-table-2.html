/Data Table 2

/Data Table 2 -- This procedure for what-if analysis calculates the
results of one formula that uses two variables. Before you can use /Data
Table 2, you must set up the table range by entering the formula in the
first cell and input values in the first row and column of the table.

1. Decide on a location for the table range.
2. Decide on a location outside of the table range for the two input cells,
    and label them to make them easier to locate.
3. In the upper left cell of the table range, enter the formula you want
    to analyze. Include the addresses of input cell 1 and input cell 2 in
    the formula.
       Input cell 1 always refers to the input values in the first column
      of the table range.
       Input cell 2 always refers to the input values in the first row
      of the table range.
4. In the first column of the table range, beginning with the second cell,
    enter the values or labels for the variable associated with input cell 1.
