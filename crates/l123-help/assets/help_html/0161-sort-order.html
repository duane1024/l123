Sort Order

Sort Order -- The collating sequence, which you can choose when you
install l123, determines the order in which l123 arranges records when
the sort keys (the fields you are sorting by) contain blank cells and
different types of labels. You can choose one of these collating sequences:

    Numbers First (default)
    Numbers Last
    ASCII

Each collating sequence changes the way l123 sorts records in a database
table or rows in a range. To change the collating sequence, use the Install
program. For more information, see "Changing Equipment Selections" in
Chapter 5 of Getting Started.

NOTE  If you choose Numbers First or Numbers Last, l123 sorts lowercase
letters before uppercase letters.
