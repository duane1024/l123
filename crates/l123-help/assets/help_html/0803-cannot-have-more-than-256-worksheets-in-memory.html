Cannot have more than 256 worksheets in memory

The following conditions could have caused this error:

* You selected /Worksheet Insert Sheet, but inserting the number of
   worksheets you specified would have caused the total number of active
   worksheets to exceed 256.

* You selected /File Open, but reading the file you specified into
   memory would have caused the total number of active worksheets to
   exceed 256.

* You selected /File New, but 256 worksheets were already active.

Before you can add new worksheets or files to memory, you must first
delete some currently active worksheets or files. To do so, save your
active worksheets and files and then use /Worksheet Delete Sheet,
/Worksheet Delete File, or /Worksheet Erase.
