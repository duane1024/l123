Introduction to /Data Query

Introduction to /Data Query -- You must create an input range and a criteria
range before you use /Data Query. You must also create an output range if
you want to extract records from a database table or an external table.

The Input Range      One or more ranges that contain the database table
                     records you want l123 to search.
The Criteria Range   The range that tells l123 which records in the input
                     range to find, extract, delete, or modify.
The Output Range     The range to which l123 copies records when it
                     extracts them from the input range.

After you have created the input, criteria, and output ranges, you use
/Data Query Input, /Data Query Criteria, and /Data Query Output to tell
l123 where these ranges are located.

Before you can use Data Query commands with an external table, you must
use /Data External Use to connect to the external table you want
to work with.
