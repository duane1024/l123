    @STDS(list)
        Calculates the sample standard deviation of the values in list.

    @STRING(x,n)
        Converts value x into a string with n decimal places.

    @SUM(list)
        Adds the values in list.

    @SUMPRODUCT(list)
        Multiplies the values in corresponding cells in multiple ranges
        and adds the products. All the ranges in list must be the same
        size and shape, or @SUMPRODUCT returns the value ERR.
