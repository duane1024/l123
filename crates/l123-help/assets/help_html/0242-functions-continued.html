    @ISSTRING(x)
        Returns 1 (true) if x is a string; returns 0 (false) if x
        is a value or blank cell.

    @LEFT(string,n)
        Returns the first n characters in string.

    @LENGTH(string)
        Counts the number of characters in string.

    @LN(x)
        Calculates the natural logarithm (base e) of x.

    @LOG(x)
