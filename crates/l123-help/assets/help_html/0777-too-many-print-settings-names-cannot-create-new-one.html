Too many print settings names; cannot create new one

You tried to create a print settings name when you have already created
the maximum number of print settings names (254).

Use /Print [B,E,F,P] Options Name Delete to delete any print settings names
you no longer need. Then use /Print [B,E,F,P] Options Name Create to create the
new print settings name.
