/Data Query Modify Extract

/Data Query Modify Extract -- Copies to the output range the records in the
input range that match your criteria, and remembers the location in the
input range for the extracted records so you can modify the records and
re-insert them in their original location.

1. Before you use /Data Query Modify Extract, you must specify the input,
    criteria, and output ranges.
2. Select /Data Query Modify Extract.
