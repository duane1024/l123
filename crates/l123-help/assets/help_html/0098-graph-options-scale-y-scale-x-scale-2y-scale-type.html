/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Type

/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Type -- Sets whether l123
uses linear or logarithmic scaling for the selected scale.

1. Select /Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Type.
2. Select Standard or Logarithmic.
    Standard     (Default) l123 uses a linear scale, that is, the
                 numbers along the scale increase at equal intervals and
                 are divided evenly between the lower and upper scale
                 limits. For example, with scale limits of 0 and 10,000,
                 l123 divides the axis into 10 equal intervals, each
                 representing 1,000 units.
    Logarithmic  l123 divides the axis evenly between progressive
                 orders of magnitude (powers of 10). For example, with
                 scale limits of 0 and 10,000, the scale numbers 10, 100,
                 and 1,000 are spaced at equal intervals along the scale.
                 Logarithmic scaling is used most frequently in scientific
                 and engineering applications.
