No external databases are in use

You tried to use /Data External Other Command or Translation when no
external database was in use.

You cannot send a command to or select a character set for an external
database that is not in use. Use /Data External Use to connect to a table
in the external database you want to send a command to or select a
character set for, and then select /Data External Other Command or
Translation.
