The following information is organized as a l123 database table. The
input range for this database table is A:A1..A:D6. The field names are
in row 1 and each row below row 1 is a record.

A:  A          B          C              D

4  Morse      July    Art Etc        2100.00
5  Robinson   May     Travel Log     1700.00
6  Smith      June    Photo Show     1850.00

An external table is a database table in a program other than l123, such
as dBASE III. If you are using an external database, the input range is the
range name of the table in that database that contains the records you want
to use. See /Data External for more information on external database tables.
