/Worksheet Global Default Printer Pg-Length

/Worksheet Global Default Printer Pg-Length -- Sets the length of each
page measured in standard lines. The Pg-Length setting determines where
l123 creates page breaks when printing and the number of printed lines
per page, given the top and bottom margins.

1. Select /Worksheet Global Default Printer Pg-Length.
2. Enter a number of lines from 1 through 1000. Generally, the Pg-Length
    setting should equal the number of standard lines per inch for your
    printer times the length of your paper in inches. The default, 66,
    defines a page length of 66 standard lines for 11-inch paper with
    6 printed lines per inch.
3. To update the l123 configuration file (123R34.CNF) so l123 uses
    the new Pg-Length setting in future sessions, select /Worksheet
    Global Default Update.

The corresponding /Print command is /Print [B,E,F,P] Options Pg-Length.
See Printer Information for page length settings for specific printers.
