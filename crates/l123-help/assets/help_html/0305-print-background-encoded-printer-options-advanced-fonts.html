/Print [Background, Encoded, Printer] Options Advanced Fonts

/Print [Background, Encoded, Printer] Options Advanced Fonts -- Selects fonts
for different sections of the printed page. This command does not affect
fonts in graphs. Use /Graph Options Advanced Text [F,S,T] Fonts for
setting fonts in graphs.

1. Select /Print [B,E,P] Options Advanced Fonts.
2. Select Range, Header/Footer, Border, Frame, or Quit.
    Range          Selects the font for a print range.
    Header/Footer  Selects the font for headers and footers.
    Border         Selects the font for borders.
    Frame          Selects the font for the frame.
    Quit           Returns you to the Advanced menu.
3. Select the number (1 - 8) that corresponds to the font you want to use.
4. Select Quit to return to the /Print [B,E,P] Options Advanced menu.
