Value Criteria

Value Criteria -- Enter value criteria to search fields that contain values.

    To search for an exact match, enter a number exactly as it appears in
   the input range. For example, 1800 matches 1800 or 1800.00.
    Enter a label prefix (' " or ^) and one of the logical perators
   (= < <= > >= and <>) before a value to search for values that match a
   specific condition. For example, '>1500 matches all entries that are
   greater than 1500.

   For example, if you specify A21..B22 as the criteria range, l123
   matches records in the input range where Sales equals 1800.
   If you specify E25..F26 as the criteria range, l123 matches records
   where Sales is greater than 1800.

        A          B                            E          F
   21  Name      Sales                    25   Name      Sales
   22             1800                    26            '>1800
