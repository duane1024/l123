Creating a Colors Range

Creating a Colors Range -- Before you use /Graph Options Advanced Colors
[A - F] Range, you must set up a colors range in the worksheet.

1. Decide on a range that is the same size as the corresponding data range.
2. In each cell in this range, enter a number from 1 through 14 or a negative
    number. The numbers 1 through 14 set colors for the corresponding graph
    data. Negative numbers hide the corresponding graph data.

If you graphed the values under Totals and used the values under Colors Range
in the example below, 500 would be hidden, 1250 would appear in color 3,
and 950 would appear in color 8.

         Totals         Colors Range
            500                  -1
           1250                   3
            950                   8
