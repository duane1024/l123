/Data Query Find

/Data Query Find -- Locates the records in the input range that match the
criteria in the criteria range. You cannot use more than one input range
or an external table with /Data Query Find.

1. Before you can use /Data Query Find, you must specify the input and
    criteria ranges.
2. Select /Data Query Find.
    1-2-3 highlights the first record that matches the criteria. Use the
    following keys to move among the matching records:
      ,          Highlights a record that matches the criteria.
      ,          Moves the cursor to fields within the highlighted record.
    ENTER/ESC  Ends /Data Query Find and returns you to the /Data Query menu.
    F2 (EDIT)  Lets you edit a field in the current record.
    F7 (QUERY) Ends /Data Query Find, leaves the cell pointer in the current
               record, and returns 1-2-3 to READY mode.
    HOME/END   Highlights the first or last record in the input range that
               matches the criteria.
