/Worksheet Global Default Other Clock

/Worksheet Global Default Other Clock -- Sets the display for the
file-and-clock indicator in the lower left corner of the screen.

1. Select /Worksheet Global Default Other Clock and select one:
    Standard       Displays the date in DD-MMM-YY format and the
                   time in HH:MM (AM/PM) format.
    International  Displays the date in Long Intn'l format (D4) and
                   the time in Short Intn'l format (D9).
    None           Removes the file-and-clock indicator from the screen.
    Clock          Always displays the date and time using Standard or
                   International format.
    Filename       (Default) If the current worksheet is part of
                   a file, displays the file's name. Otherwise displays
                   the date and time.

2. To update the l123 configuration file (123R34.CNF) so l123 uses the new
    Clock setting in future sessions, select /Worksheet Global Default Update.
