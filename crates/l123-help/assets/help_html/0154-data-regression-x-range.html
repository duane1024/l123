/Data Regression X-Range

/Data Regression X-Range -- Specifies the set of values for the independent
variable(s).

1. Select /Data Regression X-Range.
2. Specify the X range.

    The X range can contain up to 75 columns, each of which represents an
    independent variable. The columns must be adjacent to one another.
    The X range must have the same number of rows as the Y range.
