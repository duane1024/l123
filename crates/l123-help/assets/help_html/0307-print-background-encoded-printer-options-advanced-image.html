/Print [Background, Encoded, Printer] Options Advanced Image

/Print [Background, Encoded, Printer] Options Advanced Image -- Sets the
following options for printing graphs:

Rotate         Determines whether l123 rotates graphs to print them
               sideways on the page.
Image-Sz       Sets the size of printed graphs.
Density        Determines whether l123 prints graphs in final or draft
               density.
Quit           Returns you to the /Print [B,E,P] Options Advanced menu.
