Invalid X range

The following conditions could have caused this error:

* You did not specify an X range. Select /Data Regression X-Range and
   specify an X range.

* You specified an X range that contained labels, formulas, or blank cells,
   for which 1-2-3 cannot calculate a value.

* You selected /Data Regression Go without specifying any of the
   necessary ranges (the X, Y, and output ranges). Select /Data Regression
   X-Range, Y-Range, and Output-Range and specify these ranges.
