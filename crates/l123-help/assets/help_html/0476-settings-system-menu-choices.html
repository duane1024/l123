Settings System Menu Choices

Settings System Menu Choices -- Set, Cancel, Directory, Update, or Quit

Set       Specifies an add-in file to read into memory automatically
          each time you start 1-2-3. Lets you tell 1-2-3 to start the
          application in an add-in file automatically and to assign the
          application to a specified function key automatically. You can
          start only one application automatically when you start 1-2-3, but
          you can specify multiple add-ins to read into memory and assign
          automatically.
Cancel    Cancels the effect of Set on an add-in file and removes the
          specified file from memory.
Directory Specifies the default directory 1-2-3 searches for add-in files.
          Unless you specify a directory, 1-2-3 looks for add-in files in the
          default directory.
Update    Saves new settings in the add-in configuration file so that 1-2-3
          can use them each time you start 1-2-3.
Quit      Returns you to the ALT-F10 (ADDIN) menu.
