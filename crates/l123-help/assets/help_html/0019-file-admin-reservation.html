/File Admin Reservation

/File Admin Reservation -- Lets you get and release the current file's
reservation and change the reservation setting. Use this command when you
share a file on a network or in another multi-user environment and need
to control how someone gets the file's reservation.

Each file has one reservation. Only the person with the file's reservation
can save changes in the file even though several people may be using
the file at the same time.

Get           Gets the reservation for the current file if it is available
              and no one has saved the file since you read it into memory.
Release       Gives up the reservation for the current file so someone
              else can get it.
Setting       Determines if the reservation is automatically available when
              you read the file into memory or if you must manually get the
              reservation.
