More Formats for /Range Format, /Worksheet Global Format, and /Graph
Options Scale [Y, X, 2Y] Format

+/-              Displays a bar of + (plus signs) or - (minus signs) equal
                  to the integer value of the entry. For example, displays
                  5.9 as +++++. Displays a . (period) for any value
                  between -1 and 1.
Text             Displays actual formulas rather than their calculated
                  values. Displays numbers in General format.
Hidden           Hides data. When the cell pointer is on a hidden
                  cell, the data appears in the control panel unless the
                  cells are protected. To hide entire columns in the
                  worksheet, select /Worksheet Column Hide.
Reset            (/Range Format only) Resets a range to the current global
                  cell format specified with /Worksheet Global Format.
