Named table does not exist

The following conditions could have caused this error:

* You tried to connect to or delete an external table that did not exist.
   You may have misspelled the table name. When you specify the table name
   again, make sure you use the correct spelling.

* You tried to connect to, delete, or use as a model range an external
   table that you did not have access privileges to. You cannot connect to,
   delete, or use as a model range an external table to which you do not
   have access rights.

* You tried to connect to, delete, or use as a model range an external
   table that was a hidden or protected file. You cannot connect to, delete,
   or use as a model range an external table that is a hidden or protected
   file. Specify an external table that is not a hidden or protected file.
