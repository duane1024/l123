/Worksheet Titles

/Worksheet Titles -- Freezes columns and/or rows along the top and left edges
of the current worksheet so those columns and/or rows remain in view as
you scroll through the worksheet. If a file is in GROUP mode, this
command creates titles in all worksheets in that file.

1. Position the cell pointer one cell below the rows you want to freeze
    and/or one column to the right of the columns you want to freeze.
2. Select /Worksheet Titles and select one of the following:

    Both         Freezes the rows above and columns to the left of the
                 cell pointer.
    Horizontal   Freezes the rows above the cell pointer.
    Vertical     Freezes the columns to the left of the cell pointer.
    Clear        Unfreezes all title rows and columns.
