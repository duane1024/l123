/Graph View

/Graph View -- Displays the current graph on the full screen. If the
cell pointer is in an automatic graph range and you have not used /Graph X
or /Graph A - F to specify graph data ranges, 1-2-3 displays an automatic
graph when you select /Graph View.

1. Select /Graph View.
2. When you have finished viewing the graph, press any key to redisplay
    the worksheet.

You can also display the current or automatic graph in the following ways:
    Press F10 (GRAPH).
    Use Worksheet Window Graph to display the current graph in a graph
   window to the right of the current worksheet.

NOTE  If 1-2-3 displays a blank screen when you try to view a graph, one
of the following conditions exists: Your monitor cannot display graphs;
you selected a screen display driver in Install that cannot display graphs;
or you specified incorrect data ranges or scale settings for the graph.
