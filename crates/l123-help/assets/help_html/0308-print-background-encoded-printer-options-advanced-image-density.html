/Print [Background, Encoded, Printer] Options Advanced Image Density

/Print [Background, Encoded, Printer] Options Advanced Image Density -- Lets you
print graphs in either final (high) or draft (low) density.

NOTE  If your printer can print graphs in only one density or 1-2-3
supports only one density on your printer, this command has no effect.

1. Select /Print [B,E,P] Options Advanced Image Density.
2. Select Final to print the graph in high density or select Draft to
    print the graph in low density.

On some printers, graphs print faster in draft density. To save time,
use Draft while experimenting with your graph's format, and use Final to
print your final copy.
