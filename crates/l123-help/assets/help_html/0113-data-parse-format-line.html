/Data Parse Format-Line

/Data Parse Format-Line -- Creates or edits a format line that controls
the way 1-2-3 divides a long label into blocks of data that will become
individual entries, one per column.

For more information, see Format Line.

Create    Creates a format line.
Edit      Lets you edit a format line.
