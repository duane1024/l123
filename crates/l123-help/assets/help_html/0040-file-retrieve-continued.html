6. (Optional) When you retrieve a file, l123 does not automatically
    update formulas that are linked to other files. If the file contains
    linked formulas, select /File Admin Link-Refresh to update the
    formulas. If /Worksheet Global Recalc is set to Manual, l123 does
    not update links but displays the CALC indicator after you select
    /File Admin Link-Refresh. Press F9 (CALC) to update the linked
    formulas in the current file.

NOTE  When l123 retrieves a file created with l123 Release 1A (.WKS)
or Symphony (.WRK or .WR1), it converts the file to .WK3 format and
leaves the original file unchanged on disk. l123 converts a file by
creating a blank worksheet file with the same name and a .WK3 extension
on disk and in memory. You cannot read the converted file on disk into
memory because it is a place holder for the active file and is blank.
If you do not save the converted file, l123 deletes it on disk.
