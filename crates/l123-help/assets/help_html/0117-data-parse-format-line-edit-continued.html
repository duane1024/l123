/Data Parse Format-Line Edit (continued)

1. Position the cell pointer in the cell that contains the format line
    you want to edit.
2. Select /Data Parse Format-Line Edit.
    1-2-3 highlights the format line, places the cursor under the first
    character in the line, and goes into OVR (overstrike) mode.
3. Edit the format line.
4. Press ENTER when you finish editing the format line to enter the
    changes. 1-2-3 returns you to the /Data Parse menu.
