/Data Regression Intercept

/Data Regression Intercept -- Specifies whether 1-2-3 calculates the
intercept or uses zero as the y-axis intercept.

1. Select /Data Regression Intercept.
2. Select Compute or Zero.
    Compute (default) calculates the y-axis intercept.
    Zero uses zero as the y-axis intercept.

    NOTE  Do not select Zero unless your data is such that, if all the
    independent variables equaled zero, the dependent variable would also
    equal zero.

1-2-3 enters the value of the intercept in the fourth column of the second
row of the output range when you select /Data Regression Go to perform the
regression calculations.
