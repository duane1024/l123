    the first occurrence of the search string and displays a
    Replace/All/Next/Quit menu. Select one of the following:
      Replace   Replaces the current string with the replacement string
                and highlights the next occurrence of the search string.
      All       Replaces all remaining occurrences of the search string
                with the replacement string.
      Next      Finds the next occurrence of the search string without
                replacing the current string.
      Quit      Stops the search and returns 1-2-3 to READY mode without
                replacing the current string.
