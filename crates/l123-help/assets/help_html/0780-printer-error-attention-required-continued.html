Printer error - attention required (continued)

The following conditions could have caused this error:

* If there is a file name displayed as part of the error message,
   l123 could not use the file named because you are already using
   the maximum number of files that DOS allows.

   To solve this problem, do one of the following: Use /Worksheet Delete File
   to delete one or more of your active l123 files; use /File Admin
   Reservation Release to release the reservation of one or more files;
   or end another application program, if one is running. Then try to
   print again.

* You were printing a text or encoded file to a diskette and took the
   diskette out of the disk drive before you closed the print job. Before
   removing the diskette, you must select /Print [E,F] Quit or /System or
   press CTRL-BREAK or ESC to close the print job.
