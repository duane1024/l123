/Worksheet Page

/Worksheet Page -- Inserts in the current worksheet a row containing
:: (page-break symbol). This symbol tells 1-2-3 to begin a new page
at this row when printing. If a file is in GROUP mode, /Worksheet
Page affects all worksheets in the file.

1. Position the cell pointer in the leftmost column of the range you
    want to print and the row where you want the new page to begin.
2. Select /Worksheet Page.

1-2-3 inserts a row that contains :: in the current cell and moves
the remaining rows down one row. A page break overrides the
/Print [B,E,F,P] Options Pg-Length or /Worksheet Global Default Pg-Length
settings.

CAUTION  Do not enter data to be printed in the row with :: because
1-2-3 ignores data in this row when printing.
