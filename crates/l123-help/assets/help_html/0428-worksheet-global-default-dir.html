/Worksheet Global Default Dir

/Worksheet Global Default Dir -- Sets the default directory, the directory
1-2-3 uses to save or read files when you start 1-2-3.

1. Select /Worksheet Global Default Dir.
2. Press ESC or BACKSPACE to clear the current default directory setting,
    type the new default directory, and press ENTER. 1-2-3 uses the new
    default directory for the current session only, unless you complete
    step 3.
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the
    new default directory setting in future sessions, select /Worksheet
    Global Default Update.

To override the /Worksheet Global Default Dir setting for the current
session, use /File Dir or type a different directory when you type the
name of a file.
