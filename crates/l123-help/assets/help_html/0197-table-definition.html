Table Definition

Table Definition -- A table definition is a six-column range that contains
information describing an external table. The range contains the following:

Column 1 -- Field names (required). A field name is the name of a field
             as you want it to appear in the external table.
Column 2 -- Data types (required). The data type identifies the type of
             data in a field. l123 has two data types: labels and values.
Column 3 -- Field widths (required for label fields; may or may not be
             required for fields containing values).
Column 4 -- Column labels (may or may not be required). A column label is
             an alternate version of a field name.
Column 5 -- Field descriptions (may or may not be required).
Column 6 -- Field creation strings (may or may not be required).

NOTE  Columns 4, 5, and 6 may be ignored by the DataLens driver you use,
depending on the requirements of the driver. For information on whether you
need to include information in these columns, open DATALENS.WK3 and see "About
the Table Definition" in the worksheet that has information about the driver.
