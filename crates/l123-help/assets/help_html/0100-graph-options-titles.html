/Graph Options Titles

/Graph Options Titles -- Adds graph titles, axis titles, and footnotes to a
graph.

1. Select /Graph Options Titles and select one of the following:
    First       Sets the first line of the graph title.
    Second      Sets the second line of the graph title.
    X-Axis      Sets the x-axis title.
    Y-Axis      Sets the y-axis title.
    2Y-Axis     Sets the second y-axis title.
    Note        Sets the first line of graph footnotes, which are located
                in the lower left corner of the graph.
    Other-Note  Sets the second line of graph footnotes.
2. Specify a title or footnote. You can specify a title in two ways:
       Type the title and press ENTER.
       Press \ (backslash) and specify a cell that contains the title.

See Adding Text to a Graph for a list of commands that add other types of
text to a graph and commands that format graph text.
