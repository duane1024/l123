/Data Query Output

/Data Query Output -- Specifies the range to which l123 copies records from
the input range when you use /Data Query [Extract, Unique, Modify Extract].
When you use /Data Query Modify Insert, the output range contains records
you modify and want to add to the input range. You must create the output
range and use /Data Query Output before you can complete the commands
mentioned above.

1. Select /Data Query Output.
2. Specify the output range.

CAUTION  Either specify a range large enough to hold all records that
match your criteria or specify one row as the output range when the
area below it is blank. If you specify one row as the output range,
l123 erases all data in the worksheet between that row and the bottom
of the worksheet before it completes /Data Query. If you specify a
range larger than one row, l123 copies the selected records to the
output range without erasing the area below the range.
