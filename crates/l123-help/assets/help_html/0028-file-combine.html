/File Combine

/File Combine -- Incorporates data and cell formats from a worksheet file
on disk into the current file. /File Combine does not change column widths,
worksheet settings, or range names in the current file. Before you use
/File Combine, note the following:
    /File Combine changes data in the current file beginning at the
   current cell. As a precaution, save the file before using /File Combine.
    Combining formulas with three-dimensional ranges into a file that
   contains fewer worksheets than are in the three-dimensional ranges may
   produce unexpected results.

1. Move the cell pointer to wherever you want l123 to start combining data.
2. Select /File Combine and select one of the following:
    Copy        Copies specified data from a worksheet file on disk to the
                current file.
    Add         Adds numeric data in a worksheet file on disk to numbers or
                blank cells in the current file.
    Subtract    Subtracts numeric data in a worksheet file on disk from
                numbers or blank cells in the current file.
