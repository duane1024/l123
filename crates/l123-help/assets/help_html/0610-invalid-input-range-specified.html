Invalid input range specified

The following conditions could have caused this error:

* You used /Data External Create Definition Create-Definition and
   specified a model table range that was a single-cell or one-row range
   in an active file or a range in a file on disk. The model table range
   must be either an external table or a range that is in an active file
   and contains a row of field names and at least one row of data. Select
   Create-Definition again and respecify the model table range and the
   location for the table definition.

* You selected /Data Parse Go after you specified an input column that
   was an undefined range name or the range name of an external table, or
   after you specified a range in a file on disk and used a nonexistent
   or invalid file name. Select /Data Parse Input-Column and specify a
   range in an active file or, if you tried to specify a range in a file on
   disk, specify the path and file name, using the correct spelling.
