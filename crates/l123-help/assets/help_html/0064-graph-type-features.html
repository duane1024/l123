/Graph Type Features

/Graph Type Features -- Offers ten general graph features.

Vert           (Default) Displays the x-axis across the bottom of the graph
                and the y-axes along the left and right edges of the graph.

Horiz          Displays the x-axis along the left edge of the graph and
                the y-axes across the top and bottom of the graph.

Stacked        Used with line, bar, mixed, and XY graphs with two or more
                data ranges. Select Yes to stack corresponding data-range
                values, or select No (default) to cancel stacking.

100%           Used with line, bar, mixed, stacked bar, and XY graphs
                with two or more data ranges. Select Yes to graph
                corresponding data-range values as percentages of their
                total value, or select No (default) to graph the absolute
                values of graph data.
