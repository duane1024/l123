/Worksheet Window

/Worksheet Window -- Provides different ways to view worksheets, files,
and graphs.

1. To create horizontal or vertical windows or a graph window, move the
    cell pointer to the row or column you want to use as the top or left
    edge of the second window.
2. Select /Worksheet Window and select one of the following:
    Horizontal     Creates two windows divided by the current row.
    Vertical       Creates two windows divided by the current column.
    Sync           Makes windows scroll together (default).
    Unsync         Allows windows to scroll independently.
    Clear          Restores full-screen worksheet display.
    Map            Turns map view on. In map view, l123 displays # in
                   cells with numbers, + in cells with formulas or
                   annotated numbers, and " in cells with labels.
    Perspective    Displays three consecutive worksheets in perspective view.
                   Use F6 (WINDOW), CTRL-PG UP, and CTRL-PG DN to move the
                   cell pointer between worksheets.
