Data Query Find works only in active files

The following conditions could have caused this error:

* You specified an input range that was an external table or was in a file
   on disk. Either open the file you want to use or specify an input range
   that is in an active file. You cannot use /Data Query Find with external
   tables.

* You specified an input range that was in an active file, but you used a
   file reference that did not include a path. Select /Data Query Input,
   and when 1-2-3 prompts you for the range, press F3 (NAME) and select
   the file name from the displayed list. 1-2-3 automatically includes the
   path when you use this method to create the file reference.
