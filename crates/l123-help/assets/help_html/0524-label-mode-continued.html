LABEL Mode (continued)

You can enter a label that starts with a number (a phone number, for example)
or one of the symbols + - @ . ( or # in two ways:
    Type the label prefix manually. For example, to enter 25 Lucy Lane,
   type a label prefix (' ^  or ") before you type the number 2.
    Use Range Format Other Label to format a range so that l123
   automatically adds a label prefix to all new entries in the range.

l123 does not display the label prefix in the cell, but does display it
in the control panel when you highlight the cell.

You can use the following keys when l123 is in LABEL mode:
BACKSPACE     Erases the character to the left of the cursor.
ENTER         Enters the label in the current cell.
ESC           Cancels the entry and returns l123 to READY mode.
F2 (EDIT)     Switches between LABEL and EDIT mode.
