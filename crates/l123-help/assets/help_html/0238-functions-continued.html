    @IF(condition,x,y)
        Evaluates condition and returns x or y, depending
        on the result of the evaluation. If condition is true, @IF
        returns x; if condition is false, @IF returns y.

    @INDEX(range,column-offset,row-offset,[worksheet-offset])
        Finds the value in the cell located at a specified column-
        offset, row-offset, and worksheet-offset of range.
