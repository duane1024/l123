/Graph Type

/Graph Type -- Selects the type of graph you want to create and sets other
graph features.

Line           Graphs up to six sets of data as lines.
Bar            Graphs up to six sets of data as bars.
XY             Graphs up to six sets of data as points or lines, using a
               scaled x-axis.
Stack-Bar      Graphs up to six sets of data as stacked bars.
Pie            Graphs a circular chart whose slices represent a set of data.
HLCO           Graphs four sets of data as high-low-close-open lines, a
               fifth set of data as bars, and a sixth set of data as a line.
Mixed          Graphs up to three sets of data as bars and up to three sets
               of data as lines.
Features       Sets other graph features, such as graph orientation,
               stacking, use of percentages, use of the second y-axis, graph
               frame display, drop shadows, three-dimensional graphs, and graph
               tables.
