/Worksheet Global Label

/Worksheet Global Label -- Sets the global label alignment for the current
worksheet. /Worksheet Global Label affects future entries by setting which
label prefix l123 automatically adds when you enter a label in the
worksheet. The label prefix controls the way l123 displays labels.

    ' (apostrophe) aligns the label with left edge of the cell.
    " (quotation mark) aligns the label with the right edge of the cell.
    ^ (caret) centers the label in the cell.

If the current file is in GROUP mode, this command affects all worksheets
in the file.

1. Select /Worksheet Global Label.
2. Select Left (default), Right, or Center.

Use /Range Label to change the alignment of existing labels.
