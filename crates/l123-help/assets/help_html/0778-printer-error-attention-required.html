Printer error - attention required

The following conditions could have caused this error:

* A problem with your printer occurred. Be sure your printer is turned on,
   online, and has paper in it. Also, be sure you are using the correct printer
   cable and that it is firmly attached to your printer and your computer.

* Your computer did not have enough available memory to complete the print
   job. Free some memory and then try to print again. If you are printing a
   graph, you can also use this procedure: Save your active files, select
   /Worksheet Erase, and then try to print the graph again.

   NOTE  If you are printing in the background and do not want to continue
   printing, select /Print Cancel to end the print job and clear the error
   message from the screen.
