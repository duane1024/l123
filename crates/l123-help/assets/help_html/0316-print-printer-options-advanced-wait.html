/Print Printer Options Advanced Wait

/Print Printer Options Advanced Wait -- Suspends printing after each page.
Use this command when you are printing on a printer that requires single-
sheet feeding. This command overrides the /Worksheet Global Default
Printer Wait setting.

1. Select /Print Printer Options Advanced Wait.
2. Select No (default) if you do not want l123 to wait or Yes if you
    want l123 to wait after it prints each page.
    NOTE  Do not select Yes if you are using a print spooler because the
    spooler controls single-sheet feeding.

    If you selected Yes, l123 waits after it prints each page and displays
    a message prompting you to insert the next sheet of paper. Insert the
    paper and then select one of the following to continue printing:
       Select any key to complete the current print job.
       Select Hold if you want to specify additional print ranges for
      the current print job. After you specify additional ranges, select
      Resume to complete the print job.
