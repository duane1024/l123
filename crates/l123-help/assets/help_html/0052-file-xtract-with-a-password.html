/File Xtract with a Password

1. Select /File Xtract.
2. Select Formulas or Values.
3. Type a file name for the extracted range.
4. Press the space bar.
5. Type p and press ENTER.
6. Type a password and press ENTER. A password can include up to 15
    characters. 1-2-3 is case-sensitive for passwords.
    CAUTION  Remember the password. You can read the file into memory again
    only if you enter the exact password.
7. Type the same password again and press ENTER.
8. Specify the range of data you want to extract.
9. If you specified a file name that already exists, select Replace if you
    want to replace the file on disk with the extracted range in a password-
    protected file.
