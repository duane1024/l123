/Graph Name Table

/Graph Name Table -- Creates a three-column table that lists all named
graphs in the current file. The table lists the name, type, and title
(if any) of each named graph.

1. Decide on a location for the graph name table. The table can be in any
    active file. It will occupy three columns and as many rows as there
    are named graphs, plus one blank row.

    CAUTION  Graph name tables write over existing data, so be sure
    the location you decide on does not contain data you need.

2. Make sure the cell pointer is in the file whose named graphs you want
    to list.
3. Select /Graph Name Table.
4. Specify the location you decided on in step 1. You need to specify
    only the first cell of the table's location.
