Too many records for Output range

You specified a multiple-row output range for a data query operation,
but did not specify enough rows to accommodate all the records that match
the criteria in the criteria range.

Specify an output range containing more rows or, if it is safe to do so,
specify a one-row output range. Keep in mind that 1-2-3 erases all data
below the output range if you specify a one-row output range.
