/Worksheet Global Default Printer AutoLf

/Worksheet Global Default Printer AutoLf -- Sets whether 1-2-3 should insert
a line-feed character after each printed line. The correct choice depends
on whether your printer inserts a line feed automatically.

1. Select /Worksheet Global Default Printer AutoLf.
2. Select Yes if your printer automatically inserts a line feed, or
    select No (default) if your printer does not insert a line feed.
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the new
    AutoLf setting in future sessions, select /Worksheet Global Default
    Update.

To test whether the current AutoLf setting is correct for your printer,
print a range of at least two lines. If the printing is double-spaced, or
if the paper does not advance, set AutoLf to the opposite setting.

The corresponding /Print command is /Print [B,E,F,P] Options Advanced AutoLf.
