The /X Commands

The /X Commands

    /XClocation~
        Calls the subroutine at location. /XC corresponds to {subroutine}
        except that /XC does not pass arguments to subroutines.

    /XGlocation~
        Transfers macro control (that is, branches) from the current column
        of macro instructions to the macro instructions at location. /XG
        produces the same result as {BRANCH}.

    /XIcondition~
        If condition is true, 1-2-3 executes the macro instructions
        that follow the /XI command in the same cell. If condition
        is false, 1-2-3 skips to the cell below the /XI command and
        executes the macro instructions there. /XI produces the same
        result as {IF}.
