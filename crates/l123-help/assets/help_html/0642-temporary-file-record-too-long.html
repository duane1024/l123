Temporary file record too long

l123 creates and uses temporary files to perform some types of data query
operations. The temporary file l123 created for the data query operation
you tried to perform was not large enough to accommodate one of
the records in the database table you were using.

If you are running low on memory, make more memory available and repeat
the data query operation. If you are not running low on memory, specify an
input and/or output range that contains fewer fields or, if one field
contains a very long entry, shorten that entry. Then repeat the data query
operation.
