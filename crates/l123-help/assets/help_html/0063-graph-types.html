Graph Types

Graph Types -- Descriptions (continued)

HLCO       High-low-close-open graphs, also called stock market graphs,
            track fluctuations in stock prices (or any other type of data)
            over time. The high, low, close, and open data for each time
            period is depicted as a vertical line in the graph. For HLCO
            graphs, l123 uses the X data range as the x-axis labels;
            the A, B, C, and D ranges as the sets of high, low, close, and
            open values, respectively; the E data range as a set of bars
            below the high-low-close-open area of the graph; and the F data
            range as a line in the high-low-close-open area.

Mixed      Mixed graphs are combination bar and line graphs. They plot
            different types of data in the same graph. For mixed graphs,
            l123 uses the X data range as the x-axis labels; the A, B,
            and C data ranges as sets of bars in the graph; and the D, E,
            and F data ranges as lines.
