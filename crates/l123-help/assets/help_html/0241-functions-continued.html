    @ISNA(x)
        Returns 1 (true) if x is the value NA; returns 0 (false)
        if x is not the value NA.

    @ISNUMBER(x)
        Returns 1 (true) if x is a value or blank cell; returns 0 (false)
        if x is a string.

    @ISRANGE(range)
        Returns 1 (true) if range is a defined range name or valid range
        address; returns 0 (false) if range is not a defined range name
        or valid range address.
