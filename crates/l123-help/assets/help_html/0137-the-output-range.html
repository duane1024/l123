The Output Range

The Output Range -- Tells l123 where to put data from the input range that
matches the criteria in the criteria range. After you set up the output
range, use /Data Query Output to tell l123 where the output range is
located. The output range can be in a l123 worksheet or an external table.

To create an output range in a l123 worksheet

1. Choose an area in the current file that does not overlap the input
    or criteria ranges and has enough rows below it to hold the records
    that l123 will select.
2. Enter field names or formulas in the first row.

       If you use field names, copy some or all of the field names from
      the input range to the output range. Field names in the output
      range must be identical to those in the input range, but you can
      list the field names in the output range in any order.
