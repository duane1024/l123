/Data Fill

/Data Fill -- Enters a sequence of numbers, dates, times, or percentages
in a range.

1. Select /Data Fill.
2. Specify the range you want to fill.

    CAUTION  l123 erases the entire range before filling it, so make sure
    the range does not contain data you need.

3. Enter numbers, dates, times, formulas, range names, or cell addresses
    for the start, step, and stop values.

l123 fills the cells in the range from top to bottom in a column and from
left to right. If you specify a multiple-sheet fill range, l123 fills
the first worksheet in the range, continues the sequence on the second
worksheet, and so on until l123 reaches the stop value or the end of the
range.
