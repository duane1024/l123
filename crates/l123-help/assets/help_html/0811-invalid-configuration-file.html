Invalid configuration file

The following conditions could have caused this error:
* When you started l123, it located a l123 configuration file that was not a
   valid configuration file for this release of l123. End l123. If the PATH
   statement in your AUTOEXEC.BAT file includes the program directory for a
   previous release of l123, you must delete that directory from the PATH
   statement. If you leave your previous l123 directory in the PATH statement,
   this release of l123 will overwrite the configuration file in that directory
   when you use /Worksheet Global Default Update to change any default settings.

* When you started l123, it located the l123 configuration file, but
   the directory specified in the file as the l123 default directory or
   the l123 temporary directory did not exist. To see the configuration
   settings l123 is now using, select /Worksheet Global Default Status.
   If you want to change any of those settings, use the appropriate
   /Worksheet Global Default commands, and then use /Worksheet Global
   Default Update to update your configuration file with the new settings.
