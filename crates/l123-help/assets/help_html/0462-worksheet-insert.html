/Worksheet Insert

/Worksheet Insert -- Inserts one or more blank columns or rows in existing
worksheets and adds worksheets to files in memory, creating multiple-sheet
files.

CAUTION  /Worksheet Insert inserts entire columns, rows, and worksheets
in a file, not just in the part you see on the screen. If a file is in
GROUP mode, inserting rows or columns in one worksheet inserts rows or
columns in all worksheets in that file.

1. Select /Worksheet Insert.
2. Select Column, Row, or Sheet.
3. If you selected Column or Row, specify a range that includes the
    columns or rows you want to insert. If you selected Sheet, select
    Before or After to insert new worksheets before or after the current
    worksheet. Then specify the number of worksheets to insert.

When you insert columns, rows, or worksheets, l123 redefines named
ranges and adjusts cell and range addresses in formulas.
