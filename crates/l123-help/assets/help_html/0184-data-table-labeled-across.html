/Data Table Labeled Across

/Data Table Labeled Across -- Specifies the column variable range and
input cells. A column variable range is a range that contains columns of
input values, organized by rows. A column variable range can be one or
more rows, as long as each row contains a separate set of input values.

1. Select /Data Table Labeled Across if you have a column variable range.
2. Specify the column variable range.
    1-2-3 displays the address of the top row of input values in the
    column variable range. Press ENTER to identify this set of input values
    as the set to which you will next assign an input cell.
3. Specify the input cell for that row of input values.
    If you entered a column variable range that contains more than one row,
    1-2-3 repeats the prompts for assigning input cells for each additional
    row of input values, moving from top to bottom.
