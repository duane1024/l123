/Graph Name Use

/Graph Name Use -- Retrieves a named graph and displays a full-screen
view of the graph.

CAUTION  /Graph Name Use replaces the current graph settings with a
named graph's settings. To preserve the current graph settings for later
use, name them with /Graph Name Create before selecting /Graph Name Use.

1. Make sure all files that contain data used in the named graph are
    active.
2. Select /Graph Name Use.
3. Specify a named graph.
4. When you finish viewing the graph, press any key to redisplay the
    worksheet.
