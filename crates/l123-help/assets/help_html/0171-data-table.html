/Data Table -- Cross-Tabulation

When you use a what-if table for cross-tabulation, you can see relationships
among fields in a database table. For example, suppose you have a
database table that shows company sales broken down by salesperson, month
of sale, account, and dollar amount for each sale. Using /Data Table 2,
you can cross-tabulate the information to show total sales for each
person for each month.

Before you can use tables to cross-tabulate database information,
you need to be familiar with the structure of database tables and database
@functions. For information on database tables, see "l123 Database Tables"
in Chapter 3 of Reference. For information on database @functions, see
"Database @Functions" in Chapter 3 of Reference. For more information and
detailed examples of cross-tabulation, see /Data Table in Chapter 3
of Reference.
