/Graph Options Legend

/Graph Options Legend -- Creates legends for the graph's data ranges. The
legends, located below the graph, identify the data range represented by
each symbol, color, or hatch pattern in the graph.

1. Select /Graph Options Legend.
2. Select A, B, C, D, E, or F to assign a legend for an individual data
    range, or select Range to assign legends for all data ranges at once.
3. If you selected A, B, C, D, E, or F, specify the legend for the data
    range. You can specify a legend in two ways:
       Type the legend and press ENTER.
       Press \ (backslash) and specify a cell that contains the legend.

    If you selected Range, specify the range that contains all the legends.
    1-2-3 uses the first entry in the range as the legend for the A data
    range, the second entry as the legend for the B data range, and so on.
