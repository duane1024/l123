Cannot go to a range that is not in memory

You were using F5 (GOTO) and one of the following problems occurred:

* You specified a file that is not in memory as the cell-pointer
   destination. You can move the cell pointer to cells in active files only.

* The active file you specified as the cell-pointer destination
   is stored in a directory other than the current directory and you did
   not include a path in the file reference. Press ESC and enter the
   cell-pointer destination again using a path in the file reference.

* The active file you specified as the cell-pointer destination does not
   have the default file extension and you did not include an
   extension in the file reference.  Press ESC and enter the cell-pointer
   destination again using an extension in the file reference.
