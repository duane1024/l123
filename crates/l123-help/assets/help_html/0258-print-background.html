/Print Background

/Print Background -- Creates an encoded file and prints it while you continue
to work in the worksheet or create additional print jobs. 1-2-3 deletes this
encoded file after it finishes printing.

NOTE  You cannot use /Print Background to print on a network printer.

1. If you did not start BPrint before you started 1-2-3, save your work,
    select /Quit, enter bprint at the DOS prompt, and restart 1-2-3.
2. Select /Print Background.
3. Specify a file name.
4. Select Range and specify a range to print. See /Print [B,E,F,P] Range
    for more information on specifying a print range.
5. (Optional) Select Options (/Print [B,E,F,P] Options) to set
    different options.
6. Make sure the printer is online and the paper is at the top of a page.
7. Select Align.
8. Select Go to print the range.
9. Select Quit to complete the print job and return 1-2-3 to READY mode.
