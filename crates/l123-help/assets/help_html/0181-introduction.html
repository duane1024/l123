Introduction

Introduction -- /Data Table Labeled produces a table that shows the effects
of changing one or more variables in one or more formulas, much as
/Data Table 1, 2, and 3 do. The flexibility of /Data Table Labeled,
however, lets you do the following:

    Specify existing data in different areas of a file as your
   input values.
    Control the placement of formulas, input values, and results.
    Use more variables.
    Include descriptive labels that identify the items in the table.
    Include blank rows or columns to improve the layout of the table.
    Include formulas within the table so you can perform
   calculations with the results.
    Include text within the table so you can annotate its contents.

For detailed procedures and examples using /Data Table Labeled, see
/Data Table Labeled in Chapter 3 of Reference.
