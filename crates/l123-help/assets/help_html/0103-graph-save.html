/Graph Save

/Graph Save -- Saves the current graph in a graph file on disk so you can
use the graph with other programs.

1. If necessary, use /Worksheet Global Default Graph to change the
    current graph file format setting to the one you want to use. The
    choices are graphic metafile (default) and picture file.
2. Select /Graph Save.
3. Specify a graph file name.
4. If you enter an existing file name, select Cancel to end /Graph Save
    without saving the current graph or Replace to write over the existing
    graph file with the current graph.

NOTE  /Graph Save saves graphs only for use outside 1-2-3. You cannot
retrieve graph files from within 1-2-3. Use /Graph Name Create to
save the current graph in a worksheet file for future use with 1-2-3.
