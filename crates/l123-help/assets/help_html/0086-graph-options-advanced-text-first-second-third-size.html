/Graph Options Advanced Text [First, Second, Third] Size

/Graph Options Advanced Text [First, Second, Third] Size -- Sets text sizes
for the graph-text groups in printed and displayed graphs.

1. Select /Graph Options Advanced Text.
2. Select First, Second, or Third to select the graph text.
3. Select Size and select one of the following:
    1 - 9    To set the corresponding size for the selected graph-text group.
    Default  To use the default size for the selected graph-text group.
             The defaults are 8, 4, and 2 for the first, second, and third
             graph-text groups, respectively.

NOTE  1-2-3 displays graph text in only three sizes unless you use an
external utility program to add additional sizes. Select settings 1 - 3
for small-size text, 4 - 6 for medium-size text, and 7 - 9 for large-size
text. The number of text sizes available for printed graphs depends
both on your printer and on the text font you select with /Graph Options
Advanced Text [F,S,T] Font.
