/Range Name Labels

/Range Name Labels -- Assigns range names to single-cell ranges, using the
labels (up to 15 characters long) in adjacent cells as the range names.
Use /Range Name Labels to assign an existing label as the range name for the
cell immediately above, below, to the right, or to the left of the label.
See /Range Name Create for rules on characters in range names.

1. Select /Range Name Labels.
2. Select Right, Down, Left, or Up to indicate the position of the cells
    you are naming in relation to the labels.
3. Specify the range that contains the labels you want to use as range
    names.

In any formulas that use a range address (for example, B5..B5 as opposed
to just B5) to refer to a cell you named with /Range Name Labels, 1-2-3
replaces the range address with the corresponding range name. For example,
when you assign the name APRIL to B5, 1-2-3 changes the formula @MAX(B5..B5)
to @MAX(APRIL), but 1-2-3 does not change the formula +B5/30.
