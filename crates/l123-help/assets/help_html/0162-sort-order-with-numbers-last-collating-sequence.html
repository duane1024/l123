Sort Order with Numbers Last Collating Sequence

Ascending  1. Blank cells
            2. Labels beginning with a space
            3. Labels beginning with letters in alphabetical order
            4. Labels beginning with numbers in numerical order
            5. Labels beginning with other characters in LMBCS order
            6. Values in numerical order

Descending 1. Values in reverse numerical order
            2. Labels beginning with other characters in reverse LMBCS order
            3. Labels beginning with numbers in reverse numerical order
            4. Labels beginning with letters in reverse alphabetical order
            5. Labels beginning with a space
            6. Blank cells

NOTE  In sorting records, 1-2-3 ignores apostrophes and hyphens when these
characters act as separators in names, for example, O'Leary.
