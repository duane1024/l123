F6 (WINDOW) Key

F6 (WINDOW) Key -- Moves the cell pointer between windows you create with
/Worksheet Window Horizontal, Vertical, or Perspective.

ALT-F6 (ZOOM) Key -- Enlarges the current window to full-screen size or
shrinks it to its original size.

After you create windows, you can press F6 (WINDOW) to move the cell
pointer from one window to another. Press ALT-F6 (ZOOM) to switch the
current window between its original size and full-screen size. After
you press ALT-F6 (ZOOM), 1-2-3 displays the ZOOM indicator.
