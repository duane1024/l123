CTRL-F9 (DISPLAY ICONS) and CTRL-F10 (SELECT ICONS) Keys

CTRL-F9 (DISPLAY ICONS) and CTRL-F10 (SELECT ICONS) Keys -- Control the use of
the icon palette when Wysiwyg is attached.

Press CTRL-F9 (DISPLAY ICONS) to hide or display the icon palette in
READY mode.

Press CTRL-F10 (SELECT ICONS) to select an icon from the icon palette in
READY mode. When you press CTRL-F10 (SELECT ICONS), 1-2-3 activates the
icon palette, highlights the first icon in the icon palette, and displays
a brief description of the highlighted icon in the control panel. Use the
pointer-movement keys to highlight the icon you want to use, and then
press ENTER.
