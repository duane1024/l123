/Print [Background, Encoded, File, Printer] Options Margins

/Print [Background, Encoded, File, Printer] Options Margins -- Sets the left,
right, top, and bottom margins or clears all margin settings. Use this command
to set margins that are different from the default margin settings below:

    Left         4 characters from the left edge of the page
    Right       76 characters from the left edge of the page
    Top          2 lines from the top of the page
    Bottom       2 lines from the bottom of the page

1. Select /Print [B,E,F,P] Options Margins.
2. Select Left, Right, Top, Bottom, or None.
    l123 displays the current margin setting, unless you selected None.
    If you selected None, l123 sets the left, top, and bottom margins to
    0 and the right margin to 1000.
3. Enter a margin setting. For Left and Right, enter any number from 0 through
    1000. For Top and Bottom, enter any number from 0 through 240.

To restore the default printer settings, use /Print [B,E,F,P] Clear All.
