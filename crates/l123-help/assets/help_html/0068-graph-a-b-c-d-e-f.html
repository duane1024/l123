/Graph [A, B, C, D, E, F]

/Graph [A, B, C, D, E, F] -- Sets the A - F data ranges (the ranges that
contain the values to be graphed) individually. The way 1-2-3 uses these
data ranges depends on the graph type you select.

    In bar graphs, 1-2-3 graphs the A - F data ranges as sets of bars.
    In HLCO graphs, 1-2-3 graphs the A - D data ranges as the high, low,
   close, and open values, respectively; the E data range as a set of bars
   below the high-low-close-open area of the graph; and the F data range
   as a line in the high-low-close-open area.
    In line graphs, 1-2-3 graphs the A - F data ranges as lines.
    In mixed graphs, 1-2-3 graphs the A, B, and C data ranges as sets
   of bars and the D, E, and F data ranges as lines.
    In pie charts, 1-2-3 graphs the A data range as the pie slices. 1-2-3
   uses the B data range to set colors or hatch patterns for the pie slices
   (enter a value from 1 through 14 in each B-range cell), to explode one or
   more slices (add 100 to the B-range values), or to hide one or more slices
   (use negative B-range values). 1-2-3 uses the C data range to remove
   percent labels from the pie slices (enter 0 in the C-range cells).
