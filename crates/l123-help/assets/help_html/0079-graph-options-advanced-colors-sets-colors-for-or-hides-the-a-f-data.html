/Graph Options Advanced Colors - Sets colors for or hides the A - F data
ranges in displayed and printed graphs.

1. Select /Graph Options Advanced Colors.
2. Select A, B, C, D, E, or F as the data range and select
    1 - 8     To set one color for all values in the data range.
    Hide      To suppress display of the data range.
    Range     To set different colors for, or to hide, the individual values
              in the data range. If you select Range, you should already have
              created a colors range. See Creating a Colors Range.
3. If you selected Range, specify the colors range.
4. Repeat steps 2 - 4 to set colors for other data ranges.

NOTE  The colors that correspond to the numbers vary depending on your
monitor and printer. To see which colors correspond to the numbers for
your monitor, assign each of the numbers to a data range and view the graph.
