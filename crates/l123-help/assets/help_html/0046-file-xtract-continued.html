/File Xtract (continued)

1. If the CALC indicator appears at the bottom of the screen, press F9 (CALC)
    to update formulas before you use /File Xtract Values. If any formulas
    are linked to other files, also use /File Admin Link-Refresh.
2. Select /File Xtract.
3. Select Formulas or Values. Formulas extracts data including formulas.
    Values extracts data and converts formulas into values.
4. Specify a file name for the extracted range.
5. Specify the range of data you want to extract.
6. If you specified a file name that already exists, select one:
    Cancel    Returns 1-2-3 to READY mode without extracting the range.
    Replace   Writes over the file on disk with the extracted range.
    Backup    Renames the file on disk with the extension .BAK and saves
              the extracted range with the existing file name.
    CAUTION  When you select Replace, 1-2-3 erases the file on disk before it
    saves the extracted data in the file. If 1-2-3 displays an error message
    ('Disk full') after you select Replace, extract the data on another disk.
