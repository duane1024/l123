/Data Regression Output-Range

/Data Regression Output-Range -- Specifies the range in which 1-2-3 places
the results of the multiple linear regression analysis.

1. Select /Data Regression Output-Range.
2. Specify the output range. You can specify either the entire range or
    only the first cell in the range.

    CAUTION  1-2-3 creates an output range in a rectangular area that
    has nine rows and a minimum of four columns, with an additional column
    for each X variable after the second X variable. Make sure the area you
    want to specify as the output range does not contain any data you need.

1-2-3 performs the regression calculations when you select /Data Regression
Go. If you specify an output range that is too small to contain the results
of the calculations, 1-2-3 displays an error message when you select /Data
Regression Go.
