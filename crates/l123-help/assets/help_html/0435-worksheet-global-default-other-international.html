/Worksheet Global Default Other International

1. Select /Worksheet Global Default Other International and select one:
    Punctuation       Sets the characters for the decimal point and thousands
                      separator for numbers, and the argument separator for
                      @functions and macro commands.
    Currency          Sets the currency symbol 1-2-3 uses for Currency
                      format and the location of the symbol before or after
                      numbers.
    Date              Sets the two Intn'l Date formats (D4 and D5).
    Time              Sets the two Intn'l Time formats (D8 and D9).
    Negative          Sets whether 1-2-3 uses parentheses or a minus sign
                      for negative values in Currency or Comma format.
    Release-2         Sets the way 1-2-3 reads and saves characters
                      in Release 2 (.WK1) files.
    File-Translation  Sets the character-translation table, country-specific
                      or international, 1-2-3 uses when importing and
                      creating text files.
