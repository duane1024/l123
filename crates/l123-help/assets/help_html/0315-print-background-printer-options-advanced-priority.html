/Print [Background, Printer] Options Advanced Priority

/Print [Background, Printer] Options Advanced Priority -- Determines the order
in which 1-2-3 prints different print jobs on a printer.

NOTE  This release of 1-2-3 does not use this command. The command exists
in this release of 1-2-3 in order to maintain macro compatibility with earlier
1-2-3 releases.
