Remove

Remove -- Deletes one add-in file from memory.

1. Select Remove.
2. Specify the file you want to remove from memory.

If you assigned an application in the removed file to a key, l123
automatically clears the key.

Any formulas that contain @functions from an add-in file you removed from
memory evaluate to ERR.

NOTE  Once you remove an add-in file, l123 cannot use the add-in it
contains until you read that file into memory again.
