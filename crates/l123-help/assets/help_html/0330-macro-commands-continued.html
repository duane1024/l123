Macro Commands (continued)

    {IF condition}
        Evaluates condition as true or false. If condition is true,
        l123 executes the macro instructions that follow the {IF}
        command in the same cell. If condition is false, l123
        skips to the cell below the {IF} command and executes the
        macro instructions there.

        Condition is typically a logical formula (a formula that
        uses one of the logical operators <  >  =  <>  >=  <=
        #NOT# #AND# and #OR#), or a reference to a cell that
        contains a logical formula.
