Automatic Wysiwyg

Automatic Wysiwyg -- l123 loads Wysiwyg automatically at the beginning of
each l123 session. If you do not want to work with Wysiwyg, you can remove
Wysiwyg from memory for the current session or change your default add-in
start-up settings so that l123 does not load Wysiwyg automatically.

To remove Wysiwyg from memory for the current session, do the following:

1. Press ALT-F10 (ADDIN) and then select Remove.
2. Specify WYSIWYG as the add-in to remove from memory.
3. Select Quit to return l123 to READY mode.

To change your default add-in start-up settings so that l123 does not load
Wysiwyg automatically at the beginning of future sessions, do the following:

1. Press ALT-F10 (ADDIN) and then select Settings System Cancel.
2. Specify WYSIWYG as the add-in to cancel.
3. Select Update to save the new settings in the add-in configuration file
    and then select Quit twice to return l123 to READY mode.
