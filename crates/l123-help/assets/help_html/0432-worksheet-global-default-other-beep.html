/Worksheet Global Default Other Beep

/Worksheet Global Default Other Beep -- Sets whether the computer's bell
sounds when errors occur and when {BEEP} commands occur in a macro.

1. Select /Worksheet Global Default Other Beep.
2. Select No to turn the bell off or Yes (default) to turn the
    bell on.
3. To update the l123 configuration file (123R34.CNF) so l123 uses the
    new Beep setting in future sessions, select /Worksheet Global Default
    Update.
