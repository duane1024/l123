/Print [Background, Encoded, File, Printer] Options Header and Footer

/Print [Background, Encoded, File, Printer] Options Header and Footer -- Create
either a line of text just below the top margin (header) or a line of text
just above the bottom margin (footer) on every page. Headers and footers can
be up to 512 characters long but cannot extend beyond the right margin.

1. Select Print [B,E,F,P] Options.
2. Select Header or Footer.
3. Type the header or footer text and symbols (if any) or type a
    \ (backslash) and point to the cell that contains the header or
    footer text you want to use, and press ENTER.

l123 prints the header on the line below the top margin and the footer
on the line above the bottom margin. l123 reserves 3 lines for a header
and 3 lines for a footer -- 1 line for the actual header or footer text
and 2 lines to separate them from the printed data.
