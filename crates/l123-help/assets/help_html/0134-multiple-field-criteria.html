Multiple-Field Criteria

Multiple-Field Criteria -- Enter multiple-field criteria in the criteria
range in the following ways:

    Enter criteria in separate rows and different fields in the criteria
   range. l123 treats criteria in separate rows as if they were connected
   by #OR#, so l123 matches records that meet one criterion or the other.
    Enter criteria in one row under different fields in the criteria range
   to identify only those records that match all the criteria at once.
   l123 treats criteria in the same row as if they were connected by #AND#.
    Use logical perators #AND#, #NOT#, or #OR# in formulas to create
   criteria that match more than one condition in the same field. For
   example, +SALES>1500#AND#+SALES<2200 matches records with SALES entries
   greater than 1500 but less than 2200.
