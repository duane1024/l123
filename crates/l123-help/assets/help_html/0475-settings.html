Settings

Settings -- Specifies the add-in files you want l123 to read into memory
automatically and the applications you want l123 to start or assign to
keys automatically. It also lets you specify a default add-in directory.

Settings System   Tells l123 to read specified add-in files into
                  memory automatically, to start a specified application
                  automatically, and to assign applications to function
                  keys automatically when you start l123. It also lets
                  you specify the default add-in directory.

Settings File     Tells l123 to read specified add-in files into
                  memory automatically and to start an application
                  automatically when you read a certain worksheet file
                  into memory.
