/Data Query Modify Replace

/Data Query Modify Replace -- Replaces the original records in the input
range with edited records from the output range. Use /Data Query Modify
Replace after you use /Data Query Modify Extract to replace records in a
database table or external table with edited or updated records.

CAUTION  To avoid writing over formulas in the input range, make sure
the output range does not include names of fields that contain formulas.

1. Before you use /Data Query Modify Replace, you must specify the input,
    criteria, and output ranges.
2. Select /Data Query Modify Extract to copy records to the output range.
3. Edit the records in the output range.
    CAUTION  Do not add or delete rows in the output range or sort records
    in the output range. If you do, 1-2-3 cannot match correctly the
    edited records with records you are replacing in the input range.
4. Select /Data Query Modify Replace.
