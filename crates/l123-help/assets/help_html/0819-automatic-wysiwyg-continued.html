Automatic Wysiwyg (continued)

To restore automatic start-up of Wysiwyg, do the following:

1. Press ALT-F10 (ADDIN).
2. Select Settings System Set.
3. Select WYSIWYG.PLC.
4. Select Yes.
5. Select No-Key if you plan to use : (colon) or a mouse to display
    the Wysiwyg main menu. Select 1, 2, or 3 to assign Wysiwyg to
    ALT-F7 (APP1), ALT-F8 (APP2), or ALT-F9 (APP3), respectively.
6. Select Update to save the new settings in the add-in configuration file.
7. Select Quit twice to return l123 to READY mode.
