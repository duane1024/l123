Graph name does not exist

You selected /Graph Name Use or Delete, :Graph Add Named, or :Graph Settings
Graph Named and specified a nonexistent graph name.

Reselect the command and then press F3 (NAME) to see a complete list of
graph names in the current file and the names of other active files.
If you are trying to retrieve or delete a named graph in the current
file, select the appropriate name from the list. If you are trying
to retrieve or delete a named graph in another active file, select the
file name from the list and then select the appropriate graph name in
that file.
