/File New

/File New -- Creates a new worksheet file on disk and in memory before or
after the current file, and moves the cell pointer to cell A1 in the new
file. The new file contains one blank worksheet. You cannot read the new
file on disk into memory because it is a place holder and is blank. If
you do not save the new file, 1-2-3 deletes it from disk.

1. Select /File New.
2. Select Before or After.
    Before creates a new active file in front of the current file.
    After creates a new active file behind the current file.
3. Type a file name and press ENTER, or just press ENTER to use the
    default file name. The first default file name is FILE0001.WK3,
    the next default file name is FILE0002.WK3, and so on.

Use /File Save to create a permanent copy of the file on disk. To password-
protect a new file, use /File Save with a Password.
