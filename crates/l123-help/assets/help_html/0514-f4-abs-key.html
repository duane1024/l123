F4 (ABS) Key

F4 (ABS) Key -- Changes the cell references or range names in a formula when
1-2-3 is in POINT, VALUE, or EDIT mode. Press F4 (ABS) when the cursor
is on or to the right of a reference when you are entering or editing a
formula. 1-2-3 cycles the address through absolute, mixed, and relative
references by entering $ in front of the worksheet letter, column letter,
or row number.

For example, if you entered the formula +A:C5 and pressed F4 (ABS) several
times, 1-2-3 would change it in the following ways:

   +$A:$C$5     Absolute address
   +$A:C$5      Mixed address
   +$A:$C5      Mixed address
   +$A:C5       Mixed address
   +A:$C$5      Mixed address
   +A:C$5       Mixed address
   +A:$C5       Mixed address
   +A:C5        Relative address
