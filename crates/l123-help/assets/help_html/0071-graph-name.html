/Graph Name

/Graph Name -- Lets you name the graphs you create. After you name a graph
and save the file that contains it, you can display and work with that graph
whenever the file is active.

Use         Retrieves and displays a named graph.
Create      Creates or updates a named graph.
Delete      Deletes one named graph.
Reset       Deletes all named graphs in the current file.
Table       Lists all named graphs in the current file.
