Format Menu

Format Menu -- Sets the format, the way l123 displays data for a range
(/Range Format), for an entire worksheet (/Worksheet Global Format), or
along a graph axis (/Graph Options Scale [Y, X, 2Y] Format).

Fixed, Sci, Currency, Comma, General, or Percent
           Displays values in a numeric format.
Date       Displays values in a Date format.
Date Time  Displays values in a Time format.
+/-        Displays values as a series of plus or minus signs or a period.
Text       Displays actual formulas instead of their values.
Hidden     Hides data.
Other      Automatic, Color, Label, and Parentheses formats (only in
           /Range Format and /Worksheet Global Format).
Reset      Resets a range's format to the global format (only in /Range
           Format).
