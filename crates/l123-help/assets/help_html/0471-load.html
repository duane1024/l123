Load

Load -- Reads an add-in file into memory. If the file contains an
application, 1-2-3 prompts you to assign the application to a function key.

1. Select Load.
2. Specify the add-in file you want to read into memory.
3. If the add-in file contains an application, select 1 to assign the
    application to ALT-F7 (APP1), 2 to assign it to ALT-F8 (APP2), or 3 to
    assign it to ALT-F9 (APP3). Select No-Key if you do not want to assign
    the application to a key.

    NOTE 1-2-3 displays only keys that have no application assigned to them.
    For example, if you previously selected 1 to assign an application to
    ALT-F7 (APP1), 1-2-3 displays the following menu: No-Key/2/3.
