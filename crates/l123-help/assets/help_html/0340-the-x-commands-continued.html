The /X Commands (continued)

    /XLprompt~[location]~
        Displays a prompt (which can be as long as the screen width)
        in the control panel and pauses for your response. When
        you type a response and press ENTER, l123 enters your
        response as a label in location. If you do not specify a
        location (/XLprompt~~), l123 enters your response
        in the current cell. /XL produces the same result as
        {GETLABEL}.

    /XM[location]~
        Displays the macro menu that begins at location (or if you do not
        specify a location, that begins in the current cell) and suspends
        macro execution so you can select an item from the menu.
        /XM produces the same result as {MENUBRANCH}.
