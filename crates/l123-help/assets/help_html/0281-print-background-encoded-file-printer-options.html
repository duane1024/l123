/Print [Background, Encoded, File, Printer] Options

/Print [Background, Encoded, File, Printer] Options -- Performs the following:

Header         Enters a line of text just below the top margin of each page.
Footer         Enters a line of text just above the bottom margin of each page.
Margins        Sets margins for the printed page or clears all margins.
Borders        Sets borders and can print the worksheet frame.
Setup          Lets you specify additional printer attributes.
Pg-Length      Sets the length of the page.
Other          Prints cell formulas, eliminates blank header/footer lines,
               and prints data without headers, footers, or page breaks.
Name           Creates, selects, deletes, and lists named print settings.
Advanced       Sets Device, Layout, Fonts, Color, Image, Priority, AutoLf,
               and Wait options.
Quit           Returns you to the /Print [B,E,F,P] menu.

/Print [B,E,F,P] Options settings override /Worksheet Global Default Printer
settings. Lotus recommends you use /Print [B,E,F,P] Options to specify
settings for individual print jobs.
