/Data Distribution

/Data Distribution -- Counts the number of values that fall within specified
intervals in a specified range. A series of intervals forms a bin range. For
example, if the bin range contains the intervals 100 and 200, 1-2-3 counts
the number of values that are less than or equal to 100, greater than 100
and less than or equal to 200, and greater than 200 in a range you specify.

1. Decide on a column for the bin range. The column to the right of the
    bin range must be blank.
2. Enter the intervals for the bin range in ascending order. Do not include
    labels or blank cells in the range you want to use as the bin range.
3. Select /Data Distribution.
4. Specify the range containing the values you want to analyze (the values
    range).
5. Specify the bin range.

1-2-3 enters the results in the column to the right of the bin range. The
last number in this column is the number of values that exceed the largest
bin value. 1-2-3 ignores blank cells and labels in the values range.
