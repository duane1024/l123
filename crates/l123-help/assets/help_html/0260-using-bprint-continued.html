Using BPrint (continued)

Using BPrint (continued) -- To issue a BPrint command, type bprint followed by
one or more of the following arguments, and then press ENTER. You can type the
arguments in any order. If you don't use arguments, BPrint displays a list of
the files waiting to be printed.

Argument     Result

filename     Sends filename (an encoded file or text file) to the printer.
-c filename  Cancels the printing of filename and removes it from the print
              queue. filename must include a path and cannot be the file that
              is currently printing.
-h           Displays a Help screen about BPrint command arguments.
-p=x         Directs printing to parallel port x. Replace x with 1 or 2.
-s=x         Directs printing to serial port x. Replace x with 1 or 2.
-pa          Pauses background printing.
-q           Removes BPrint from memory.
-r           Resumes printing jobs temporarily suspended with -pa.
-t           Stops the current print job and cancels any remaining print jobs.
