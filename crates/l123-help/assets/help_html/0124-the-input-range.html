The Input Range

The Input Range -- The input range is the database table or tables that
contain the information you want 1-2-3 to use when you select /Data
Query. Each input range must be in a 1-2-3 worksheet or an external table.

A 1-2-3 database table is a range of related data organized in rows and
columns in a 1-2-3 worksheet. A worksheet file can contain many database
tables. A database table consists of records and fields.

A record is the information entered in one row of a database table.
A field is a category that each record in the database table has in common.

Remember the following rules when you create a 1-2-3 database table:
    Enter the field names (categories) in the first row of the database
   table. The field names identify the information entered in each column.
   The field names must be unique labels within a database table.
    Enter records in the rows below the field names.
    The entries in each field should be all labels or all values.
