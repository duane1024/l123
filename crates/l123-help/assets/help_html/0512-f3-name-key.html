F3 (NAME) Key

F3 (NAME) Key -- Displays a list of names related to the command you selected
or the formula you are creating. To select a name from the list, highlight
the name and press ENTER. Use F3 (NAME) at the following times:

    After selecting a command that prompts you for a range, press F3 (NAME) to
   display range names in the current file and the names of active files.
    After selecting a command that automatically displays a list of names
   in the control panel (for example, /File Retrieve), press F3 (NAME) to
   display a full-screen list of names.
    To move the cell pointer to a named range in the current file or to
   another active file, press F5 (GOTO) and then press F3 (NAME).
    When entering an @function, type @ and press F3 (NAME) twice to display
   a full-screen list of @functions.
    When entering a macro, type { (open brace) and press F3 (NAME) twice to
   display a full-screen list of macro key names and macro commands.
    When entering a formula, press F3 (NAME) after typing any operator
   ( + -  / ^ ( * ! ; = < > # or &) to list range names in the current file
   and names of other active files.
