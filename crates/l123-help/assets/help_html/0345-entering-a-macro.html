Entering a Macro

Entering a Macro -- A macro consists of a series of instructions in a
worksheet. These instructions tell 1-2-3 what actions to perform when
you run the macro. There are two types of macro instructions: keystroke
instructions and macro commands.

Keystroke instructions represent keys on the keyboard. There are two types
    of keystroke instructions: those that consist of a single character,
    such as / or w, and those that consist of a macro key name within
    { } (braces), such as {RIGHT}. In a macro, / (slash) displays the
    main menu and ~ (tilde) represents the ENTER key.

    For example, the macro /wir~{RIGHT} inserts a row in the worksheet
    with /Worksheet Insert Row and then moves the cell pointer to the right.
    This macro contains six keystroke instructions.

Macro commands cause 1-2-3 to perform built-in programming
    functions. See the Macro Index for a complete list of these commands.
