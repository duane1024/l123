Error loading driver, file name-

The following conditions could have caused this error:

* Not enough memory was available to read the file named in the error
   message. Do one of the following: free some memory and try to print
   again; or use /Worksheet Global Default Printer Name and Interface and
   /Worksheet Global Default Update to make the printer you want to use
   your default printer. Then save your active files, end 1-2-3, and
   restart 1-2-3.

* The file named in the error message was damaged or deleted from your
   hard disk. Use the Install program to reinstall your equipment.
   For information on starting Install, see "Starting Install After
   First-Time Installation" in Chapter 5 of Getting Started.
