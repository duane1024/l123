The Criteria Range

The Criteria Range -- Tells 1-2-3 which records to search for in the input
range. After you set up the criteria range, use /Data Query Criteria to tell
1-2-3 where the criteria range is located. To create a criteria range

1. Choose a blank range either several rows above, to the right of, or to
    the left of the input range or in a separate worksheet.
2. In the first row of the criteria range, copy some or all of the field
    names from the input range. You must copy the field names exactly as
    they appear in the input range. When a field name that you want to use
    in the criteria range appears in more than one database table, you must
    enter the table name, then a . (period), and then the field name. For
    example, SALES92.QTR1 refers to QTR1 (the field) in the table named SALES92
    and SALES93.QTR1 refers to QTR1 (the field) in the table named SALES93.
3. In the second and subsequent rows of the criteria range, enter your
    criteria under the related field name. The different types of criteria
    are Label Criteria, Value Criteria, Formula Criteria, and
    Multiple-Field Criteria.
