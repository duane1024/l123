    If more than one file is active, 1-2-3 displays [ALL MODIFIED FILES].
       To save all modified files with their existing file names, press ENTER.
       To save only the current file, press F2 (EDIT) to display the file name

4. If you are updating files on disk, select Cancel, Replace, or Backup.
    Cancel     Returns 1-2-3 to READY mode without saving any files.
    Replace    Writes over the files on disk with copies of the active files.
    Backup     Renames the files on disk with the extension .BAK and saves
               active files with their existing file names.

    CAUTION  When you select Replace, 1-2-3 erases the files that you are
    replacing on disk before it saves the active versions of the files. If
    1-2-3 displays an error message ('Disk full') after you select Replace,
    save the active files on another disk.
