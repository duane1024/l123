/Data Sort Primary-Key

/Data Sort Primary-Key -- Specifies the column (field) the contents of which
determine the primary order of the rows (records) in the range to be sorted.

1. Select /Data Sort Primary-Key.
2. Specify the address of any cell in the field you want l123 to use to
    determine the new order for your records.
3. Select A (ascending) or D (descending).

    Ascending order is from A to Z if you are sorting labels that begin with
    letters or from smallest to greatest value if you are sorting values;
    descending order is from Z to A or from greatest to smallest value. If
    the primary-key field contains both labels and values, ascending order
    places labels before values; descending order places values before labels.

If the field contains a mixture of different types of labels -- labels
beginning with letters, labels beginning with numbers, and labels beginning
with other characters -- the collating sequence determines the final order
of the labels. See Sort Order for more information on collating sequences.
