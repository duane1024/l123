Cannot convert extension. File already exists in memory.

You used /File Retrieve or /File Open to read a l123 Release 1A or
Symphony file into memory, and the file name matched the name of an
active file. l123 could not convert the file to .WK3 format because
you had an active file by that name.

Before you try to read the file into l123, you can

* Delete the active file from memory with /Worksheet Delete File.
* Save the active file with a different name.
* Rename the Release 1A or Symphony file on disk.
