Graph Types

Graph Types -- Descriptions (continued)

Stack-Bar   Stacked bar graphs compare values by stacking them one on
             top of another. For stacked bar graphs, 1-2-3 uses the X data
             range as the x-axis labels and the A - F data ranges as portions
             of the bars. The A-range values form the bottom portion of the
             bars, the B-range values form the next-highest portion of the
             bars, and so on.

Pie         Pie charts relate two or more positive values to one another by
             depicting the values as slices of a pie. For pie charts, 1-2-3
             uses the A data range as the pie slices and the X data range as
             the pie-slice labels. Use the B data range to set colors or
             hatch patterns for the pie slices (enter values from 1 through 14),
             to hide pie slices (enter a negative number), or to explode
             slices away from the pie (add 100 to the B-range values).
             Use the C data range to hide the percent labels 1-2-3 displays
             for pie slices (enter 0).
