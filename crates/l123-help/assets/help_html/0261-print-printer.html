/Print Printer

/Print Printer -- Prints data or graphs on a printer.

1. Select /Print Printer.
2. Select Range.
3. Specify the range or ranges you want to print. The print range can
    include any number of ranges and named graphs in any active file. See
    /Print [B,E,F,P] Range for more information on specifying a
    print range.
4. (Optional) Select Options (/Print [B,E,F,P] Options) to set
    different print options.
5. Make sure the printer is online and the paper is at the top
    of a page.
6. Select Align.
7. Select Go to print the range.
8. (Optional) Select Page to advance the paper to the top of the next page
    and to print the footer (if you specified one) on the last page of the
    print job.
9. Select Quit to complete the print job and return 1-2-3 to READY mode.
