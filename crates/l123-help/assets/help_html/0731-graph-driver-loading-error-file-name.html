Graph driver loading error, file name

Graph driver loading error, file name-

The following conditions could have caused this error:

* Not enough memory was available to read into memory the file named in
   the error message. Free some memory so 1-2-3 can read the file into
   memory. Increasing Available Memory lists various methods you can use
   to free memory.
* The file named in the error message was damaged or deleted from your
   hard disk. Use the Install program to reselect your equipment. For
   information on starting Install, see "Starting Install After First-Time
   Installation" in Chapter 5 of Getting Started.
* 1-2-3 cannot use the file named in the error message because you are
   already using the maximum number of files DOS allows. To solve this
   problem, do one of the following: Use /Worksheet Delete File to
   delete one or more of your active files; use /File Admin Reservation
   Release to release the reservation of one or more active files;
   or end another application program, if one is running.
