Procedure for /Data Parse

1. Select /File Import Text to import a text file as one column of long
    labels.
2. Position the cell pointer in the cell that contains the first long label
    you want to parse.
3. Select /Data Parse Format-Line Create. 1-2-3 inserts a new row into the
    worksheet at the location of the cell pointer and enters the format line
    in the new row.
4. (Optional) Select Format-Line Edit and then edit the format line.
5. (Optional) Select Quit, move the cell pointer to another label, and repeat
    steps 3 and 4 for each subsequent label or set of labels that requires a
    format different from the format shown in the preceding format line.
6. Select Input-Column and specify the single-column range that contains
    all the long labels and format lines.
7. Select Output-Range and specify the range where you want the parsed data
    to go.
8. Select Go.
