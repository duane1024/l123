Saving Files in 1-2-3 Release 2 (.WK1) Format (continued)

The following list describes what happens to Release 3 features when saved
in a .WK1 file:

    Labels that contain more than 240 characters are truncated.
    Formulas that contain more than 240 characters are saved in their
   entirety, but Release 2 truncates the formulas at 240 characters if you
   try to edit them.
    Release 3 @functions and @functions that have new arguments are treated
   as add-in @functions, which are saved as @? with a list of arguments
   following the ?. Cells that contain these @functions evaluate to NA.
    If the file contains formulas linked to other files, the links are
   converted to @@("<<filename>>range").
    Formulas that contain undefined range names evaluate to ERR.
    All range name notes, formula annotations, and Release 3 /Data,
   /Graph, /Print, and /Worksheet settings are lost.
    New formats are changed to the default Release 2 formats.
