Background printing is currently active, cannot access printer

You selected /Print Printer when BPrint was printing a background print job
on your printer.

You can wait for BPrint to complete the background print job or select /Print
Cancel to cancel the background print job. Then try to use /Print Printer
again.

If you selected /Print Suspend, make sure you select /Print Resume to resume
the background print job or /Print Cancel to cancel the background print job
before you try to use /Print Printer again.
