/Print [Background, Encoded, File, Printer] Options Pg-Length

/Print [Background, Encoded, File, Printer] Options Pg-Length -- Sets the
length of each page in standard lines. Use this command to override the
/Worksheet Global Default Printer Pg-Length setting for the current session.
The 1-2-3 default page length is 66 lines, which is appropriate for printers
that print 6 lines per inch on 11" paper.

1. Select /Print [B,E,F,P] Options Pg-Length.
2. Type a number from 1 through 1000 and press ENTER.

See Printer Information for page-length settings for specific printers.

NOTE  If you use /Print [B,E,P] Options Advanced Layout Line-Spacing Compressed
to print compressed text, 1-2-3 automatically adjusts the number of
compressed lines (usually 88) for a page that is 66 standard lines long.
However, if you use /Print [B,E,P] Options Setup to specify a setup string
to set line spacing, 1-2-3 does not automatically adjust the number of
lines, so you will need to change the page-length setting.
