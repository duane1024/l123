No external table specified for creation

The following conditions could have caused this error:

* You selected /Data External Create Go to create an external table, but
   did not specify a name for the external table. Use /Data External Create
   Name to specify a table name.

* You selected /Data External Create Name, returned l123 to READY mode
   without selecting Go, and then selected /Data External Create Go. You
   must respecify the table name with /Data External Create Name before you
   select Go again to create the table.
