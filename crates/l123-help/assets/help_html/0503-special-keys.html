Special Keys

Special Keys

ALT            Used in combination with the function keys, performs special
                operations. Used with a single-letter macro name, runs a
                a macro.

CTRL-BREAK     Cancels the current 1-2-3 command or operation and returns
                1-2-3 to READY mode.

ESC            Cancels the current entry in the control panel, cancels
                the current range in a command, or returns you to the
                previous menu.
