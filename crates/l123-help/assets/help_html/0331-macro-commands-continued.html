Macro Commands (continued)

    {INDICATE [string]}
        Displays string as the mode indicator in the upper right
        corner of the screen. {INDICATE} with no argument restores
        standard operation of the mode indicator.

    {LET location,entry}
        Enters a number or label in location. For the entry argument, you
        can use a number, literal string, formula, or reference to a cell
        containing a number, label, or formula. If you use a formula, l123
        enters the result in location. You can add the suffix :s or :v to
        the entry argument to tell l123 whether to treat the argument as a
        literal string or to evaluate the argument before entering it.

    {LOOK location}
        Records in location the first keystroke in the typeahead buffer.
        If the buffer is empty, l123 enters ' (apostrophe) in location.
