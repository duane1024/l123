ALT-F2 (RECORD) Copy

ALT-F2 (RECORD) Copy -- As you use l123, the program records your
keystrokes in the record buffer, a 512-byte area of computer memory.
To create a macro by copying keystrokes from the record buffer to
a worksheet, do the following:

1. Move the cell pointer to wherever you will be starting the task you
    want to record.
2. Press ALT-F2 (RECORD) and select Erase to clear the record buffer.
3. Perform the task you want to record.
4. Press ALT-F2 (RECORD) and select Copy to display the contents of the
    record buffer in the control panel. Edit those keystrokes, if necessary.
5. Select the keystrokes to copy: Move the cursor to the beginning or
    end of the keystrokes; press TAB to anchor the cursor; use the
    pointer-movement keys to highlight the keystrokes; and press ENTER.
6. Specify the range to which you want to copy the selected keystrokes.
7. Assign a macro range name to the first cell of the range to which you
    copied the keystrokes.
