/Worksheet Global Prot

/Worksheet Global Prot -- Turns global protection on or off for the
current worksheet. If the current file is in GROUP mode, this command
affects all worksheets in the file.

1. Select /Worksheet Global Prot.
2. Select Enable or Disable.

    Enable turns on global protection. When protection is on, 1-2-3 displays
    PR in the control panel when the cell pointer is on a protected cell.
    You cannot change data in the current worksheet except in cells you
    have unprotected with /Range Unprot.

    Disable (default) turns off global protection.
