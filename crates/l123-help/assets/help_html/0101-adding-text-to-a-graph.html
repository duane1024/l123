Adding Text to a Graph

Adding Text to a Graph -- The following commands add text to a graph and
format graph text.

/Graph X
      Adds x-axis labels to a line, bar, stacked bar, mixed, or HLCO
      graph. Adds pie-slice labels to a pie chart.
/Graph Options Data-Labels
      Labels the points or bars in a graph, using worksheet data as the
      labels.
/Graph Options Legends
      Creates legends for a graph's data ranges. The legends, located
      below the graph, identify the data range represented by each
      symbol, color, or hatch pattern in the graph.
/Graph Options Titles
      Adds graph titles, axis titles, and footnotes to a graph.
/Graph Options Advanced Text
      Sets colors, fonts, and sizes for graph text. l123 uses these
      settings both for displaying the graph and for printing it.
