Entering Formulas (continued)

    A formula can contain up to 512 characters.

    When a formula looks like a date in Date 1, Date 2, or Date 4
   format, begin the formulas with +. Otherwise, 1-2-3 enters a date number
   instead of the formula. For example, +30-NOV-95 is a valid formula.

    To annotate a formula (include an explanatory note with the formula),
   type ; (semicolon) immediately after the formula and then type the note.
   The annotation does not appear in the cell with the formula unless the
   cell is formatted as Text; it appears only in the control panel when
   the cell pointer is on the cell.

    To display a formula instead of its result, use /Range Format Text
   or /Worksheet Global Format Text.
