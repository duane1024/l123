/Worksheet Global

/Worksheet Global -- Some of the following commands affect only the current
worksheet or file, and others affect all active worksheets and files.

Format      Sets the global cell format for the current worksheet.
Label       Sets the global label alignment for the current worksheet.
Col-Width   Sets the global column width for the current worksheet.
Prot        Turns global protection on and off for the current worksheet.
Zero        Sets the display for cells with a value of zero in the
            current worksheet.
Recalc      Sets the recalculation method for all active
            worksheets and files. 1-2-3 uses one recalculation
            setting at a time.
Default     Changes the 1-2-3 configuration settings, which affect
            all active worksheets and files.
Group       Turns GROUP mode on and off for the current file.
            When a file is in GROUP mode, the following Worksheet Global
            commands affect all worksheets in that file: Format, Label,
            Col-Width, Prot, and Zero.
