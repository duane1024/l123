/Print [Background, Encoded, Printer] Options Advanced Layout

/Print [Background, Encoded, Printer] Options Advanced Layout -- Performs the
following tasks:

Pitch            Determines how much space 1-2-3 allocates for each
                 printed character and, therefore, how many characters
                 fit on a printed line.
Line-Spacing     Determines how much space 1-2-3 allocates for each
                 printed line and, therefore, how many lines fit on a
                 printed page.
Orientation      Determines whether 1-2-3 prints in portrait mode or
                 landscape mode (if available on the printer).
Quit             Returns you to the /Print [B,E,P] Options Advanced menu.
