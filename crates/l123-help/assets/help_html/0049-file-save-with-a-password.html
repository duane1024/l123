/File Save with a Password

1. Make sure the cell pointer is in the file you want to password-protect.
2. Select /File Save.
3. If l123 displays [ALL MODIFIED FILES], press F2 (EDIT) to display the
    current file name.
4. Press the space bar once to enter a space after the current file name,
    or edit the file name and then press the space bar once.
5. Type p and press ENTER.
6. Type a password and press ENTER. A password can include up to 15
    characters. l123 is case-sensitive for passwords.
    CAUTION  Remember the password. You can read the file into memory again
    only if you enter the exact password.
7. Type the same password again and press ENTER.
8. If you are updating a file, select Replace to save the file with the
    password.
