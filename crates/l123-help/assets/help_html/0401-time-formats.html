Time Formats

Time Formats -- In 1-2-3, time is represented by decimal values .00000 (the
time number for 12:00 AM) through .99999 (the time number for 11:59:59 PM).
To display a value as time, enter a time number in the worksheet or an
@function that calculates a time number, such as @NOW or @TIME.
Then format the cell to display the time.

    Time Format                          Examples for 1:30 PM
    1  HH:MM:SS AM/PM                     1:30:00 PM
    2  HH:MM AM/PM                        1:30 PM
    3  Long Intn'l                       13:30:00
       HH:MM:SS (default), HH.MM.SS,
       HH,MM,SS, or HHhMMmSSs
    4  Short Intn'l                      13:30
       HH:MM (default), HH.MM,
       HH,MM, or HHhMMm

Use /Worksheet Global Default Other International Date Time to change the
Long Intn'l and Short Intn'l Time formats.
