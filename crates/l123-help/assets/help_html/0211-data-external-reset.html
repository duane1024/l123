/Data External Reset

/Data External Reset -- Disconnects an external table, ending all data
exchange between l123 and the external table. After you use this command,
any data queries or database @functions that refer to the table may result
in errors. l123 will not update these queries or @functions until you
select /Data External Use and specify that table again.

1. Select /Data External Reset.
2. Specify the range name of the external table you want to stop using.

If the table you specified was the only one in use in the associated
external database, l123 ends the connection to the external database. If
the external database containing the table you specified was the only one
in use for a particular DataLens driver, l123 disconnects from the driver.
