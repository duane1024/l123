/Graph Options Data-Labels

/Graph Options Data-Labels -- Sets labels for individual data points or bars
in a graph, using worksheet data as the labels.
1. Select /Graph Options Data-Labels.
2. Select A, B, C, D, E, or F to set data labels for one data range, or
    select Group to set data labels for all data ranges at once.
3. Specify the range that contains the data labels (the data-label range).
    The data-label range should be the same size as the corresponding data
    range. If you selected Group, the data-label range should have the same
    number of columns or rows as all the data ranges combined.
4. If you selected Group, select Columnwise or Rowwise to specify whether
    l123 should divide the data-label range by columns or by rows.
5. Select Center, Left, Above, Right, or Below to specify the placement of
    the data labels relative to the corresponding data. (If you are creating
    data labels for bars, you can specify Above or Below only. Selecting
    Center, Left, or Right is equivalent to selecting Above.)
6. Select Quit to return to the /Graph Options menu.

NOTE  For pie charts, use /Graph X to set labels for the pie slices.
