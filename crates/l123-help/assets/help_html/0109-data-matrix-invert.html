/Data Matrix Invert

/Data Matrix Invert -- Creates the inverse of a square matrix.

1. Select /Data Matrix Invert.
2. Specify the matrix range you want to invert.
    The matrix range must have the same number of columns as rows and can
    contain up to 80 columns and 80 rows.
3. Specify the output range where you want l123 to enter the inverted
    matrix. Specify either the entire range or only the first cell.

    CAUTION  l123 writes over any existing data in the specified range,
    so make sure the output range does not contain data you need.
