/Data External Create Name

/Data External Create Name -- Connects to the external database in which you
want to create a new table and assigns a name to the new table.

NOTE  You should be familiar with the procedure for creating a new external
table before you use this command.

1. Select /Data External Create Name.
2. Specify the DataLens driver for the external database in which you want
    to create the table.
3. Specify the external database in which you want to create the table.
4. Type a new name for the new table and press ENTER.

    NOTE  Make sure you do not specify a table name with more characters
    than the external database allows.
