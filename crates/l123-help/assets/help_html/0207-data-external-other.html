/Data External Other

/Data External Other -- Performs the following tasks:

Refresh       Re-executes Data Query and Data Table commands and
              updates database @functions at regular intervals.
Command       Sends a command to a database management program.
Translation   Selects the character set a DataLens driver uses
              to translate information extracted from or copied
              to external tables in an external database.
