ALT-F4 (UNDO)

ALT-F4 (UNDO) -- Lets you undo some l123 operations. To turn on the undo
feature, use /Worksheet Global Default Other Undo Enable.

1. Press ALT-F4 (UNDO) when undo is on.
2. Select Yes to undo the most recent command that changed the worksheet.
    Select No to return l123 to READY mode without undoing the command.

NOTE  For information about the undo feature, see "Undoing an Action"
in Chapter 1 of Reference.
