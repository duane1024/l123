File Password Protection

File Password Protection -- If a file contains sensitive or confidential
data, you might want to protect it with a password. You can limit access
to a worksheet file by saving it with a password when you use /File Save
or /File Xtract.

When you protect a worksheet file with a password, only a user who
knows the password can retrieve or open the file or combine data from
it into another file. No DOS command can display the contents of the
worksheet file.

To save a file with a password, see /File Save with a Password.
To change a password, see Changing a File Password During /File Save.
To delete a password, see Deleting a File Password During /File Save.
To extract a file with a password, see /File Xtract with a Password.
To read a password-protected file, see Reading Password-Protected Files.
