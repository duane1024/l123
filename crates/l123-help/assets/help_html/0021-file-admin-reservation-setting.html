/File Admin Reservation Setting

/File Admin Reservation Setting -- Determines if the reservation is
automatically available when you read the file into memory or if you must
manually get the reservation the next time you open or retrieve the file.

1. Select /File Admin Reservation Setting.
2. Select Automatic or Manual.
    Automatic   Sets the file's reservation setting so the first person who
                reads the file into memory gets the reservation. No one else
                can get the reservation until the first user releases it.
    Manual      Sets the file's reservation setting so the reservation is
                never automatically available. You have read-only access
                when you retrieve or open the file. To get the reservation,
                use /File Admin Reservation Get.

Use /File Save to save the reservation setting in the file.
Use /File Admin Seal to seal the file's reservation setting so no one
can change it.
