/Print [Background, Encoded, File, Printer] Options Other

/Print [Background, Encoded, File, Printer] Options Other -- Prints the
displayed data or the formulas underlying the data, suppresses or prints
headers, footers, and page breaks.
1. Select /Print [B,E,F,P] Options Other and select one of the following:
    As-Displayed   Prints the data as it appears on the screen, returning
                   l123 to regular printing after you have selected
                   Cell-Formulas.
    Cell-Formulas  Prints the following information about each cell in
                   the print range, one cell per line: the cell address,
                   cell format, protection status, the cell contents, and
                   any annotation in the cell.
    Formatted      Prints with the current formatting options, returning
                   l123 to regular formatting after you have selected
                   Unformatted.
    Unformatted    Prints without page breaks, headers, footers, and top and
                   bottom margins. Use Unformatted if you are printing
                   to a text file and do not want spaces in the file for
                   page breaks, headers, or footers.
