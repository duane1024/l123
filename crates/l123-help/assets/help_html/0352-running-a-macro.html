Running a Macro

Running a Macro -- You run a macro in one of two ways: with ALT or
with ALT-F3 (RUN). The method you use depends on the macro's name.

NOTE  Before you run a macro, use /File Save to save any files that
the macro affects so that if the macro produces unexpected results, you
can easily retrieve the original versions of the files.

Use ALT to run a macro when the macro's name consists of a backslash
and a single letter (for example, \N).

1. Make sure the file that contains the macro is the current file or
    that no other active file contains a macro of the same name.
2. Press and hold down ALT and then press the letter in the macro's
    name. For example, to run a macro named \N, press ALT-n.
