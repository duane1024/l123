Incompatible character set

You retrieved a .WK1 file that contains a character set other than
LICS (Lotus International Character Set).

Do not save the file. Saving the file could cause loss of data
because 1-2-3 will write over the file using LICS, which is part of
LMBCS (Lotus Multibyte Character Set), the character set used in this
release of 1-2-3.
