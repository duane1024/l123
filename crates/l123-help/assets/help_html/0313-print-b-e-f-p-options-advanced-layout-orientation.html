/Print [B,E,F,P] Options Advanced Layout Orientation

/Print [B,E,F,P] Options Advanced Layout Orientation -- Determines whether
1-2-3 prints graphs and text in portrait mode (across the width of the paper)
or landscape mode (sideways across the length of the paper), if landscape
mode is available on your printer.

1. Select /Print [B,E,P] Options Advanced Layout Orientation.
2. Select Portrait (default) or Landscape.

When you select Landscape, 1-2-3 prints both text and graphs sideways on
the page. Use /Print [B,E,P] Options Advanced Image Rotate to rotate graphs
but not text.

NOTE  When you print in landscape mode, you may need to change the margin
and page-length settings.
