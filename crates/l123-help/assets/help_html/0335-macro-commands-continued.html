Macro Commands (continued)

    {RECALC location,[condition],[iterations]}
        Recalculates the values in location row by row. The optional
        condition argument tells 1-2-3 to repeat the recalculation until
        condition is true. The optional iterations arguments tell 1-2-3
        to perform the specified number of recalculation passes.

    {RECALCCOL location,[condition],[iterations]}
        Like {RECALC} except recalculation proceeds column by column.

    {RESTART}
        Used in a subroutine to clear the subroutine stack. When 1-2-3
        encounters {RESTART}, it finishes the current subroutine,
        clears the subroutine stack, and ends the macro.
