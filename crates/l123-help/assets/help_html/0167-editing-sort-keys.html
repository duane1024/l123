Editing Sort Keys

Editing Sort Keys -- To edit information for a primary, secondary, or extra
sort key -- for example, to change the sort order of a sort key -- follow
the steps below.

1. If you are editing this key         Select this command

    Primary sort key                    /Data Sort Primary-Key
    Secondary sort key                  /Data Sort Secondary-Key
    An extra sort key                   /Data Sort Extra-Key

2. If you are editing an extra sort key, specify the number of the sort key
    by typing a number (or pressing    or   ) and pressing ENTER.
    1-2-3 displays the current address for that sort key.

3. Specify a new address and/or sort order for the sort key.
