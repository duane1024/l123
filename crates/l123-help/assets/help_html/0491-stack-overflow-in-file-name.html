Stack overflow in "file_name"

Stack overflow in "file_name" --  Your add-in uses too much program
memory to run in 1-2-3. If you are a developer, you must edit your program
so it takes less memory, or use compiler option Stack Size, which makes more
memory available to your program. Otherwise, contact your add-in developer.

Stop statement in "file_name" returns value "value" -- Your add-in called the
procedure Stop, which causes the add-in to halt when some condition,
specified by "value," occurs. Refer to the documentation provided by
your add-in developer if it lists the values and the conditions they refer
to, or else contact your add-in developer.

Unhandled "signal_name" signal in "file_name" -- A statement in your add-in
raised a signal, but did not handle it. If you are a developer, add an
exception handler to that statement. Otherwise, you must contact your add-in
developer.
