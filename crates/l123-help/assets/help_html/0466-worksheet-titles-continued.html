/Worksheet Titles (continued)

You cannot use the pointer-movement keys to move the cell pointer to the
cell that contains a worksheet title if l123 is in READY mode. To move
the cell pointer to a worksheet title, use F5 (GOTO).

When you press F5 (GOTO) and specify a cell in a title row or column,
l123 displays a second set of titles immediately below or to the
right of the titles so you can edit them. To clear the second set of
titles, press PG DN and then PG UP (for rows), or CTRL-   and then
CTRL-   (for columns).

If you move the cell pointer into the titles area when you specify a
range during a command or while entering a formula (in POINT mode),
l123 will temporarily display a second set of titles. When you return
l123 to READY mode, however, the second set disappears.
