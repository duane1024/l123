Error loading driver file

The following conditions could have caused this error:

* Not enough memory was available to load the printer, display, or DataLens
   driver you specified. To free memory, do one or more of the following:

   Save your active l123 file(s), use /Worksheet Erase, and then retrieve
   your file(s) again.

   If the undo feature is turned on, use /Worksheet Global Default Other
   Undo Disable to turn the undo feature off.

   Delete worksheets, files, and data that you no longer need in memory.

   If you have programs in memory besides l123, end one or more of those
   programs. For additional information on freeing memory, see Appendix D
   of Reference.
