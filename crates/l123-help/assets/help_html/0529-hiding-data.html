Hiding Data

Hiding Data -- Prevents displaying data in the worksheet and printing data.
You can use several commands to hide data in a l123 worksheet.

/Range Format Hidden
    Hides data in a range. The data appears in the control panel when the
    cell pointer is on a hidden cell unless the worksheet is globally
    protected.
/Worksheet Column Hide
    Hides columns to prevent displaying and printing data entered there.
/Worksheet Global Zero
    Hides zeros in cells in the current worksheet that contain either the
    number zero or a formula that evaluates to zero.
/Worksheet Global Format Hidden
    Hides data in a worksheet. The data appears in the control panel when
    the cell pointer is on a hidden cell.
/Worksheet Hide
    Hides and redisplays one or more worksheets.
