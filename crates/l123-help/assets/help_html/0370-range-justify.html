/Range Justify

/Range Justify -- Rearranges a column of labels so the labels fit within a
width you specify. To use this command, global protection must be off for
the worksheet that contains the column of labels.

1. Move the cell pointer to the first cell in the column of labels
    you want to justify.
2. Select /Range Justify.
3. Specify the justify range.
    The width of the justify range determines the maximum width of the
    justified labels. The depth of the justify range determines how l123
    handles data below the range.

       Specify a single-row justify range when all cells below the labels
      are blank or when moving data below the labels is acceptable.
       Specify a multiple-row justify range to prevent movement of data
      below the range. Be sure the range is wide and deep enough to hold the
      entire series of justified labels.
