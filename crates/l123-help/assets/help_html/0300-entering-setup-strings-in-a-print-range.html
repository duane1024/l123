Entering Setup Strings in a Print Range

Entering Setup Strings in a Print Range -- You can enter a setup string
in a print range to switch from one print style to another within the
print range. To enter a setup string in a print range:

1. Insert a blank row above the range whose print style you want to change.
2. In the first column of the blank row, enter || (double vertical bars)
    and the setup string.
3. At the point where you want to turn off the print style, insert
    another blank row and enter the appropriate setup string.

    For example, to print a line in double-strike mode on an Epson printer,
    enter ||\027G. To return to standard print, enter ||027H at the point
    where you want to turn off double-strike mode. l123 does not print
    the blank rows that contain the setup strings.
