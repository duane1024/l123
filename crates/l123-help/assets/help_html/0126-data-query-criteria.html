/Data Query Criteria

/Data Query Criteria -- Specifies the range that contains the selection
criteria for records in the database table (the input range). You must use
/Data Query Input and /Data Query Criteria before you can complete other
Data Query commands.

1. Select /Data Query Criteria.
2. Specify the criteria range. The criteria range must be in one worksheet.
       The first row of the criteria range must contain some or all of the
      field names from the input range.
       Each row below the field names should contain the criteria for
      selecting records from the database table. The criteria can be labels,
      numbers, or formulas entered in one or more rows below the field names.

    The different types of criteria are Label Criteria, Value Criteria,
    Formula Criteria, and Multiple-Field Criteria.
