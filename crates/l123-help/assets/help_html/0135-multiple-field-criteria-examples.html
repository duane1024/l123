Multiple-Field Criteria Examples

Multiple-Field Criteria Examples -- The input range is A:A1..A:D6. If you
specify A9..D10 as the criteria range, l123 matches only the record in
row 5 where Name is Robinson and Month is May. If you specify F1..G3 as
the criteria range, the records in rows 3, 4, and 6 match the criteria
where the Month is June or where Sales is greater than 2000.
