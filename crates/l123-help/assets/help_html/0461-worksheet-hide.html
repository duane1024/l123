/Worksheet Hide

/Worksheet Hide -- Hides and redisplays one or more worksheets.

1. Select /Worksheet Hide.
2. Select Enable or Disable.
3. If you selected Enable, specify the range of worksheets to hide.
    If you selected Disable, specify the range of worksheets to redisplay.
    l123 temporarily displays hidden worksheets with an * (asterisk)
    next to the worksheet letter.

Note the differences between /Worksheet Hide and the following commands:

/Range Format Hidden hides data in a range.
/Worksheet Global Format Hidden hides all data in a worksheet except in
    cells formatted with /Range Format.
/Worksheet Column Hide hides columns in a worksheet.
