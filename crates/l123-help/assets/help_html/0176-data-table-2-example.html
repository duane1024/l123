principal and interest rate in an @PMT formula to calculate monthly loan
payments. The table range is B3..E7, input cell 1 is B1, and input cell 2
is B2. 1-2-3 calculates the results in the table range by substituting
pairs of input values (B4..B7 and C3..E3) in the formula. Find the monthly
payment by locating the intersection of a column and row. For example,
the monthly payment on a $90,000 loan at 10% interest is $789.81.
