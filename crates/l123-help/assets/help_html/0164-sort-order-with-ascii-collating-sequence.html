Sort Order with ASCII Collating Sequence

Ascending  1. Blank cells
            2. Labels -- all characters evaluated in LMBCS order
            3. Values in numerical order
Descending 1. Values in reverse numerical order
            2. Labels -- all characters evaluated in reverse LMBCS order
            3. Blank cells

Uppercase letters precede lowercase letters. The following example shows
how 1-2-3 sorts labels when some contain uppercase letters and some contain
lowercase letters:

      Original order           Ascending        Descending
      AAAA                     AAAA             bbbb
      aaaa                     BBBB             aaaa
      BBBB                     aaaa             BBBB
      bbbb                     bbbb             AAAA
