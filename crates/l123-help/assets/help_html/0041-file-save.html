/File Save

/File Save -- Saves worksheet data and settings in worksheet files on
disk. You use /File Save both to create new files on disk and to update
existing files. You can save just the current file or all modified active
files simultaneously.

See File Password Protection for the procedure to password-protect a file
when you save it. See Saving Files in 1-2-3 Release 2 (.WK1) Format if
you want to save a file in Release 2 (.WK1) format.

1. Make sure the cell pointer is in the file you want to save if you want
    to save only the current file but are using several active files.
2. Select /File Save.
3. If only one file is active, 1-2-3 displays that file's name. If you
    are saving the file for the first time, 1-2-3 displays the default
    file name (for example, FILE0001.WK3).
       To save the file with the displayed file name, press ENTER.
       To change the file name, either edit the name or type a new name
      and press ENTER.
