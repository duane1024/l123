/Range Label

/Range Label -- Changes the alignment of labels in a range by changing
their label prefix (' for left-aligned, " for right-aligned, ^ for
centered). Labels that exceed the width of a column appear left-aligned
regardless of the label prefix.

1. Select /Range Label.
2. Select Left, Right, or Center.
3. Specify the range of labels whose alignment you want to change.
