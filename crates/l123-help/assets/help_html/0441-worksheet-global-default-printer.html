/Worksheet Global Default Printer

/Worksheet Global Default Printer -- Determines the printer settings 1-2-3
uses at the start of each session.

NOTE  Printer settings you select with /Print [B,E,F,P] Options override
corresponding /Worksheet Global Default Printer settings. Lotus recommends
you use /Print [B,E,F,P] Options to specify the settings for individual
print jobs.

1. Select /Worksheet Global Default Printer and select one of the following:

Interface                  Identifies the printer port you are using.
AutoLf                     Sets the automatic line feed to Yes or No.
Left, Right, Top, Bottom   Sets the margins.
Pg-Length                  Sets the length of each page.
Wait                       Tells 1-2-3 whether to pause after printing
                            each page.
Setup                      Lets you enter a setup string.
Name                       Identifies the printer you are using.
