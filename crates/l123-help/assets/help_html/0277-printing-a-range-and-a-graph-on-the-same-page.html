Printing a Range and a Graph on the Same Page

1. Select /Print Printer Range.
2. Specify the range you want to print. See /Print [B,E,F,P] Range
    for more information on specifying a range to print.
3. (Optional) Select Options to set print options, and then select Quit
    to return to the menu that begins with Range.
4. Make sure the printer is online and the paper is at the top
    of a page.
5. Select Align.
6. Select Go to print the range.
7. Select Image.
8. Select Current or Named-Graph.
9. Select Options Advanced Image to specify how you want your graph to
    appear on the page. When you print text and a graph on the same page,
    you may need to use /Print [B,E,P] Options Advanced Image Image-Sz to
    make the graph fit on the page.
