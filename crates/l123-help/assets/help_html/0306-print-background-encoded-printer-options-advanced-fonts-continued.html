/Print [Background, Encoded, Printer] Options Advanced Fonts (continued)

Font 1 is the default that l123 uses for the print range, headers, footers,
borders, and the frame. The font depends on the printer you use, but the
following table shows typical font assignments. The width of printed text
may vary from font to font.

    Number     Font

    1          Serif
    2          Bold serif
    3          Italic serif
    4          Bold italic serif
    5          Sans serif
    6          Bold sans serif
    7          Italic sans serif
    8          Bold italic sans serif
