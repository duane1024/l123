Printing an Encoded File

Printing an Encoded File -- To print an encoded file on a printer, use
the DOS COPY command.

Enter the following command at the DOS prompt:

     copy [path]filename/b device

For example, enter copy c:\123r34\budget.enc/b lpt1 to print the file
BUDGET.ENC in the C:\123R34 directory on device LPT1.

For more information about the DOS COPY command, see your DOS documentation.
