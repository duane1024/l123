Unable to reserve file: it has read-only access

You used /File Admin Reservation Get to try to get a reservation for a
file that was marked read-only on disk, or you tried to save a file that
is marked read-only on disk.

You can clear the read-only attribute from the file on disk by using
the DOS ATTRIB command. Then you can read the file into 1-2-3, get
the file's reservation, and save the file.
