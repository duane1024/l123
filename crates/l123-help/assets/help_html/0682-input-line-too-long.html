Input line too long

The following conditions could have caused this error:

* You used /File Import with a text file that had a line longer than
   511 characters. This error occurs when the file you are importing
   does not have a carriage return at the end of each line. As a result,
   l123 cannot find the end of the file's lines. Use the text editor with
   which the file was created to shorten the line(s) by entering carriage
   returns.

* You used /Range Search to replace a string of text with a longer string,
   but making that replacement in the highlighted cell would have created an
   entry longer than 511 characters. Use a shorter replacement string, specify
   a search range that does not include that cell, or select Next when l123
   highlights that cell.
