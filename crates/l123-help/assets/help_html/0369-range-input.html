/Range Input

/Range Input -- Limits cell-pointer movement and data entry to unprotected
cells in a range so you can edit data in those cells and no others. This
command works in conjunction with /Range Unprot.

1. (Optional) Set up a fill-in-the-blanks entry form.
2. Use /Range Unprot to unprotect the cells in which you want to enter or
    edit data when you use /Range Input. If you created an entry form,
    unprotect the blank cells.
3. Select /Range Input.
4. Specify the data-input range, including the cells you unprotected in
    step 2. If you set up an entry form, be sure to include the entire
    form, not just the blank cells, in the data-input range. l123 moves
    the data-input range to the upper left corner of the screen, with the
    cell pointer in the first unprotected cell in the range.
5. Enter or edit data in the unprotected cells. You can move the cell
    pointer only to the unprotected cells in the data-input range.
6. To end /Range Input, press ENTER or ESC when l123 displays the READY
    mode indicator.
