/Graph Type Features (continued)

Drop-Shadow    Used with any type of graph except an HLCO graph. Select
                Yes to add drop shadows to a graph, or select No (default)
                to remove drop shadows from a graph.

3D             Used with any type of graph except an HLCO graph. Select
                Yes to display pie slices, lines, or bars in 3-D in a graph,
                or select No (default) to restore 2-D display of graphs.

Table          Used with line, bar, stacked bar, or mixed graphs. Select
                Yes to display a table of the graph values below the graph, or
                select No (default) to remove a table from a graph.

Quit           Returns you to the /Graph Type menu.
