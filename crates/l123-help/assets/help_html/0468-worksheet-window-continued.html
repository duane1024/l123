NOTE  l123 keeps track of each file's window settings, which may differ
from the current window settings. When you change window settings with
/Worksheet Window, l123 uses the new settings for all active worksheets
and files. When you read a file into memory, l123 uses that file's
window settings for all active worksheets and files.
