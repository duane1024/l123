Naming a Macro

Naming a Macro -- You name a macro by assigning a range name to its first
cell with /Range Name Create or /Range Name Labels. You must name the
macro before you can run it. How you run the macro depends on the name you
assign.

    If you name a macro with a \ (backslash) and a single letter,
   such as \N, you can run the macro by pressing ALT and the letter.
   Using ALT is the simplest way to run a macro.
    If you name a macro with a series of up to 15 characters, such
   as PRINT_WIDE, you must use ALT-F3 (RUN) to run the macro.
    If you name a macro \0 (backslash zero), 1-2-3 runs the macro
   automatically when it reads the file with the macro into memory. To run
   the macro manually, you must also assign another range name to that macro.

NOTE  If you start all macro range names with the same character, such
as \, you can easily distinguish macro range names from other range names
in a range-name list.
