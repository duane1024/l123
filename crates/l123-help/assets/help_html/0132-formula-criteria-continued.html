Formula Criteria (continued)

    If you use more than one named database table for the input range
   and the tables contain duplicate field names, you must enter the table
   name and the field name separated by a period in a formula. For example,
   +SALES93.QTR1=2200 matches records with the value 2200 in the field named
   QTR1 in the database table named SALES93.
