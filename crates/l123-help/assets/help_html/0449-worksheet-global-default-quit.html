/Worksheet Global Default Quit

/Worksheet Global Default Quit -- Returns l123 to READY mode.

Before you select /Worksheet Global Default Quit, select Update if you
changed any global settings and want to save them.

/Worksheet Global Default Update saves the l123 configuration settings
in the l123 configuration file (123R34.CNF). l123 uses these settings
automatically when you start the program.

1. Select /Worksheet Global Default Quit.
