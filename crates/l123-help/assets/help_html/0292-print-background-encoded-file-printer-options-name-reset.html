/Print [Background, Encoded, File, Printer] Options Name Reset

/Print [Background, Encoded, File, Printer] Options Name Reset -- Deletes all
print settings names in the current file.

CAUTION  When you select /Print [B,E,F,P] Options Name Reset, l123
immediately deletes all print settings names in the current file.
There is no confirmation step.

1. If you have more than one file active, be sure the cell pointer is
    in the file whose print settings names you want to reset.
2. Select /Print [B,E,F,P] Options Name Reset.
