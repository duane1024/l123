/Range Format Other and /Worksheet Global Format Other

Automatic         Formats the cell according to the appearance of the entry,
                   in a numeric, date, time, or label format. For example,
                   if you enter 10-Apr-93, l123 stores the entry as the
                   date number 32608 and formats the cell as (DD-MMM-YY).
Color Negative    With a color monitor, displays negative numbers in a
                   different color. With a monochrome monitor, uses a
                   brighter intensity for negative numbers.
Color Reset       Turns off the Color Negative formatting.
Label             Stores new entries as labels by automatically adding a
                   label prefix (except for entries you create using Data
                   commands). Displays existing numbers in General format.
Parentheses Yes   Encloses numeric values in parentheses.
Parentheses No    Removes the parentheses added with Parentheses Yes.
