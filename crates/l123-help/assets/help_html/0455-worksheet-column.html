/Worksheet Column

/Worksheet Column -- Sets or resets the width of one or more columns, and
hides and redisplays columns. If a file is in GROUP mode, this command
affects all worksheets in the file.

1. Select /Worksheet Column and select one of the following:

    Set-Width      Sets the width of the current column.
    Reset-Width    Resets the width of the current column to the global
                   column width.
    Hide           Hides one or more columns to prevent display and
                   printing of those columns.
    Display        Redisplays hidden columns.
    Column-Range   Sets and resets the width of a range of columns.

2. If you selected Set-Width, specify a width either by typing a number
    from 1 through 240 or by pressing    or   , and then press ENTER.
