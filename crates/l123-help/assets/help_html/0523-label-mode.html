LABEL Mode

LABEL Mode -- A label is a text entry that begins with a letter or a
label prefix. If you type a letter or a label prefix as the first entry
in a cell, 1-2-3 changes the mode indicator to LABEL. A label can be up
to 512 characters long. 1-2-3 has five label prefixes:

    '    Left-aligns the label in the cell.
    "    Right-aligns the label in the cell.
    ^    Centers the label in the cell.
    \    Repeats the characters in the label to fill the cell.
    |    1-2-3 displays the label on the screen but will not print it.
         Also used to enter a printer control code in a worksheet.
         See Entering Setup Strings in a Print Range.

When you type any letter, 1-2-3 automatically adds the default label
prefix ('). Use /Worksheet Global Label to change the default label prefix
for an entire worksheet. Use /Range Label to change the label prefix for
a range.
