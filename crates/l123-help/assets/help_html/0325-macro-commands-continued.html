Macro Commands (continued)

    {FORM input-location,[call-table],[include-list],[exclude-list]}
        Suspends macro execution so you can enter and edit data in
        the unprotected cells in input-location. Macro execution
        resumes when you press ENTER or ESC while the mode indicator
        displays READY.

           Input-location is a range that includes unprotected cells.
           Call-table is a two-column range. The first column contains
          a list of macro key names; the second contains subroutines that
          1-2-3 executes when you press the keys listed in the first column.
           Include-list and exclude-list are ranges that contain lists
          of allowable keystrokes and keystrokes to ignore, respectively.
          (Use one or the other of these arguments; if you use both, 1-2-3
          ignores the exclude-list argument.)

    For more information on using {FORM}, see Chapter 13 of Reference.
