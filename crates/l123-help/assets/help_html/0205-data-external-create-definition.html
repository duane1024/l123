/Data External Create Definition

/Data External Create Definition -- Specifies a range (1-2-3 database table
or external table) that 1-2-3 can use as a model for a table definition
and enters the table definition in a location you specify; or specifies a
range that contains a table definition. A table definition is a description
of an external table. See Table Definition for more information.

Create-Definition   Creates a table definition using field names, data
                     types, and field widths derived from a 1-2-3 database
                     table or external table you specify.

Use-Definition      Specifies a range that contains a table definition
                     for a new table.

See /Data External Create Procedure for more information on using
/Data External Create Definition to create a table definition for a new
external table.
