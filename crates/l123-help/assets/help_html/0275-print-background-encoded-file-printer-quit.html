/Print [Background, Encoded, File, Printer] Quit

/Print [Background, Encoded, File, Printer] Quit -- Closes the current print
job and returns l123 to READY mode.

After you set up a print job and select Go, select Quit.

    If you are printing to a file, select Quit so l123 will close
   the file on disk.
    If you are using a print spooler, select Quit so the spooler
   will send data to the printer.
    If your printer is attached to a network, select Quit so the
   printer will begin printing the print job.
