/Print [Background, Encoded, File, Printer] Options Advanced

/Print [Background, Encoded, File, Printer] Options Advanced -- Performs the
following tasks:
Device       Selects the printer on which to print, the printer port,
             and the baud rate.
Layout       Selects layout options for the printed page.
Fonts        Selects fonts for your printed page.
Color        Selects a color for your printed text.
Image        Selects formatting options for printed graphs.
Priority     Determines the order for different print jobs. (This release of
             l123 does not use this setting, but it is retained for
             compatibility with earlier l123 releases.)
AutoLf       Tells l123 whether your printer inserts a line feed
             automatically after each printed line.
Wait         Suspends printing after each page.
Quit         Returns you to the /Print [B,E,F,P] Options menu.

NOTE  Not all printers are capable of using all of the advanced options.
For example, many printers cannot print in color or use different fonts.
