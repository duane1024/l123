Graph Types

Graph Types -- Descriptions

Line        Line graphs usually plot changes in data over time. For
             line graphs, 1-2-3 uses the X data range as the x-axis
             labels and the A - F data ranges as the lines in the graph.

Bar         Bar graphs compare related data at a given point in time
             or track changes in data over time. For bar graphs, 1-2-3
             uses the X data range as the x-axis labels and the A - F
             data ranges as the sets of bars in the graph.

XY          XY graphs, also called scatter charts, show correlations
             between two types of numeric data. 1-2-3 uses the A - F data
             ranges as points in the graph and the X data range as the
             x-axis scale. Only XY graphs use a numeric X data range.
