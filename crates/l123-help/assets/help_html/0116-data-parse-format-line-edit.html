/Data Parse Format-Line Edit

/Data Parse Format-Line Edit -- Lets you edit a format line to change the
way 1-2-3 parses the long labels below the format line.

To edit a data block (for example, to change values to labels), change
the corresponding letter in the format line. To adjust the width of a
data block (for example, to fit entries that may be longer than the entry
immediately below the format line), add or delete > or * characters. To
join data blocks, delete the intervening symbols.

Besides the standard editing keys, use the following:

   ,            Scrolls the labels below the format line by rows.
CTRL-BREAK   Cancels edits to the format line and returns 1-2-3 to READY mode.
ESC          Erases the format line, but does not delete the row.
HOME         Moves cursor to beginning of format line, and if you scrolled
              the labels, displays labels that were visible before scrolling.
PG UP/PG DN  Scrolls the labels below the format line by screen lengths.
