/Data Matrix Multiply

/Data Matrix Multiply -- Multiplies the columns of one matrix with the rows of
a second matrix and creates a third matrix that contains the results of the
multiplication. The number of columns in the first matrix must equal the
number of rows in the second matrix. The number of rows in the first
matrix multiplied by the number of columns in the second matrix must not
exceed 6553.

1. Select /Data Matrix Multiply.
2. Specify the first range (matrix) to multiply.
3. Specify the second range (matrix) to multiply.
4. Specify the output range where you want l123 to enter the results.
    Specify either the entire range or only the first cell.

    CAUTION  l123 writes over any existing data in the specified range, so
    make sure the output range does not contain data you need.

l123 creates an output range that contains the number of rows in the first
range and the number of columns in the second range.
