Part of combine file lost

You used /File Combine to combine a named range from a file on disk,
and the data in the named range extended beyond the worksheet boundaries.
When you pressed ENTER, l123 combined the data that fit in the available
rows and columns, but the data that did not fit in the worksheet
was lost.

When you use /File Combine, select an area of the worksheet that is large
enough to hold all the incoming data.
