/Data Table 3

/Data Table 3 -- This procedure for what-if analysis calculates the
results of one formula that uses three variables. Data Table 3 is similar
to Data Table 2 except you enter the formula outside the table range
and can use more than one worksheet for the table range.

Before you use /Data Table 3, you must set up the table range by entering
input values in the first column, row, and left corner cell of the table
range. You must also enter a formula to calculate the three variables.

To set up the table range

1. Decide on a location for the table range. The size of the table
    range depends on the number of input values you want to use for each
    of the three variables in the formula.
2. Decide on three cells outside of the table range for the three input
    cells and label them to make them easier to locate.
3. In a cell outside of the table range, enter the formula you want
    to analyze. The formula must refer to all three input cells.
