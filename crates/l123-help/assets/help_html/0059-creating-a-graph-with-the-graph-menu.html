Creating a Graph with the /Graph Menu

1. Select /Graph Type and select a graph type: Line (default), Bar, XY,
    Stack-Bar, Pie, HLCO, or Mixed.
2. Use either /Graph A - F and /Graph X or /Graph Group to set the graph
    data ranges.
3. (Optional) Use /Graph Options and /Graph Type Features to add titles,
    legends, and other enhancements to the graph.
4. Select /Graph View or press F10 (GRAPH) to display a full-screen view of
    the graph, or use /Worksheet Window Graph to display the graph in a
    window to the right of the worksheet.
5. If you displayed a full-screen view of the graph, press any key to
    redisplay the worksheet.
6. To save the graph, name it with /Graph Name Create, and then use
    /File Save to save it with the current file.
