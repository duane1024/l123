/Worksheet Column (continued)

    If you selected Hide, specify the range of columns to hide.

    If you selected Display, 1-2-3 temporarily displays all hidden columns
    with an * (asterisk) next to their column letters. Specify the range of
    columns to redisplay.

    If you selected Column-Range, select Set-Width or Reset-Width.

      If you select Set-Width, specify the range of columns whose widths
      you want to set. Then specify the new width either by typing a
      number from 1 through 240 or by pressing    or   , and then press ENTER.

      If you select Reset-Width, specify the range of columns whose widths
      you want to reset.
