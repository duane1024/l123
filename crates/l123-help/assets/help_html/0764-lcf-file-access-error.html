LCF file access error

The following condition could have caused this error:

* l123 could not use the LCF because you were already using the
   maximum number of files that DOS allows. To solve this problem,
   do one of the following: Use /Worksheet Delete File to delete one
   or more of your active l123 files; use /File Admin Reservation Release to
   release the reservation of one or more files; or end another application
   program, if one is running. Then try to print again. If you are using DOS,
   you may need to increase the number of open files the FILES statement in
   your CONFIG.SYS file allows. For information on changing your
   CONFIG.SYS file, see Chapter 4 of Getting Started.
