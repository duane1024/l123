/Print [Background, Encoded, Printer] Image

/Print [Background, Encoded, Printer] Image -- Prints the current graph or a
named graph on a printer or to an encoded file. If your current printer cannot
print graphs, 1-2-3 leaves a blank space on the page or in the file.
1. Select /Print Printer Image to print a graph on a printer, /Print
    Encoded Image to save a graph in an encoded file, or /Print Background
    Image to print a graph in the background.
2. Select Current or Named-Graph.
3. (Optional) Select /Print [B,E,P] Options Advanced Image to specify
    how you want your graph to appear on the page, and then select Quit
    three times to return to the /Print [B,E,P] menu.
4. If you are printing to a printer, make sure that the printer is online
    and the paper is at the top of a page.
5. Select Align.
6. Select Go to print the graph.
7. (Optional) Select Page to advance the paper to the top of the next page
    and to print the footer (if you specified one) on the last page of the
    print job.
8. Select Quit to complete the print job and return 1-2-3 to READY mode.
