Format Line

Format Line -- A format line marks the places at which 1-2-3 will divide
the long label below the format line into individual entries, called data
blocks. A format line is preceded by a | (vertical bar) and can contain
these symbols:

      L   Represents the first character of a label block.
      V   Represents the first character of a value block.
      D   Represents the first character of a date block.
      T   Represents the first character of a time block.
      S   Skips the data block below the symbol when parsing data.
      >   Represents any character in a data block.
      *   Represents a blank space between data blocks.

When you select /Data Parse Go, 1-2-3 copies the contents of each data block
to a separate column. For example, the following format line results in three
adjacent columns of data: one of labels, one of values, and one of dates.

                   |L>>>>>**V>>>*D>>>>>>>
