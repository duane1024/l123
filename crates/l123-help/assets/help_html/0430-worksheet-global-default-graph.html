/Worksheet Global Default Graph

/Worksheet Global Default Graph -- Sets automatic graph ranges (by columns or
by rows) and sets the file format for graph files.

1. Select /Worksheet Global Default Graph and select one of the following:
    Columnwise  (Default) Tells 1-2-3 to graph data in an automatic
                graph range by columns.
    Rowwise     Tells 1-2-3 to graph data in an automatic graph range by rows.
    Metafile    (Default) Tells 1-2-3 to save graphs in graphic
                metafile (.CGM) format when you use /Graph Save and to list .CGM
                files when you select /File List Graph and /File Erase Graph.
    PIC         Tells 1-2-3 to save graphs in picture file (.PIC) format when
                you use /Graph Save and to list .PIC files when you select
                /File List Graph and /File Erase Graph. You can print graph
                files in picture file format using the 1-2-3 Release 2 and 1A
                PrintGraph program.
2. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the
    new default Graph settings in future sessions, select /Worksheet
    Global Default Update.
