/Graph Options Scale [Y, X, 2Y] Automatic, Manual, Lower, or Upper

/Graph Options Scale [Y, X, 2Y] Automatic, Manual, Lower, or Upper --
Determines whether l123 chooses the scale's upper and lower limits
automatically or uses the limits you specify.

1. Select /Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Automatic
    or Manual.
    Automatic  (Default) l123 chooses upper and lower limits for the
               selected scale based on the values you are graphing.
    Manual     l123 uses the Upper and Lower settings as upper and lower
               limits for the selected scale, and displays only the graph
               values that fall between those settings.
2. If you selected Manual, select Upper and Lower from the /Graph Options
    Scale [Y-Scale, X-Scale, 2Y-Scale] menu and specify the upper and lower
    limits for the selected scale.
