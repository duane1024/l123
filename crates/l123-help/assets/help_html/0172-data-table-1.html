/Data Table 1

/Data Table 1 -- This procedure for what-if analysis calculates the
results of one or more formulas with one variable. Before you can use
/Data Table 1, you must set up the table range by entering one or more
formulas in the first row and input values in the first column of the table.

1. Decide on a location for the table range.
2. Decide on a location outside of the table range for the input cell and
    label it to make it easier to locate.
3. In the first row of the table range, beginning in the second cell,
    enter one or more formulas you want to analyze. The formulas must
    refer to the input cell as the variable.
4. In the first column of the table range, beginning in the second cell,
    enter the input values you want 1-2-3 to use in the formulas.
5. Select Data Table 1.
6. Specify the table range.
7. Specify the input cell.
1-2-3 calculates each formula, using all input values, and enters the
results in the table range.
