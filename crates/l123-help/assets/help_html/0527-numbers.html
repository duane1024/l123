Numbers

l123 accepts as an entry any positive or negative number from (1E-99) to
(9.99E+99) and the number 0. You can type a number with or without thousands
separators, for example, 123000 or 123,000. You can also type a number
as a percentage, in scientific notation, as currency, or as a fraction.
For example, to enter the number 1.75, you can type 175%, 17.5E-1,
$1.75, or 1 3/4.

You can control the way l123 displays numbers by setting the cell
format with /Worksheet Global Format or /Range Format.
