/Data Query Del

/Data Query Del -- Deletes records in the input range that match the
criteria in the criteria range and shrinks the input range to remove the
blank rows. You can use only one input range with this command.

1. Before you can use /Data Query Del, you must create an input range
    and a criteria range.
2. Select /Data Query Del.
3. Select Cancel to return to the /Data Query menu without deleting
    any records, or select Delete to delete all the records in the
    input range that match the criteria in the criteria range.
