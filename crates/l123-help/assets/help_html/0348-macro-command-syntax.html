Macro Command Syntax

Macro Command Syntax -- {KEYWORD arg1,arg2,...,argn}

A macro command consists of a keyword and, in most cases, one or
more arguments, enclosed in { } (braces). The keyword tells l123 what
action to perform. The arguments tell l123 how to complete the action.

When you enter arguments in a macro command, separate the
keyword from the first argument with a space and the arguments from one
another with argument separators (commas, semicolons, or periods, depending
on the /Worksheet Global Default Other International Punctuation setting).

Macro commands use four types of arguments: number, string,
location, and condition.

Number        Use a number, numeric formula, or reference to
               a cell that contains a number or numeric formula.
