/Worksheet Global Default Printer [Left, Right, Top, Bottom]

/Worksheet Global Default Printer [Left, Right, Top, Bottom] -- Sets margins
on the printed page measured in standard lines and standard characters.

1. Select /Worksheet Global Default Printer.
2. Select a margin and enter a margin size.
    Left      Sets the left margin in standard characters from 0 through 1000.
              (Default is 4 characters.)
    Right     Sets the right margin in standard characters from the left edge
              of the page, from 0 through 1000. (Default is 76 characters.)
    Top       Sets the top margin in standard lines from 0 through 240.
              (Default is 2 lines.)
    Bottom    Sets the bottom margin in standard lines from 0 through 240.
              (Default is 2 lines.)
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the new
    margin settings in future sessions, select /Worksheet Global Default
    Update.

The corresponding /Print command is /Print [B,E,F,P] Options Margins [L,R,T,B].
