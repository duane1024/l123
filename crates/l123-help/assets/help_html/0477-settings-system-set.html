Settings System Set

After you select Set, specify an add-in file to read into memory
automatically each time you start l123. l123 displays the name of the
add-in file on the settings sheet. If the add-in file contains an
application, l123 asks you if you want it to start the application
automatically each time you start l123. You can start only one
application automatically when you start l123. If you select Yes for a
second application, l123 automatically changes the auto-invoke setting for
the first application to No.

If the add-in file contains an application you want assigned to the same key
each time you start l123, select 1 to assign the application to
ALT-F7 (APP1), 2 to assign it to ALT-F8 (APP2), or 3 to assign it to
ALT-F9 (APP3). Select No-Key if you do not want to assign the application
to a key. l123 only lets you select a key that has no application
assigned to it already. Select Update to save new settings in the add-in
configuration file so that l123 can use them each time you start l123.
