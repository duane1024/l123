ALT-F2 (RECORD) Trace

ALT-F2 (RECORD) Trace -- The macro trace feature helps you debug a macro by
showing each macro instruction in the status line as you run the macro.
If the macro fails, the macro trace feature shows you which step failed.

NOTE  The macro trace feature runs macros at normal speed. If you want to run
a macro one step at a time select ALT-F2 (RECORD) Step.

1. With 1-2-3 in READY mode, press ALT-F2 (RECORD) and select Trace to turn on
    the macro trace feature.

2. Start the macro.

    As 1-2-3 performs each macro instruction, the cell address of the macro
    instruction and the contents of the cell appear in the status line.

3. To turn off the macro trace feature, press ALT-F2 (RECORD) in READY mode
    and select Trace.
