    @INT(x)

    @IRR(guess,range)
        Calculates the internal rate of return expected from a
        series of cash flows generated by an investment. The
        internal rate of return is the percentage rate at which
        the present value of an expected series of cash flows is
        equal to the present value of the initial investment.

    @ISERR(x)
        Returns 1 (true) if x is the value ERR; returns 0 (false) if
        x is not the value ERR.
