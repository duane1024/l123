Entering a Macro (continued) -- How-To Tips

    Specify consecutive uses of a keystroke instruction by including
   a number in the instruction. For example, {UP 4} is the same as
   {UP}{UP}{UP}{UP}.

    Precede a cell or range reference in a macro with a file reference
   (for example, <<SUMMARY.WK3>>A:A1..A:B5) when the cell or range is not
   in the same file as the macro.

    To include a macro command as a macro instruction, type
   { (open brace) and press F3 (NAME). Highlight the name of the command
   you want to use and press ENTER.
