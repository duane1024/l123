Printer error - attention required (continued)

The following conditions could have caused this error:

* l123 could not print to a text or encoded file or save your graph. The
   disk to which you were printing could be damaged, could have no space left
   for your file, or could contain the maximum number of files in the root
   directory. If you are printing to a disk in a diskette drive, be sure the
   drive contains a formatted disk, the disk is not write-protected, and the
   drive door is closed.

* l123 could not read into memory a file it needed because the file was
   missing or damaged. Use the Install program to reselect your equipment.
   For information on starting Install, see "Starting Install After
   First-Time Installation" in Chapter 5 of Getting Started. If the driver
   for your printer is not part of the Install program but was supplied by a
   third party, see the installation directions that came with the driver.
