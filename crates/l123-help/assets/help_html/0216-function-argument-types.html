@Function Argument Types

@Functions use four types of information as arguments: values, strings,
locations, and conditions.

Value arguments: you can use a number, a numeric formula, or the range
name or address of a cell that contains a number or numeric formula.

String arguments: you can use a literal string, a string formula, or
the range name or address of a cell that contains a label or string formula.

Location arguments: you can use a range name, address, or any formula
that evaluates to a range name or address.

Condition arguments: you typically use a logical formula or the range
name or address of a cell that contains a logical formula. However, you can
also use any numeric or string formula, number, literal string, or cell
reference as a condition argument.
