/Data Parse

/Data Parse -- Converts an imported text file, imported into l123 as one
column of long labels, into separate columns of data of one or more
types (values, dates, times, and labels).

Format-Line     Creates or edits a format line that controls the way l123
                divides one or more long labels into separate entries.
Input-Column    Specifies the single-column range that contains the
                format lines and labels you want to parse.
Output-Range    Specifies the range where you want l123 to place the
                parsed data.
Reset           Clears the settings for the input and output ranges.
Go              Parses the labels in the input column, enters the results
                in the output range, and returns l123 to READY mode.
Quit            Returns l123 to READY mode without parsing the labels
                in the input column.
