/Print [Background, Encoded, File, Printer] Range

/Print [Background, Encoded, File, Printer] Range -- Specifies the print range,
which is the data l123 prints when you select /Print [B,E,F,P] Go. The print
range can include any number of ranges and named graphs in any active file.

1. Select /Print [B,E,F,P] Range.
2. Specify the range or ranges you want to print.
       To set a single print range, specify the range.
       To set multiple print ranges, specify the ranges with
      ; (semicolon) between each one. For example, to print ranges
      A:A1..C:B5 and D:D1..D:F2 in the current file and a range named
      SUM in a file named Q1.WK3, enter a:a1..c:b5;d:d1..d:f2;<<q1.wk3>>sum.
       To specify a named graph as part of a print range, type an *
      (asterisk) followed by the name of the graph. For example, to specify
      a graph named FIRST_QTR, enter *first_qtr.
       To specify a range and a graph, specify them with
      ; (semicolon). For example, to print the data in the range A:A1..C:B5
      and the graph named FIRST_QTR, enter a:a1..c:b5;*first_qtr.
