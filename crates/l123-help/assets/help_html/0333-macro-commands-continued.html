Macro Commands (continued)
    {ONERROR branch-location,[message-location]}
        Traps errors that occur while a macro is running. If an error occurs
        during a macro after an {ONERROR} command, l123 branches to the
        macro instructions in branch-location (optionally recording the error
        message in message-location) instead of terminating the macro.

    {OPEN file-name,access-type}
        Opens a new text file with write (w) access, or opens an existing
        text file with either read (r), modify (m), or append (a) access.

    {PANELOFF [clear]}
        Freezes the control panel and status line until l123 encounters a
        {PANELON} command or the macro ends. The optional clear argument
        clears the control panel and status line before freezing them.

    {PANELON}
        Unfreezes the control panel and status line after {PANELOFF}.
