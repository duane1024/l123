/Data Table Labeled Input-Cells

/Data Table Labeled Input-Cells -- Specifies the input cells for row,
column, and worksheet variables. An input cell is a cell in which l123
temporarily stores values while it performs the calculations for the
table.

1. Select /Data Table Labeled Input-Cells. You can use Input-Cells if
    you need to review or edit any of the input cells you selected with
    /Data Table Labeled [Across, Down, Sheets].

    l123 cycles through each set of input values and input cells.
    Press ENTER to accept the entries or specify new addresses.
    When you finish viewing the last input cell you specified,
    press ENTER to return to the /Data Table Labeled menu.

    NOTE  You can only change input cells here. To change or add sets of
    input values, you must re-enter Down, Across, or Sheets.
