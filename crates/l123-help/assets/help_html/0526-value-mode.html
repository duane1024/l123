VALUE Mode

VALUE Mode -- A value is any entry that begins with a number (0 through 9)
or one of the symbols + - @ . ( or #. If you type one of these characters
as the first entry in a cell, l123 changes the mode indicator to VALUE.
A value can be a number or a formula.

You can use the following keys when l123 is in VALUE mode:

BACKSPACE     Erases the character to the left of the cursor.
ENTER         Enters the value in the current cell.
ESC           Cancels the entry and returns l123 to READY mode.
F2 (EDIT)     Switches between VALUE and EDIT modes.
F4 (ABS)      Switches a reference between relative, mixed, and
               absolute.
F9 (CALC)     Converts a formula to its current value.
