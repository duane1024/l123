/Worksheet Delete

/Worksheet Delete -- Deletes one or more columns, rows, worksheets, or
active files, closing up the space left by the deletion.

CAUTION  /Worksheet Delete deletes entire columns, rows, or worksheets
from memory, not just the part you see on the screen, so make sure you
do not delete data you need. If a file is in GROUP mode, deleting rows
or columns in one worksheet deletes the same rows or columns in all
worksheets in that file.

1. Select /Worksheet Delete and select one of the following:
    Column     Deletes one or more columns.
    Row        Deletes one or more rows.
    Sheet      Deletes one or more worksheets.
    File       Deletes an entire active file.
2. If you selected Column, Row, or Sheet, specify the range to delete.
    The range must include at least one cell from each of the columns,
    rows, or worksheets to be deleted.
    If you selected File, specify the active file to delete.
