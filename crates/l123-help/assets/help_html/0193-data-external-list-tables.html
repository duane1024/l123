/Data External List Tables

/Data External List Tables -- Lists the names of the tables in an external
database. Use this command if you do not remember which external database
contains tables you want to use.

1. Select /Data External List Tables.
2. Specify the name of the DataLens driver for the database
    containing the tables you want to list.
3. Specify the name of the external database containing the tables
    you want to list.
4. Specify the first cell of the range or the name of the range in
    which you want the list to appear.

    CAUTION  1-2-3 writes over existing data in the location you specify
    for the list of table names. Make sure you specify a location that does
    not contain data you need.
