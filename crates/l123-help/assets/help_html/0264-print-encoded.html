/Print Encoded

/Print Encoded -- Creates an encoded file on disk. An encoded file can
include data, graphs, and printer codes that tell a printer how to
format printed work. Printer codes are specific to your current printer.

1. Make sure the printer on which you will print the file is current.
    Use /Print [B,E,P] Options Advanced Device Name to make a printer current,
    and then select Quit twice to return to the /Print [B,E,F,P] menu.
2. Select /Print Encoded.
3. Specify a file name.
4. Select Align.
5. Select Range and specify a range to print. See /Print [B,E,F,P] Range
    for more information on specifying a range to print.
6. (Optional) Select Options (/Print Printer Options) to set different
    print options.
7. Select Go to print data to an encoded file.
8. If you want to include more ranges in the file, repeat steps 5 and 7.
9. Select Quit to complete the encoded file on disk and return 1-2-3 to
    READY mode.
