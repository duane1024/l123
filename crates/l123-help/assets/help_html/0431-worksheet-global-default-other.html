/Worksheet Global Default Other

/Worksheet Global Default Other -- Performs the following:

International  Sets the following:
                  The characters 1-2-3 uses as the decimal point, the
                 thousands separator, and the argument separator for
                 @functions and macro commands
                  The currency symbol for numbers formatted as Currency
                  The international Date and Time formats
                  The way 1-2-3 displays negative numbers formatted as Comma
                 or Currency
                  The way 1-2-3 reads and saves characters in .WK1 files
                  The way 1-2-3 translates characters in text files
Help           The Help access setting. (This release of 1-2-3 does not
               use this setting, but it is retained for compatibility with
               earlier 1-2-3 releases.)
Clock          Sets the display for the file-and-clock indicator.
Undo           Turns the undo feature on and off.
Beep           Turns the computer's bell on and off.
