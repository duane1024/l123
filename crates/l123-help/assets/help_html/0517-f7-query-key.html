F7 (QUERY) Key

F7 (QUERY) Key -- Repeats the last Data Query command you selected if you
selected one of the following:

/Data Query Del       Deletes the records in the input range that match
                      the criteria you entered in the criteria range.

/Data Query Extract   Copies to the output range records from the input
                      range that match the criteria you entered in the
                      criteria range.

/Data Query Find      Locates records in the input range that match the
                      criteria you entered in the criteria range. During a
                      Data Query Find command, pressing F7 (QUERY)
                      switches 1-2-3 between FIND mode and READY mode.

/Data Query Unique    Similar to /Data Query Extract except /Data
                      Query Unique eliminates any duplicate records.
