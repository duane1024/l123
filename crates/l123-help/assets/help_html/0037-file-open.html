/File Open

/File Open -- Reads a worksheet file into memory and moves the cell pointer
to the cell it was in when you last saved the file. Use /File Open to work
with more than one active file.

1. Select /File Open.
2. Select Before or After.
    Before reads a file into memory in front of the current file.
    After reads a file into memory behind the current file.
3. Specify the name of the file you want to open.
4. If the file on disk is password-protected, type the password and
    press ENTER.
5. If someone else is using the file and has the file reservation,
    1-2-3 displays a Yes/No menu. Select Yes to open the file without the
    reservation, or select No if you do not want to open the file without
    the reservation.

    CAUTION  If you do not have the reservation, 1-2-3 will not let you
    save changes to the file using the same file name.
