/File List

/File List -- Displays a list of files that temporarily overlays the
current worksheet. When you highlight a file name in the list, l123
displays specific information about that file.

1. Select /File List.
2. Select Worksheet, Print, Graph, Other, Active, or Linked.
3. Highlight a file name to display information about the file.
       If you selected Worksheet, Print, Graph, or Other, l123 displays the
      file name, the date and time the file was last saved, and the file size.
       If you selected Active, l123 also displays the number of worksheets
      in the file, MOD or UNMOD to indicate if you have modified the file
      since you read it into memory, and RO for read-only if you do not have
      the file reservation.
       If you selected Linked, l123 displays the paths and names of files
      linked to the current file.
4. Press ENTER to return l123 to READY mode and redisplay the current
    worksheet.
