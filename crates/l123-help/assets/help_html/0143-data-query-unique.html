/Data Query Unique

/Data Query Unique -- Performs the same operation as /Data Query Extract
(copies to the output range all records in the input range that match the
criteria in the criteria range), but it eliminates duplicate records and
sorts the records (in ascending order) in the output range by the first
field listed there.

/Data Query Unique uses the fields in the output range to determine
duplicates. If two records are not exactly alike but the output range
contains only fields that are the same for those two records, l123
identifies the records as duplicates and enters only one of them in
the output range. To avoid confusion about unique records, you can
enter all the field names in the output range.

1. Before you use /Data Query Unique, you must specify the input,
    criteria, and output ranges.
2. Select /Data Query Unique.
