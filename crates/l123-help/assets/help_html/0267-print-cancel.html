/Print Cancel

/Print Cancel -- Stops the current background print job and cancels any
other background print jobs waiting to be printed.

1. Select /Print Cancel.

/Print Cancel resets the page and line counter so 1-2-3 will begin the next
print job at the top of the page. Before you print again, realign
the paper in the printer.

Use /Print Suspend rather than /Print Cancel if you need to interrupt
background printing temporarily.
