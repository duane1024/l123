ALT-F1 (COMPOSE) Key

ALT-F1 (COMPOSE) Key -- Enters characters in l123 that you cannot enter
directly from the keyboard. l123 uses the Lotus Multibyte Character Set
(LMBCS) to display, store, and print characters.

You can enter a LMBCS character in a worksheet by using @CHAR and the
LMBCS code for the character or by using a compose sequence. A compose
sequence is a series of keystrokes you use to enter a character that
is not on your keyboard.

To enter a LMBCS character, press ALT-F1 (COMPOSE) and then enter the
appropriate keystrokes. For example, to enter the character
(British pound sterling symbol), press ALT-F1 (COMPOSE) and type L=.

For more information on using ALT-F1 (COMPOSE), see Appendix B of Reference.
Appendix B also contains a complete list of LMBCS codes, compose sequences,
and the characters you can enter in l123 using compose sequences.
