Long Labels and Values

Long Labels and Values -- l123 displays long labels and values differently.

Long Labels: A label can include up to 512 characters. When a label is
    longer than the cell's column width, l123 displays as much of the
    label as it can. If cells to the right of the label are blank, l123
    displays all of the label that overlaps those cells. If cells to the
    right of the label contain data, l123 does not display the overlapping
    part of the label. A long label never writes over cells that contain
    data.

Long Values: A long value is a value that, as displayed, is wider than
    the cell's column width minus 1. Unlike long labels, l123 does not
    let long values overlap into adjacent blank cells, nor does it display
    only part of the value when adjacent cells contain data. Depending on
    how the cell is formatted, l123 either displays the value in scientific
    notation, rounds the value's decimal part, or displays asterisks across
    the cell. To display the actual value, either change the cell format with
    /Range Format or widen the column with /Worksheet Column Set-Width.
