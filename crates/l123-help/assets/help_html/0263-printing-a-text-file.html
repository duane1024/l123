Printing a Text File

Printing a Text File -- You can use the DOS COPY command to print a text file
on most printers.

NOTE  Some printers cannot print text files because they require special
printing commands. For example, the Apple LaserWriter prints files that
are in the PostScript format only. Use /Print Encoded to create files
for printers that require special printing commands.

To use the DOS COPY command to print a text file, enter the following
command at the DOS prompt:

     copy [path]filename device

For example, enter copy c:\123r34\budget.prn lpt1 to print the file
BUDGET.PRN in the C:\123R34 directory on device LPT1.

For more information about the DOS COPY command, see your DOS documentation.
