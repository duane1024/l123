Running a Macro (continued)

Use ALT-F3 (RUN) to run a macro when the macro's name is anything other
than a backslash and a single letter, for example, PRINT_WIDE.

1. Make sure the file that contains the macro is active.
2. Press ALT-F3 (RUN). 1-2-3 displays a list of all range names in the
    current file and the names of other active files.
3. Specify the macro to run in one of the following ways:
       Type the macro range name or address and press ENTER. (If the
      macro is in another file, precede the range name or address
      with a file reference.)
       Highlight the macro range name and press ENTER.
       Press ESC to enter POINT mode, move the cell pointer to the first
      cell of the macro, and press ENTER.
