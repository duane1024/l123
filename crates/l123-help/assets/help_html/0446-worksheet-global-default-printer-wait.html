/Worksheet Global Default Printer Wait

/Worksheet Global Default Printer Wait -- Tells 1-2-3 whether to pause
after it prints each page. Use this command when you are printing on
a printer that requires single-sheet feed paper.

1. Select /Worksheet Global Default Printer Wait.
2. Select Yes to have 1-2-3 pause after it prints each page so you can
    insert a piece of paper. Select No (default) if you do not want
    1-2-3 to pause after it prints each page.
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses
    the new Wait setting in future sessions, select /Worksheet Global
    Default Update.

The corresponding /Print command is /Print Printer Options Advanced Wait.
