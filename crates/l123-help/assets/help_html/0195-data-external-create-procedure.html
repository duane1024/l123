/Data External Create Procedure

1-2-3 uses a table definition to create a new external table. A table
definition is a six-column range of data that describes the external
table. You can create a table definition as part of the procedure for
creating a new external table, or you can create a table definition
separately and then specify that table definition when you create a new
external table. The following procedure for creating a new external table
includes steps for creating a table definition. (For more information,
see Table Definition.)

1. Select /Data External Create Name to connect to the external database in
    which you want to create the new table and to assign a name to the table.

2. Select Definition Create-Definition to specify the external table or
    1-2-3 database table you want 1-2-3 to use as a model for the table
    definition, and to specify the range in which you want 1-2-3 to enter the
    table definition.
