EDIT Mode (continued)

Key                       Effect

DEL                       Erases the character above the cursor.
END                       Moves the cursor to the last character in the
                           entry.
ENTER                     Completes editing.
F2 (EDIT)                 Switches 1-2-3 between EDIT and READY, VALUE, or
                           LABEL mode.
ESC                       Erases all characters in the entry.
HOME                      Moves the cursor to the first character in the
                           entry.
INS                       Switches between insert mode and overstrike mode.
PG DN or PG UP            Completes editing and moves the cell
                           pointer down or up one screen.
