Mode Indicator

Mode Indicator -- l123 displays the mode indicator in the upper right
corner of the screen to tell you what mode l123 is in. The following
table lists the l123 modes:

Mode    Explanation

EDIT    You pressed F2 (EDIT) to edit an entry, selected /Data Parse
         Format-Line Edit, or made an incorrect entry.
ERROR   l123 is displaying an error message. Press F1 (HELP) to get Help,
         and press ESC or ENTER to clear the error message.
FILES   l123 is displaying a list of file names.
FIND    You selected /Data Query Find or pressed F7 (QUERY).
HELP    You pressed F1 (HELP) and l123 is displaying a Help screen.
LABEL   You are entering a label.
MENU    You pressed / (slash) and l123 is displaying a menu of commands.
NAMES   l123 is displaying a list of range, graph, print settings,
         @function, macro command, or external table names.
