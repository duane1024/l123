Cannot print ranges or named graphs not in memory

The following conditions could have caused this error:

* You tried to print a range or graph from a file that is not in
   memory or from an external database table. You can print ranges
   and graphs only from active files.

* You used a print range that included a file reference but did not
   include a path. You must include a path in a file reference, unless
   the file is stored in the l123 default directory.
