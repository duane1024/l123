Creating an Automatic Graph

Creating an Automatic Graph -- The automatic graphing feature lets you
create a graph by placing the cell pointer anywhere within the range
of data you want to graph and pressing F10 (GRAPH) or selecting /Graph View.

To create an automatic graph, the data you want to graph must be organized
in the following way:

     All the data for the graph (that is, the graph's A - F and X
    ranges) must be in a single-sheet range that is separated from other
    data in the worksheet by at least two blank rows and columns.

     The X data range must be in the first row or column of the range.
    For line, bar, pie, stacked bar, HLCO, and mixed graphs, the X data
    range must consist entirely of labels or of numbers formatted as Date or
    Time. For XY graphs, the X data range must consist of numbers.

     The data you want to use as the graph's A - F data ranges must be
    in the rows or columns immediately following the X data range.
