The Print Commands

The Print Commands -- Print files on a printer or to a file and perform
the following tasks:

Printer     Selects a printer as the destination for printing, then
            lets you select options for printing.
File        Selects a text file as the destination for printing, then
            lets you select options for printing.
Encoded     Selects an encoded file as the destination for printing, then
            lets you select options for printing.
Background  Selects background printing as the destination for printing, then
            lets you select options for printing.
Suspend     Temporarily stops background printing on a printer.
Resume      Continues printing a suspended l123 background print job and
            clears a background print error message.
Cancel      Cancels all l123 background printing.
Quit        Returns l123 to READY mode.
