/Print File

/Print File -- Creates a text file on disk.

1. Select /Print File.
2. Specify a file name.
3. Select Align.
4. Select Range and specify a range to print. See /Print [B,E,F,P] Range
    for more information on specifying a range to print.
5. (Optional) Select Options (/Print Printer Options) to set different
    print options.
6. (Optional) If you are creating a text file so you can use l123 data
    with another program, you may not want headers, footers, or page breaks
    in your file. Select Quit to return to the menu that begins with Range.
    Select Options Other Unformatted to eliminate headers, footers, and
    page breaks, and then select Quit.
7. Select Go to print data to a text file.
8. If you want to include more ranges in the file, repeat steps 4 and 7.
9. Select Quit to complete the text file on disk and return l123 to
    READY mode.
