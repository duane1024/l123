Macro Commands (continued)

    {GETPOS location}
        Enters a number in location. This number reports the current
        byte-pointer position (the position at which data is read
        from or written to) in the open text file. After executing a
        {GETPOS} command, 1-2-3 skips any further macro instructions
        in the same cell and goes directly to the next cell in the
        macro. If no text file is open, 1-2-3 ignores {GETPOS} and
        executes the instructions that follow it in the same cell.

    {GRAPHOFF}
        Removes a graph displayed by a {GRAPHON} command from the screen
        and redisplays the worksheet.
