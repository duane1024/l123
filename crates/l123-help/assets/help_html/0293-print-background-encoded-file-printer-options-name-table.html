/Print [Background, Encoded, File, Printer] Options Name Table

/Print [Background, Encoded, File, Printer] Options Name Table -- Creates an
alphabetical list of the print settings names in the current file.

1. Decide on a location in the current file for the print settings
    name table. The table will occupy one column and as many rows as
    there are print settings names.

    CAUTION  A print settings name table writes over existing data in the
    table location. Make sure the location does not contain data you need.

2. Select /Print [B,E,F,P] Options Name Table.
3. Specify the location you decided on in step 1. You need to specify only
    the first cell of the location.
