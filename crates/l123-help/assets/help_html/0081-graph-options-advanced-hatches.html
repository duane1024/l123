/Graph Options Advanced Hatches

/Graph Options Advanced Hatches -- Sets hatch patterns for the A - F data
ranges in displayed and printed graphs. Use /Graph Options Advanced Hatches
to change the default patterns 1-2-3 uses for black-and-white graph display
or to use hatch patterns in color graph display.

1. Select /Graph Options Advanced Hatches.
2. Select A, B, C, D, E, or F as the data range and select
    1 - 8     To set one hatch pattern for all values in the data range.
    Range     To set different hatch patterns for, or to hide, individual
              values in the data range. If you select Range, you should
              already have created a hatches range, as described in
              Creating a Hatches Range.
3. If you selected Range, specify the hatches range.
4. Repeat steps 2 and 3 to set hatch patterns for other data ranges.
