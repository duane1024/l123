Using BPrint

Before you can use the BPrint utility to print in the background on a local
printer, BPrint must be in memory. You start BPrint by entering bprint
at the DOS prompt. Then, start 1-2-3 and use /Print Background or
:Print Background to print a file in the background. If you print several
files in the background in a row, BPrint keeps track of the files in a
print queue and prints the files in the order that you specified them.

NOTE  You cannot use BPrint to print on a network printer.

After you start BPrint, you can issue BPrint commands at the DOS prompt.
BPrint commands provide printing options that are not available when you print
from 1-2-3 or Wysiwyg. You issue these commands by using /System to suspend
1-2-3 temporarily and display the DOS prompt, or by using /Quit to leave 1-2-3
and return to DOS.
