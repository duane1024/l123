/Worksheet Global Default Printer Name

/Worksheet Global Default Printer Name -- Selects the printer to use if
you specified more than one printer when you installed 1-2-3.

1. Select /Worksheet Global Default Printer Name.
2. Select a printer from the menu.
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the
    new Name setting in future sessions, select /Worksheet Global
    Default Update.

NOTE  If you change the Name setting, you may also need to change
the /Worksheet Global Default Printer Interface setting because the
new printer may be connected to your computer through a different
printer port.

The corresponding /Print command is /Print [B,E,F,P] Options Advanced Device
Name.
