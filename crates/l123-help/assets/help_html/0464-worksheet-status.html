/Worksheet Status

/Worksheet Status -- Displays information about available memory, hardware
options, recalculation settings for all active files, circular references,
cell display, and global protection for the current worksheet.

1. Select /Worksheet Status.
2. Press any key to remove the status screen and redisplay the worksheet.

The initial settings are
    Recalculation:      Method (Automatic), Order (Natural), Iterations (1)
    Cell Display:       Format (G), Label prefix ('), Column width (9),
                        Zero setting (No)
    Global protection:  Off

Use /Worksheet Global [Recalc, Col-Width, Format, Label, Zero, Prot]
commands to change these settings.
