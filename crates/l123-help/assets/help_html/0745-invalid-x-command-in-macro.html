Invalid /X command in macro

You were running a macro and one of the following problems occurred:

* The macro contained /X followed by a character other than the letter
   C, G, I, L, M, N, Q, or R. The sequence did not represent one of the
   /X commands (/XC, /XG, /XI, /XL, /XM, /XN, /XQ, and /XR).

* The macro used the letter X to specify a menu selection, but none of
   the items in that menu started with X.
