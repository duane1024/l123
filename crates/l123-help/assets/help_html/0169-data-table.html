/Data Table

/Data Table -- Creates what-if tables that show how the results of formulas
change when you change the numbers used in the formulas. The type of
what-if table you create depends on the number of variables you want to
use in formulas.

Data Table 1  Calculates the results of one or more formulas that use one
              variable.
Data Table 2  Calculates the results of one formula that uses two variables.
Data Table 3  Calculates the results of one formula that uses three variables.
Data Table Labeled
              Calculates the results of one or more formulas that use one
              or more variables.
Reset         Clears the ranges and input-cell settings for all tables.

Use the Data Table commands to do the following:
    Perform what-if or sensitivity analysis.
    Cross-tabulate information in a 1-2-3 database table to reveal
   relationships among data entered in different fields.
