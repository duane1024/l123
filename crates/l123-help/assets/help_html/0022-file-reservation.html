File Reservation

File Reservation -- Each worksheet file has a reservation to ensure
that only one person can save changes to the file even though several
people may be using the file at the same time.

    When you have the file reservation, you are the only user allowed to
   save changes to the file. Other users can retrieve or open the file at
   the same time you are using it, but 1-2-3 does not let them save the
   file as long as you have the reservation. To release the reservation
   when you no longer need it, use /File Admin Reservation Release.

    When you do not have the reservation, 1-2-3 displays the RO (read-only)
   indicator at the bottom of the screen. You can use the file, but you
   cannot save changes to the file. Use /File Admin Reservation Get to
   try to get the reservation. You can get the reservation if no one else
   has it and no one has saved the file since you read it into memory.

CAUTION  When 1-2-3 displays the RO indicator, you do not have the file
reservation and cannot save the file with its current name.
