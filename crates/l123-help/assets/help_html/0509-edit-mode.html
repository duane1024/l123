EDIT Mode

EDIT Mode -- When you press F2 (EDIT), 1-2-3 enters EDIT mode. Use the
following keys in EDIT mode:
Key                 Effect
    or                 Moves the cursor one character to the right or left.
    or                 Completes the entry and moves the cell pointer up or
                     down one cell if the entry is only one line in the control
                     panel. If the entry is on more than one line in the
                     control panel, moves the cursor up or down one line.
BACKSPACE           Erases the character to the left of the cursor.
CALC                Converts a formula to its current value.
CTRL-   or           Moves the cursor five characters to the left.
    SHIFT-TAB
CTRL-   or           Moves the cursor five characters to the right.
    TAB
CTRL-PG DN or       Completes editing and, if you are using multiple
    CTRL-PG UP       worksheets, moves the cell pointer back or forward
                     one worksheet.
