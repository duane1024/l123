/Graph [A, B, C, D, E, F] (continued)

    In stacked bar graphs, l123 graphs the A - F data ranges as portions
   of bars, with the A-range data at the bottom of the bars, the B-range
   data above the A-range data, and so on.
    In XY graphs, l123 graphs the A - F data ranges as points, creating
   one set of points for each data range.

1. Select /Graph A, B, C, D, E, or F.
2. Specify the range you want to use as the A, B, C, D, E, or F data range.
    The range can be in any file, active or on disk.

NOTE  When the A - F data ranges are in consecutive columns or rows
immediately following the X data range, you can use /Graph Group instead of
/Graph A - F and X to set all data ranges at once.
