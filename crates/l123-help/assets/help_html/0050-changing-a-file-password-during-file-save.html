Changing a File Password During /File Save

1. Make sure the cell pointer is in the file whose password you want
    to change.
2. Select /File Save.
3. If l123 displays [ALL MODIFIED FILES], press F2 (EDIT) to display the
    current file name and [PASSWORD PROTECTED].
4. Press BACKSPACE once to clear the [PASSWORD PROTECTED] prompt.
5. Press the space bar once.
6. Type p and press ENTER.
7. Type a password and press ENTER. A password can include up to 15
    characters. l123 is case-sensitive for passwords.
    CAUTION  Remember the password. You can read the file into memory again
    only if you enter the exact password.
8. Type the same password again and press ENTER.
9. Select Replace to save the file with the new password.
