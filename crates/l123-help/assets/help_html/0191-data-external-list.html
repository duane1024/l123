/Data External List

/Data External List -- Lists the names of the fields in an external
table or lists the names of tables in an external database.

Tables   Lists the names of tables in an external database.

Fields   Lists the names of the fields in an external table to which you
         are currently connected and lists the data type, width, column label
         (if any), and description (if any) of each field in the table.
         You can use the sixth column for field creation strings.

         Use /Data External List Fields if you want to

            Extract information from or copy information to an external
           table, but are not sure what types of fields the table contains.

            Create a new external table similar in structure to an existing
           table (by using the resulting field information as the table
           definition for a new table). See Table Definition.
