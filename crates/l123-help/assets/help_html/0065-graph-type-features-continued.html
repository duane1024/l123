/Graph Type Features (continued)

2Y-Ranges      Assigns one or more data ranges to the second y-axis.
                Select Graph to assign all data ranges to the second
                y-axis, A - F to assign individual data ranges to the
                second y-axis, or Quit to return to the /Graph Type
                Features menu.

Y-Ranges       (Default) Reassigns one or more data ranges to the first
                y-axis. Select Graph to reassign all data ranges to the
                first y-axis, A - F to reassign individual data ranges
                to the first y-axis, or Quit to return to the /Graph Type
                Features menu.

NOTE  The 2Y-Ranges and Y-Ranges settings do not affect HLCO graphs.

Frame          Adds or removes the frame around part or all of a graph.
