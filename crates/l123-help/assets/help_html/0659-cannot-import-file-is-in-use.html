Cannot import - file is in use

You used /File Import to copy data from a text file that was not
closed on disk.

If you were creating the text file, close the file by selecting /Print
File Quit before you try to import data from the file.

If someone else was saving changes to the file on disk, wait for
l123 to close the file and then repeat /File Import.
