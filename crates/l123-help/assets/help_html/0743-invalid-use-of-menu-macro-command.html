Invalid use of Menu macro command

You were running a macro that includes a {MENUBRANCH}, {MENUCALL},
or /XM command and the first cell of the range specified as the
location argument for that command was blank.

The {MENUBRANCH}, {MENUCALL}, or /XM location argument tells l123
where to find the macro menu for the command. The macro menu should
start in the upper left corner of location.

A macro menu usually consists of at least three rows: The first includes
the menu items (up to eight), the second includes descriptions for each
menu item, and the third includes macro instructions for each menu item.
For more information on creating macro menus, see the descriptions of
{MENUBRANCH} and {MENUCALL} in Chapter 13 of Reference.
