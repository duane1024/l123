More than one file has the same range name

In a command, you tried to specify a range by typing <<?>> (the wildcard
file reference) in front of a range name, but more than one active file
contained that range name.

You can use the wildcard file reference to specify a range only when
the range name that follows it is unique among all active files. When
more than one file contains the same range name, you must type the
appropriate file name in the file reference.
