Error loading add-in resource file

Error loading add-in resource file -- Not enough memory was available to read
the add-in resource file; free some memory and try the add-in operation
again.  Or, 1-2-3 could not use the add-in resource file because
you were already using the maximum number of files that your operating system
allowed.

Error running add-in macro -- 1-2-3 could not run an add-in macro command
to completion. If you are a developer, use the macro command procedure in
an add-in application to generate a more specific error message. Otherwise,
contact your add-in developer.

Invalid "file_name" -- Your add-in is damaged; the file is readable but
includes bytes that do not belong in a .PLC file, such as an illegal header.
If you are a developer, recompile the file. Otherwise, you must contact
your add-in developer.
