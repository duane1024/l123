/Data Table Labeled Sheets

/Data Table Labeled Sheets -- Specifies the worksheet variable range and
input cells. A worksheet variable range is a range that contains input
values organized in a three-dimensional range. It can contain more than
one set of input values.

1. Select /Data Table Labeled Sheets if you have a worksheet variable range.
2. Specify the worksheet variable range.
    l123 displays the address of the first group of cells in the worksheet
    variable range. Press ENTER to identify this set of input values as the
    set to which you will next assign an input cell.
3. Specify the input cell for that group of input values.
    If you entered a worksheet variable range that contains more than one
    one-cell-wide, three-dimensional group of cells, l123 repeats the
    prompts for assigning input cells for each additional group of input
    values.
