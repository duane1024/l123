/Range Name Note Table

/Range Name Note Table -- Creates an alphabetical list of range names and
notes in the current file. The list includes all defined range names
along with their range addresses and notes, if they have notes. The list
includes only those undefined range names that have notes.

1. Decide on a location in the current file for the range name note table.
    The table will occupy three columns and as many rows as there are
    range names plus one blank row.

    CAUTION  Make sure the table location does not contain data you need,
    because l123 writes over existing data when it creates the table.

2. Make sure the cell pointer is in the file whose range names and notes
    you want to list.
3. Select /Range Name Note Table.
4. Specify the location you decided on in step 1. You need to specify only
    the first cell of the table's location.
