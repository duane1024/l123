/Range Search

/Range Search -- Finds or replaces a specified string in labels and/or
formulas within a range. l123 will search hidden cells, but not hidden
columns or worksheets.

1. Select /Range Search.
2. Specify the range you want l123 to search.
3. Enter the string you want l123 to search for (the search string).
4. Select Formulas, Labels, or Both.
      Formulas   l123 searches for the string only in formulas.
      Labels     l123 searches for the string only in labels.
      Both       l123 searches for the string in formulas and labels.
5. Select Find or Replace. Find highlights occurrences of the string
    in the search range. Replace replaces occurrences of the string
    with a different string.
6. If you selected Find, l123 highlights the first occurrence of the
    search string and displays a Next/Quit menu. Select Next to
    highlight the next occurrence of the string. Select Quit to stop
    the search and return l123 to READY mode.
