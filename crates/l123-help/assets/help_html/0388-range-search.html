/Range Search

/Range Search -- Finds or replaces a specified string in labels and/or
formulas within a range. 1-2-3 will search hidden cells, but not hidden
columns or worksheets.

1. Select /Range Search.
2. Specify the range you want 1-2-3 to search.
3. Enter the string you want 1-2-3 to search for (the search string).
4. Select Formulas, Labels, or Both.
      Formulas   1-2-3 searches for the string only in formulas.
      Labels     1-2-3 searches for the string only in labels.
      Both       1-2-3 searches for the string in formulas and labels.
5. Select Find or Replace. Find highlights occurrences of the string
    in the search range. Replace replaces occurrences of the string
    with a different string.
6. If you selected Find, 1-2-3 highlights the first occurrence of the
    search string and displays a Next/Quit menu. Select Next to
    highlight the next occurrence of the string. Select Quit to stop
    the search and return 1-2-3 to READY mode.
