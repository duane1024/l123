/Data Table 1 Example -- This example uses two @PMT formulas to calculate
the monthly payments on two 30-year loans for $80,000 and $90,000 with
different interest rates (the input values in column B). The table range
is B2..D6 and input cell 1 is B1. 1-2-3 calculates the results by
substituting the input values in each formula and entering the results
in the table range. Find the monthly payment by locating the intersection
of a column and row. For example, the monthly payment on an $80,000 loan
at 10.5% interest is $731.79. The table range is formatted to display
values with 2 decimals.

    A:    A         B               C                           D

    1  Input 1:
    2                     @PMT(80000,B1/12,30*12)   @PMT(90000,B1/12,30*12)
    3              9.5%                    672.68                    756.77
    4             10.0%                    702.06                    789.81
    5             10.5%                    731.79                    823.27
    6             11.0%                    761.86                    857.09
