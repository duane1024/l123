/Print [B,E,F,P] Options Header and Footer (continued)

You can use the following five symbols in headers and footers:
    # (number sign) prints a page number on each page starting
   with the number 1.
    ## (double number sign) followed by a number specifies
   a particular number for the first printed page.
    @ (at sign) prints the current date on each page.
    | (vertical bar) separates the header or footer into
   left-aligned, centered, and right-aligned sections. For example,
   if you enter the footer @|BUDGET|Page #, l123 prints the date on
   the left, BUDGET in the center, and the page number on the right.
    \ (backslash) followed by a cell address or range name
   copies the contents of a cell to a header or footer.

Use /Print [B,E,F,P] Options Other Blank-Header to suppress the six blank
lines l123 reserves for headers and footers.
Use /Print [B,E,F,P] Page to print a footer on the last page of a print job.
