/Worksheet Global Default

/Worksheet Global Default -- Changes the settings that are stored in the
1-2-3 configuration file (123R34.CNF). The settings in this file take effect
automatically whenever you start 1-2-3.

Printer    Defines the type of printer you are using, the interface between
            the printer and 1-2-3, and the default printed-page settings.
Dir        Sets the 1-2-3 default directory.
Status     Displays the current /Worksheet Global Default settings.
Update     Saves the current /Worksheet Global Default settings in the
            1-2-3 configuration file (123R34.CNF) for use in future sessions.
Other      Changes the default Beep, Clock, International, and
            Undo settings.
Graph      Tells 1-2-3 to use either columns or rows as data ranges in
            automatic graphs and sets the file format for graph files.
Temp       Sets the directory in which 1-2-3 saves temporary files.
Ext        Sets the extension 1-2-3 uses with various File commands.
Autoexec   Tells 1-2-3 whether to run auto-execute macros in files.
Quit       Returns 1-2-3 to READY mode.
