/Range Name Note

/Range Name Note -- Creates, deletes, or lists range name notes. Use a range
name note to describe the contents of a named range. /Range Name Note
performs the following tasks:

Create     Creates a note for a defined range name or lets you edit
           an existing note.
Delete     Deletes a note for a defined range name.
Reset      Deletes all range name notes in the current file, including notes
           for undefined range names.
Table      Lists the defined range names in the current file, along with
           their addresses and (for those that have notes) range name notes.
           Also lists any undefined range names with notes.
Quit       Returns 1-2-3 to READY mode.
