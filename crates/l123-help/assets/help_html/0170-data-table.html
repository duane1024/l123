/Data Table -- What-If Analysis

The Data Table procedures and examples described in Help are what-if
tables. In these tables, you use one, two, or three variables in
formulas, and the table displays the results.

For example, a what-if analysis of the formula 5+X lets you substitute
values for X in the formula and see the results in a 1-way table. A
what-if analysis of the formula 5+X+Y lets you substitute different
values for X and Y in the formula and see the results in a 2-way table.
If you used the formula X+Y+Z, you would use a 3-way table to calculate
the results.

For more information, see /Data Table in Chapter 3 of Reference.
