/Data Sort Extra-Key

/Data Sort Extra-Key -- Specifies a field as an additional sort key after the
primary and secondary sort keys. You can use this command to specify up to
253 extra sort keys or edit the information for an extra sort key.

1. Select /Data Sort Extra-Key. l123 displays a number for the extra
    sort key. If you have not yet specified any extra sort keys, l123
    displays a 1 (for the first extra key). If you have specified one or
    more extra sort keys, l123 displays the number of the next available
    sort key.
2. Press ENTER to accept the displayed sort-key number or specify another
    number for the extra sort key by typing a number (or pressing    or
      ) and then pressing ENTER.
3. Specify a cell address in the field you want to use as the extra
    sort key.
4. Select A (ascending) or D (descending) to specify the sort order.
