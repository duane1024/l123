No output range specified

The following conditions could have caused this error:

* You tried to perform a data query operation without specifying an
   output range. Use /Data Query Output to specify an output range, and
   repeat the data query operation by pressing F7 (QUERY) or selecting
   a Data Query command.

* You tried to parse a column of long labels without specifying
   an output range. Select /Data Parse Output-Range, specify an output
   range, and then select Go.
