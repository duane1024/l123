/Print [Background, Encoded, File, Printer] Options Name Create (continued)

To change the settings assigned to a print settings name

1. Select /Print [B,E,F,P] Options Name Use.
2. Specify the name of the print settings you want to change.
3. Use the Print commands to change the settings.
4. Select /Print [B,E,F,P] Options Name Create.
5. Specify the same print settings name as in step 2.
