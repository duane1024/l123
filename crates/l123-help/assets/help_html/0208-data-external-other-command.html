/Data External Other Command

/Data External Other Command -- Sends a command to a database management
program. For information on the proper syntax for a command you send using
/Data External Other Command, see the documentation for the database
management program to which you are sending the command.

1. Make sure you are connected to an external table in the external
    database to which you want to send a command. If you are not, use
    /Data External Use to connect to a table in that database.
2. Select /Data External Other Command.
3. Specify the name of the DataLens driver for the external database to
    which you want to send a command.
4. Specify the name of the external database to which you want to send
    a command.
5. Enter the command as a string.
