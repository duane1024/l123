Deleting a File Password During /File Save

1. Make sure the cell pointer is in the file whose password you want
    to delete.
2. Select /File Save.
3. If l123 displays [ALL MODIFIED FILES], press F2 (EDIT) to display the
    current file name and [PASSWORD PROTECTED].
4. Press BACKSPACE once to clear the [PASSWORD PROTECTED] prompt and
    press ENTER.
5. Select Replace to save the file without a password.
