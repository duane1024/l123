Device driver not installed

The following conditions could have caused this error:

* You have not used the Install program to install a printer, or the printer
   driver in your 1-2-3 program directory is damaged. Use the Install program
   to install your printer. For information on starting Install, see "Starting
   Install After First-Time Installation" in Chapter 5 of Getting Started.

* The settings for printer name and printer port saved with the file you were
   trying to print did not match your current settings for printer name and
   port. Select /Print [B,E,F,P] Clear Device to clear the settings for printer
   name and port saved with the file and try to print again. 1-2-3 will print to
   your default printer unless you use /Worksheet Global Default Printer Name
   and Interface to change the default printer name and port or /Print [B,E,P]
   Options Advanced Device Name and Interface to change the current printer
   name and port. If you are printing with Wysiwyg, use :Print Config Printer
   and Interface to change your printer name and port and try to print again.
