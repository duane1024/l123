/Data Table Labeled

/Data Table Labeled -- Before you use /Data Table Labeled, you should know
how to use /Data Table 1, /Data Table 2, or /Data Table 3. For detailed
procedures and examples on these commands, see /Data Table in Chapter 3
of Reference.

Formulas       Specifies the formula range and the formula label range.
Down           Specifies the row variable range and input cells.
Across         Specifies the column variable range and input cells.
Sheets         Specifies the worksheet variable range and input cells.
Input-Cells    Specifies the input cells for column, row, and worksheet
               variables. You use Input-Cells to verify or edit the input
               cells you selected with Down, Across, and Sheets.
Label-Fill     Specifies a label-fill character for the formula label
               range and/or column variable range.
Go             Calculates the results of the labeled table.
Quit           Returns l123 to READY mode without calculating the results.
