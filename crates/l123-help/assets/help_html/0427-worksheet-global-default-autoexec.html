/Worksheet Global Default Autoexec

/Worksheet Global Default Autoexec -- Tells 1-2-3 whether to run
auto-execute macros when it reads files that contain the macros. You
can create one auto-execute macro in each file by assigning the range
name \0 (backslash zero) to the macro.

1. Select /Worksheet Global Default Autoexec.
2. Select Yes (default) to have 1-2-3 run auto-execute macros,
    or No if you do not want 1-2-3 to run auto-execute macros.
3. To update the 1-2-3 configuration file (123R34.CNF) so 1-2-3 uses the
    new Autoexec setting in future sessions, select /Worksheet Global
    Default Update.
