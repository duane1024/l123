Automatic Wysiwyg (continued)

If you cancel automatic start-up of Wysiwyg, restart 1-2-3, and then want to
use Wysiwyg, you can read Wysiwyg into memory for the current session by using
ALT-F10 (ADDIN). ALT-F10 (ADDIN) also lets you restore automatic start-up
of Wysiwyg.

To read Wysiwyg into memory for the current session, do the following:

1. Press ALT-F10 (ADDIN) and select Load.
2. Specify WYSIWYG.PLC as the add-in file you want to load.
3. Select No-Key if you plan to use : (colon) or a mouse to display
    the Wysiwyg main menu. Select 1, 2, or 3 to assign Wysiwyg to
    ALT-F7 (APP1), ALT-F8 (APP2), or ALT-F9 (APP3), respectively.
4. Select Quit to return 1-2-3 to READY mode.
