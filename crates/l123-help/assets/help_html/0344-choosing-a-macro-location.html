Choosing a Macro Location

Before you enter the macro in a worksheet, you must choose a location for
it. You can put the macro in a file that contains only macros (a macro
library file) or in a file that contains other data as well.

    If you plan to use the macro with a number of data files, put it in a
   macro library file.

    If you plan to use the macro with only one data file, put it in
   that file. If possible, enter it in a separate worksheet from the data.
   This keeps you from writing over data when you enter the macro and from
   damaging the macro when you insert or delete rows and columns in the data
   area.

   If you must enter the macro in a worksheet that also contains other
   data, put the macro below and to the right of, or above and to the left
   of, the data.
