/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Format

/Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Format -- Sets the format
for numbers on the selected scale.

1. Select /Graph Options Scale [Y-Scale, X-Scale, 2Y-Scale] Format.
2. Select Fixed, Scientific, Currency, Comma, General (default),
    +/-, Percent, Date, Date Time, Text, or Hidden.

    See the format menu for information on each type of format.
