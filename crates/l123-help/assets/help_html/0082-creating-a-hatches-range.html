Creating a Hatches Range

Creating a Hatches Range -- Before you use /Graph Options Advanced Hatches
[A - F] Range, you must set up a hatches range in the worksheet.

1. Decide on a range that is the same size as the corresponding data range.
2. In each cell in this range, enter a number from 1 through 14 or a negative
    number. The numbers 1 through 14 set hatch patterns for the corresponding
    graph data. Negative numbers hide the corresponding graph data.

If you graphed the values under Totals and used the values under Hatches Range
in the example below, 1500 would appear in hatch pattern 5, 250 and 950
would appear in hatch pattern 2.

         Totals      Hatches Range
           1500                  5
            250                 -5
            950                  2
