/Graph Options Advanced

/Graph Options Advanced -- Set the colors and hatch patterns of the A - F
data ranges and the colors, fonts, and sizes of graph text. /Graph Options
Advanced settings control the appearance of both displayed and printed
graphs.

The Graph Options Advanced commands perform the following tasks:

Colors      Sets colors for or hides the A - F data ranges.
Text        Sets colors, fonts, and sizes for graph text.
Hatches     Sets hatch patterns for the A - F data ranges.
Quit        Returns you to the /Graph Options menu.
