F9 (CALC) Key

F9 (CALC) Key -- Recalculates formulas in active files or converts a formula
to its current value. When a worksheet contains formulas that link to data
in files on disk, use /File Admin Link-Refresh to update those formulas.

    When l123 is in READY mode and /Worksheet Global Recalc
   is set to Manual, use F9 (CALC) to recalculate the formulas in active
   files. l123 displays a blue (or white reverse-video) CALC indicator
   whenever you change data to remind you to press F9 (CALC) to update
   formulas.

   When /Worksheet Global Recalc is set to Automatic, l123
   automatically recalculates formulas that are affected by changes to
   data. Automatic recalculation occurs in the background, so you can
   continue working while l123 performs the recalculation.

    When l123 is in EDIT or VALUE mode and the current cell contains
   a formula, use F9 (CALC) to convert the formula to its current value.
