The Output Range (continued)

       You can enter @AVG, @COUNT, @MIN, @MAX, @SUM to create a special type
      of computed column called an aggregate column in the output range.
      An aggregate column is a column in which l123 calculates a total
      for a group of related values. For example, if you enter Month
      and the formula @SUM(SALES) in the output range, l123 sums the sales
      amounts by month. For more information, see "The Output Range"
      in Chapter 3 of Reference.

       You can create an output range that contains records composed
      of fields from two or more database tables or the results of
      calculations based on fields from different database tables. For
      more information, see "The Output Range" in Chapter 3 of Reference.

       If the output range is an external table, you must connect to
      the external table by using /Data External Use.
