    When you use /File Combine Add or /File Combine Subtract, data in the

3. Select Entire-File or Named/Specified-Range.
    Entire-File combines all data from a file on disk.
    Named/Specified-Range combines a range of data from a file on disk.
4. If you selected Named/Specified-Range, specify a range in the file
    on disk by using either a range name or address.
5. Specify the file on disk.
6. If the file on disk is password-protected, type the password and
    press ENTER.
