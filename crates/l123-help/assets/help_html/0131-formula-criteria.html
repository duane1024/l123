Formula Criteria -- Enter formula criteria to search for labels or values.

    The formula must include the field name or the address of the first
   record in a field, a logical operator, and a value or label. For
   example, the formula +B2="May" matches records for May when column B
   contains months.
    You can use the logical operators #AND#, #NOT#, and #OR#. For
   example, the formula +D2>1800#AND#+D2<2100 matches records with values
   between 1800 and 2100 in column D.
    You can combine criteria from two fields in a formula. For example,
   enter +MONTH="May"#AND#SALES>1800 to match all records for May with sales
   greater than 1800.
    Use relative cell addresses or field names to refer to fields in the
   input range. Use absolute cell addresses or field names to refer to values
   outside the input range. For more information on relative and absolute
   addresses, see Types of Cell and Range References.
