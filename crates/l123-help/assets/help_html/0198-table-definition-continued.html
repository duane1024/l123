Table Definition (continued)

You could use the following table definition to set up a new external
table containing three fields: NAME, MONTH, and SALES. The NAs in the
fourth, fifth, and sixth columns indicate that column labels, field
descriptions, and field creation strings are not available -- that is,
not required -- for the external database in which the external table will
be created.

     NAME     Character        10          NA          NA         NA
     MONTH    Character        12          NA          NA         NA
     SALES    Numeric           9          NA          NA         NA

This table definition describes a field called NAME that contains labels
and has a width of 10; a field called MONTH that also contains labels and
has a width of 12; and a field called SALES that contains values and has
a width of 9.
