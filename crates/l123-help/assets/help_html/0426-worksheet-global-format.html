/Worksheet Global Format

/Worksheet Global Format -- Sets the global cell format for the current
worksheet. Changing the cell format affects only the display of data,
not the data itself. If the current file is in GROUP mode, this command
affects all worksheets in the file.

1. Select /Worksheet Global Format.
2. Select a format from the format menu and complete the prompts and
    menu selections for that format.
3. If a column is not wide enough to display a value in the format
    you selected, l123 fills the cell with asterisks. Use
    /Worksheet Column Set-Width to widen the column and redisplay
    the value.

Use /Range Format to change the format of data for a range. /Worksheet Global
Format does not affect cells formatted with /Range Format.
