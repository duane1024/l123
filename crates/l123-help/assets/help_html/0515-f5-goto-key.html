F5 (GOTO) Key

F5 (GOTO) Key -- Moves the cell pointer to a specific cell or a named range,
another worksheet, or another active file. When you go to another worksheet
or file, l123 moves the cell pointer to the cell you last highlighted
in that worksheet or file. Press F5 (GOTO) and do one of the following:

    To go to a specific cell, type the cell address and press ENTER.
    To go to a named range in the current file, type the range name and press
   ENTER, or press F3 (NAME), highlight the range name and press ENTER. l123
   moves the cell pointer to the first cell in the range.
    To go to another worksheet in the current file, type the worksheet letter
   and : (colon), for example C:, and press ENTER.
    To go to another active file, type the file reference enclosed in <<>>
   (double angle brackets) and press ENTER, or press F3 (NAME), highlight the
   file name and press ENTER. If you type the file name, include the path
   and the extension.
    To go to a specific cell in a different active file, type the file
   reference enclosed in <<>> (double angle brackets) and a cell address and
   press ENTER, for example, <<C:\DEPTA\BUDGET.WK3>>A:C5.
