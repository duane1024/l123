Label Criteria

Label Criteria -- Enter label criteria to search fields which contain labels.

    To search for an exact match, enter a label exactly as it appears in
   the input range.
    Use the wildcard characters ? and * to search for similar labels.
     ?  Matches any single character. For example, h?t matches hat, hot,
        and hut, but not huts; h??d matches head and hood, but not heel.
     *  Matches all characters to the end of the label. For example, cat*
        matches cat, cats, catch, and catsup, but not cart.
    Precede a label with a ~ (tilde) to match all labels except that one.
   For example, ~Mosley matches all records that are not Mosley.
    Combine characters to create label criteria. For example, ~B* matches
   all records that do not begin with B.
    Enter a label prefix (' " or ^) and one of the logical perators
   (= < <= > >= and <>) before a label to search for labels that match
   a specific condition. For example, '<>Smith matches all entries that
   are not Smith.
