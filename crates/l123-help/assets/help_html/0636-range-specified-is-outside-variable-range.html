Range specified is outside variable range

You selected /Data Table Labeled Down, Across, Sheets, or Input-Cells and
tried to specify a set of input values that was not in the variable
range or ranges.

If you selected /Data Table Labeled Down, specify a set of input values
within the column variable range. If you selected /Data Table Labeled
Across, specify a set of input values within the row variable range. If
you selected /Data Table Labeled Sheets, specify a set of input values
within the worksheet variable range. If you selected /Data Table Labeled
Input-Cells, specify a set of input values within any of the variable
ranges you specified for the table.
