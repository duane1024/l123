/Graph Options Scale (continued)

    Automatic     (Default) Creates a scale whose upper and lower limits
                  are based on the values in the graph data ranges.
    Manual        Creates a scale using the upper and lower limits
                  you specify.
    Lower         Sets the lower limit for a graph with manual scaling.
    Upper         Sets the upper limit for a graph with manual scaling.
    Format        Sets the format for scale numbers.
    Indicator     Specifies the contents of the scale indicator.
    Type          Sets the scale to linear or logarithmic.
    Exponent      Sets the order of magnitude for a scale.
    Width         Sets the maximum width of scale numbers.
    Quit          Returns you to the /Graph Options menu.
