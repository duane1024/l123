Disk error

1-2-3 was unable to save a file on or read a file from disk. The disk
may be damaged, or the disk-drive head may be out of alignment.

If you are trying to save a file on a disk, try another formatted disk.
If you cannot save the file on the second disk, have the alignment of
the disk-drive head checked.

If you are trying to read the file, try using another computer.
