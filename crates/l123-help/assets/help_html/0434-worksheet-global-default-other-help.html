/Worksheet Global Default Other Help

This release of 1-2-3 always uses Removable Help, regardless of the setting
you select with /Worksheet Global Default Other Help. When Help is Removable,
1-2-3 opens the Help file each time you press F1 (HELP) and closes the
file again when you press ESC to leave Help.

NOTE  This command exists in this release of 1-2-3 in order to maintain macro
compatibility with earlier 1-2-3 releases.
