/File Admin

/File Admin -- Controls reservations for sharing worksheet files; creates a
table of information about files; seals some settings in worksheet files;
and recalculates all formulas in active files.

Reservation   Lets you get and release the current file's reservation
              and change the reservation setting. Use this command when
              you share worksheet files on a network or in another
              multi-user environment.
Table         Creates a table of information about active files, files on
              disk, or files linked to the current file.
Seal          Prevents changes to some graph, print, range, worksheet, and
              reservation settings.
Link-Refresh  Recalculates formulas in active files that refer to data
              in other active files or files on disk.
