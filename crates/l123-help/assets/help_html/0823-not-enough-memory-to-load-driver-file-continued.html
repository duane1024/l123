Not enough memory to load driver file (continued)

You can prevent this error message from occurring in the future by making
the driver your default driver. To make the driver your default driver:

* If the driver is a printer driver, use /Worksheet Global Default
   Printer Name and Interface and select the correct printer and port.
   Then select Quit Update Quit.

* If the driver is a display driver, end 1-2-3, start Install from your
   hard disk, and select Change Selected Equipment from the Main Menu.
   Make the display you want to load your primary display. (The primary
   display must have the number 1 beside it.) Then save your changes,
   end Install, and start 1-2-3.
