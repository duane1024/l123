/Graph Options Advanced Text [First, Second, Third] Color

/Graph Options Advanced Text [First, Second, Third] Color -- Sets colors for
or hides the graph-text groups. For l123 to use the color settings for
displayed graphs, you must have a color monitor and the graph display must
be set to color. For l123 to use the color settings for printed graphs, you
must have a color printer and the graph display must be set to color.

1. Select /Graph Options Advanced Text.
2. Select First, Second, or Third to select the graph-text group.
3. Select Color and select one of the following:
    1 - 8    To set the corresponding color for the selected
             graph-text group.
    Hide     To hide the selected graph-text group in the graph.

NOTE  To see which colors correspond to the numbers 1 through 8 for your
monitor, assign each of the numbers to a graph-text group and view the
graph using /Graph View.
